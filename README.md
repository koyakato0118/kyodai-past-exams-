# 京大工学部 過去問解説 — Codex引き継ぎパッケージ

このパッケージは、過去問の解答内容とWebサイト実装要件を分離して管理するためのものです。

## 基本方針

1. ChatGPT側で問題を正確に読み取り、解答・初心者向け解説・試験答案をMarkdownで完成させる。
2. Codex側ではMarkdownを唯一の内容ソースとして読み込み、UIとページ遷移を実装する。
3. Codexは数式や解答内容を独自に変更しない。
4. 読み取りが曖昧な問題は、推測せず `needs_confirmation` として止める。

- 工業数学F2 2021年度
- 制御工学1 2023年度

## 未収録

`content-manifest.md` を参照してください。


## 問題文の表示方針

問題文は原文のまま表示します。要約や言い換えは行いません。
図・回路図・グラフ・状態図は元PDFの該当画像を併記し、クリックで拡大できるようにします。
