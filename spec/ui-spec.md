# UI実装仕様

## デザイン

- 白と薄い青を基調にする。
- 参考モックアップと同様に、PCでは3カラム構成にする。
  - 左: 科目・年度・問題ナビゲーション
  - 中央: 問題文・解説・答案
  - 右: 「なぜ？」「重要ポイント」「関連知識」「タグ」
- スマートフォンでは左サイドバーをドロワー式にし、右カラムは本文内へ移動する。
- 数式はKaTeXまたはMathJaxで表示する。

## クリック可能要素のホバー

クリック・タップできる要素には、すべて同じホバー表現を適用する。

対象例:

- `a`
- `button`
- サイドバーのトグル
- 年度・問題のナビゲーション
- 「なぜ？」ボタン
- 前へ・次へボタン
- タグ
- カード型リンク
- 検索結果
- 上へ戻るボタン

必須挙動:

```css
:is(a, button, [role="button"], summary, .clickable) {
  transition: opacity 160ms ease, transform 160ms ease;
  cursor: pointer;
}

:is(a, button, [role="button"], summary, .clickable):hover {
  opacity: 0.7;
}
```

無効状態には適用しない。

```css
button:disabled {
  opacity: 0.4;
  cursor: not-allowed;
}
```

キーボード操作のため、`:focus-visible` も実装する。

```css
:is(a, button, [role="button"], summary):focus-visible {
  outline: 3px solid rgba(47, 128, 237, 0.35);
  outline-offset: 3px;
}
```

## サイドバーの階層

「工業数学F2」を開いたときだけ年度を表示する。

```text
▶ 工業数学F2

▼ 工業数学F2
   ▶ 2023年度
   ▶ 2021年度

▼ 工業数学F2
   ▼ 2023年度
      予備知識
      大問1
        1-1
        1-2
        1-3
        1-4
      大問2
      大問3
      試験答案まとめ
   ▶ 2021年度
```

要件:

- 科目 → 年度 → 大問 → 小問の順にネストする。
- 初期状態では現在の科目・年度・問題に対応する親トグルを開く。
- 他の年度へ移動しても、同じ科目のトグルは開いたままにする。
- `aria-expanded` と `aria-controls` を設定する。
- `summary/details` またはアクセシブルな独自Accordionを使用する。
- URLを直接開いた場合も、該当するナビゲーションをアクティブ表示する。

## 「なぜ？」の展開

- 「なぜ？」は初期状態では閉じる。
- 押すと、その数式または説明に対応する補足だけを展開する。
- 再度押すと閉じる。
- 展開状態を示すアイコンを変更する。
- 内容はMarkdown内の `why` ブロックから生成する。
- 知識がある利用者が補足を読まずに進める構造にする。

## 問題ページ

各ページに次を表示する。

1. パンくず
2. 問題番号とタイトル
3. 前へ・次へ
4. 問題文
5. 必要な予備知識
6. 解法方針
7. 詳しい途中式
8. 「なぜ？」補足
9. 最終解答
10. 試験答案
11. 重要ポイント
12. 関連知識
13. タグ

## 内容の扱い

- Markdownの数式・解答を勝手に変更しない。
- `status: needs_confirmation` の問題は公開しない。
- `status: reviewed` の問題だけ本番表示する。
- ソースPDFをWebサイトへ転載する場合は、権利・共有範囲を別途確認する。初期実装では問題文を必要最小限に要約表示する。


## 問題文の原文表示

- 問題文は要約せず、元PDFの原文をそのまま表示する。
- 中央カラムの最上部に「問題文（原文）」を配置する。
- 図・回路図・グラフ・状態図は元PDFの画像を併記する。
- 「原本を表示」ボタンで該当ページまたは切り出し画像を拡大表示できるようにする。
- 原本画像もクリック可能要素なので、ホバー時の `opacity: 0.7` を適用する。
- 原文と解説の境界を、背景色・見出し・余白で明確にする。
- 詳細は `spec/content-spec.md` を参照する。
