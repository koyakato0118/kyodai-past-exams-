# この年度で必要な予備知識

## 1. Canonical分布と分配関数

温度 \(T\) の熱浴と平衡にある系で、微視的状態のエネルギーを \(E\) とすると、状態の重みは

\[
\exp\left(-\frac{E}{k_{\mathrm B}T}\right)
\]

に比例する。

分配関数は、その重みをすべての状態について足し合わせたもの：

\[
\boxed{
Z=\sum_{\text{states}}
\exp\left(-\frac{E}{k_{\mathrm B}T}\right)
}
\]

連続変数の場合は和を積分へ置き換える。

## 2. Helmholtz自由エネルギー

Canonical分布において、

\[
\boxed{
F=-k_{\mathrm B}T\ln Z
}
\]

である。

また、

\[
\boxed{
F=U-TS
}
\]

なので、自由エネルギーは内部エネルギーとエントロピーの競合を表す。

## 3. 平均値

物理量 \(A\) の平均は、

\[
\boxed{
\langle A\rangle
=
\frac{
\int A e^{-E/(k_{\mathrm B}T)}d\Gamma
}{
\int e^{-E/(k_{\mathrm B}T)}d\Gamma
}
}
\]

で求められる。

## 4. Gaussian積分

\[
\int_{-\infty}^{\infty}e^{-ax^2}dx
=
\sqrt{\frac{\pi}{a}}
\qquad(a>0)
\]

\[
\int_{-\infty}^{\infty}x^2e^{-ax^2}dx
=
\frac{\sqrt\pi}{2a^{3/2}}
\]

を使う。

## 5. Fermi分布

電子の占有確率は、

\[
\boxed{
f(\epsilon)
=
\frac1{
\exp[(\epsilon-\mu)/(k_{\mathrm B}T)]+1
}
}
\]

である。

絶対零度では、

\[
f(\epsilon)=
\begin{cases}
1 & \epsilon\le\mu_0\\
0 & \epsilon>\mu_0
\end{cases}
\]

という階段関数になる。

## 6. Shannonの情報エントロピー

確率 \(p_i\) の事象について、

\[
\boxed{
S=-\sum_i p_i\log_2p_i
}
\]

である。

単位は bit。

- 結果がほぼ確定：\(S\) は小さい
- 結果が同程度に起こる：\(S\) は大きい

二択で確率が \(1/2,1/2\) なら最大値 \(1\) bit。

## 7. Bayesの定理

\[
\boxed{
p(x|y)
=
\frac{p(y|x)p(x)}
{\sum_xp(y|x)p(x)}
}
\]

である。

「原因から結果」の確率 \(p(y|x)\) と事前確率 \(p(x)\) から、「結果を見た後の原因」の確率 \(p(x|y)\) を求める。

## 8. Gaussian鎖

長さ \(b\) のセグメントがランダムな方向を取る鎖をGaussian鎖という。

セグメント数が \(N\) のとき、末端間距離の二乗平均は、

\[
\boxed{
\langle x^2\rangle=Nb^2
}
\]

したがって、典型的な広がりは、

\[
\boxed{
\sqrt{\langle x^2\rangle}=\sqrt N\,b
}
\]

である。

# 問1 古典1次元調和振動子

## 問題文（原文）

:::problem-original
温度 \(T\) の熱浴と接している1次元調和振動子の分配関数は、古典極限においては、次式で表される：

\[
Z(T)=\frac1h
\int_{-\infty}^{\infty}dx
\int_{-\infty}^{\infty}dp
\exp\left[
-\frac{
\frac12cx^2+\frac1{2m}p^2
}{
k_{\mathrm B}T
}
\right].
\tag{1-1}
\]

ただし、\(c\) はバネ定数、\(m\) は粒子質量である。なお、座標 \(x\) と運動量 \(p\) についての不確定性原理を正しく考慮するためにPlanck定数 \(h\) で割ってある。また、力学ではバネ定数を記号 \(k\) で表すことが多いが、Boltzmann定数 \(k_{\mathrm B}\) と紛らわしいので、ここでは \(c\) を用いている。

この系について、以下の問に答えよ。

[1] 積分を実行し、\(Z(T)\) を具体的に求めよ。

[2] Helmholtz自由エネルギー \(F(T)\) の表式を求め、温度の関数としてグラフの概形を描け。

[3] ポテンシャルエネルギーおよび運動エネルギーの平均値をそれぞれ求め、「古典極限ではそれぞれの運動の自由度に、等しくエネルギーが分配される」というエネルギー等分配則が成り立つことを示せ。
:::

## Q1-1 分配関数 \(Z(T)\)

### 解法方針

指数関数の中は、

\[
\frac12cx^2+\frac{p^2}{2m}
\]

という、\(x\) だけの項と \(p\) だけの項の和である。

したがって、

\[
e^{-(E_x+E_p)/(k_{\mathrm B}T)}
=
e^{-E_x/(k_{\mathrm B}T)}
e^{-E_p/(k_{\mathrm B}T)}
\]

となり、二重積分を2つのGaussian積分へ分離できる。

### 積分の分離

\[
Z(T)
=
\frac1h
\left[
\int_{-\infty}^{\infty}
\exp\left(
-\frac{cx^2}{2k_{\mathrm B}T}
\right)dx
\right]
\left[
\int_{-\infty}^{\infty}
\exp\left(
-\frac{p^2}{2mk_{\mathrm B}T}
\right)dp
\right].
\]

1つ目の積分は、

\[
\int_{-\infty}^{\infty}
\exp\left(
-\frac{cx^2}{2k_{\mathrm B}T}
\right)dx
=
\sqrt{\frac{2\pi k_{\mathrm B}T}{c}}.
\]

2つ目は、

\[
\int_{-\infty}^{\infty}
\exp\left(
-\frac{p^2}{2mk_{\mathrm B}T}
\right)dp
=
\sqrt{2\pi mk_{\mathrm B}T}.
\]

したがって、

\[
Z(T)
=
\frac1h
\sqrt{\frac{2\pi k_{\mathrm B}T}{c}}
\sqrt{2\pi mk_{\mathrm B}T}.
\]

\[
=
\frac{2\pi k_{\mathrm B}T}{h}
\sqrt{\frac{m}{c}}.
\]

調和振動子の角振動数を、

\[
\omega=\sqrt{\frac{c}{m}}
\]

とすると、

\[
\sqrt{\frac{m}{c}}=\frac1\omega.
\]

また、

\[
\hbar=\frac{h}{2\pi}
\]

なので、

### 最終解答

\[
\boxed{
Z(T)
=
\frac{2\pi k_{\mathrm B}T}{h}
\sqrt{\frac{m}{c}}
=
\frac{k_{\mathrm B}T}{\hbar\omega}
}
\]

である。

:::why title="なぜ分配関数を h で割るの？"
古典位相空間では状態が連続に無限個あるため、そのまま積分すると状態数の単位が残る。量子力学の不確定性原理では位相空間の1状態がおよそ \(h\) の面積を占めるので、\(\int dx\,dp\) を \(h\) で割って状態数を無次元化する。
:::

### 試験答案

\[
Z
=
\frac1h
\sqrt{\frac{2\pi k_{\mathrm B}T}{c}}
\sqrt{2\pi mk_{\mathrm B}T}
=
\frac{2\pi k_{\mathrm B}T}{h}
\sqrt{\frac mc}.
\]

\[
\omega=\sqrt{c/m},
\qquad
\hbar=h/(2\pi)
\]

より、

\[
\boxed{
Z(T)=\frac{k_{\mathrm B}T}{\hbar\omega}
}.
\]

---

## Q1-2 Helmholtz自由エネルギー

\[
F(T)=-k_{\mathrm B}T\ln Z(T).
\]

前問の結果を代入すると、

\[
\boxed{
F(T)
=
-k_{\mathrm B}T
\ln\left(
\frac{2\pi k_{\mathrm B}T}{h}
\sqrt{\frac mc}
\right)
}
\]

または、

\[
\boxed{
F(T)
=
-k_{\mathrm B}T
\ln\left(
\frac{k_{\mathrm B}T}{\hbar\omega}
\right)
}
\]

である。

### 無次元化

\[
X=\frac{k_{\mathrm B}T}{\hbar\omega}
\]

と置くと、

\[
\boxed{
\frac{F}{\hbar\omega}
=
-X\ln X
}
\]

となる。

### グラフの特徴

関数

\[
Y=-X\ln X
\qquad(X>0)
\]

を考える。

- \(X\to0+0\)：

\[
-X\ln X\to0
\]

- \(X=e^{-1}\) で最大：

\[
Y_{\max}=\frac1e
\]

- \(X=1\) で：

\[
Y=0
\]

- \(X>1\) では負
- \(X\to\infty\) では：

\[
Y\to-\infty
\]

模式図：

```text
F/(ℏω)
  │       ● 最大
  │     ／  ＼
0 ├────┘─────＼──────── X=kBT/(ℏω)
  │             ＼
  │               ＼
```

### 微分

\[
\frac{dF}{dT}
=
-k_{\mathrm B}
\left[
\ln\left(
\frac{k_{\mathrm B}T}{\hbar\omega}
\right)+1
\right].
\]

\(T\to0+0\) では、

\[
\frac{dF}{dT}\to+\infty.
\]

ただし、この低温での発散的な挙動は、古典近似を低温まで無理に延長したために生じる。実際の量子調和振動子では低温挙動が異なる。

:::why title="なぜ T→0 で F は0なのに、傾きは無限大になるの？"
\(T\ln T\to0\) なので関数値は0へ近づくが、微分には \(\ln T\) が残り、\(\ln T\to-\infty\) となるからである。
:::

## Q1-3 エネルギー等分配則

## ポテンシャルエネルギー

\[
U_{\mathrm p}
=
\frac12cx^2.
\]

平均は、

\[
\left\langle
\frac12cx^2
\right\rangle
=
\frac{
\int dx\int dp\,
\frac12cx^2
e^{-[cx^2/2+p^2/(2m)]/(k_{\mathrm B}T)}
}{
\int dx\int dp\,
e^{-[cx^2/2+p^2/(2m)]/(k_{\mathrm B}T)}
}.
\]

p積分は分子と分母で共通なので打ち消し合う。

\[
=
\frac{
\frac c2
\int_{-\infty}^{\infty}
x^2e^{-cx^2/(2k_{\mathrm B}T)}dx
}{
\int_{-\infty}^{\infty}
e^{-cx^2/(2k_{\mathrm B}T)}dx
}.
\]

Gaussian積分を用いると、

\[
\boxed{
\left\langle
\frac12cx^2
\right\rangle
=
\frac12k_{\mathrm B}T
}
\]

である。

## 運動エネルギー

同様に、

\[
\boxed{
\left\langle
\frac{p^2}{2m}
\right\rangle
=
\frac12k_{\mathrm B}T
}
\]

である。

### 全平均エネルギー

\[
\boxed{
\langle E\rangle
=
\frac12k_{\mathrm B}T
+
\frac12k_{\mathrm B}T
=
k_{\mathrm B}T
}
\]

となる。

### 最終結論

座標 \(x\) の二次項と運動量 \(p\) の二次項に、それぞれ等しく、

\[
\boxed{
\frac12k_{\mathrm B}T
}
\]

ずつエネルギーが分配される。

これが、この系におけるエネルギー等分配則である。

> [!IMPORTANT]
> 等分配則は「二次形式で現れる独立な自由度1つにつき \(\frac12k_{\mathrm B}T\)」である。任意のポテンシャルで常に位置エネルギーと運動エネルギーが等しくなるわけではない。

# 問2 金属電子とSeebeck効果

## 問題文（原文）

:::problem-original
金属や半導体などの固体材料の両端に温度差があると、電位差が生じる現象が古くから知られており、発見者 Thomas Seebeck (1770-1831) の名をとって、ゼーベック効果 Seebeck effect とよばれている。ゼーベック効果は熱電対による温度測定の原理として広く使われているほか、最近では、小さな温度差から電気エネルギーを取り出す環境発電の原理としても、再注目されている。

このようなゼーベック効果のメカニズムを、金属の簡単なモデルに基づいて考察しよう。固体バンド理論によれば、金属とは「伝導帯の途中まで電子が詰まっている物質」である。ここでは、体積 \(V\) の金属中の電子について、微視的エネルギー \(\epsilon\) の多重度密度（状態密度）として、自由電子モデルの関数形

\[
g(\epsilon)
=
\frac{
2^{7/2}\pi m_e^{3/2}V
}{
h^3
}
\sqrt\epsilon
\tag{2-1}
\]

を仮定する。ただし、\(m_e\) は電子の質量である。ここでは、エネルギーは伝導帯の下端 \(\epsilon_c\) からはかる。一方、温度 \(T\) の熱平衡状態では、電子の分布（占有数）は、化学ポテンシャル \(\mu\)（やはり \(\epsilon_c\) からはかる）のFermi分布関数

\[
f(\epsilon)
=
\frac1{
\exp[
(\epsilon-\mu)/(k_{\mathrm B}T)
]+1
}
\tag{2-2}
\]

に従う。これにより、伝導帯中の電子の総数 \(N\) は次式で表される：

\[
N
=
\int_{\epsilon_c}^{\epsilon_t}
g(\epsilon)f(\epsilon)d\epsilon
\simeq
\frac{
2^{7/2}\pi m_e^{3/2}V
}{
h^3
}
\int_0^\infty
\frac{\sqrt\epsilon}{
\exp[
(\epsilon-\mu)/(k_{\mathrm B}T)
]+1
}
d\epsilon.
\tag{2-3}
\]

ただし、\(\epsilon_t\) は伝導帯の上端のエネルギーであるが、そこでの占有数 \(f(\epsilon_t)\) は、通常の温度ではほぼゼロなので、積分範囲を無限大に拡大している。

以下の問いに答えよ。

[1] 絶対零度 \(T\to0\) では、Fermi分布は階段関数となるので、積分 (2-3) は容易に実行できる。これにより、絶対零度での化学ポテンシャル \(\mu_0\) の表式を求めよ。

[2] 銅（原子量53.5、密度 \(8.92\,\mathrm{g/cm^3}\)）の場合に、\(\mu_0\) を求め、温度単位であらわせ。ただし、結晶中では、1つの銅原子から1つの電子が伝導帯に放出されるものとする。（すなわち、銅のFermi温度 \(T_F\) を求めるという問題である。）

[3] \(\mu\) は温度の関数であり、低温 \(T\ll T_F\) では次の近似式が成り立つことが知られている：

\[
\mu(T)
\simeq
\mu_0
\left[
1+
A
\left(
\frac{k_{\mathrm B}T}{\mu_0}
\right)^2
\right].
\tag{2-4}
\]

ここで、\(A\) は大きさ1程度の正の無次元定数である。金属棒の両端に温度差があると、式 (2-4) にしたがって \(\mu\) の差が生じるので、電子は \(\mu\) の大きいほうから小さいほうへ移動し、結果として両端に電位差が生じる。これが、Seebeck効果の原理である。電位 \(V\,[\mathrm V]\) の中での電子のエネルギーは \(\epsilon-eV\)（\(e\) は電気素量、電子は \(-e\) の電荷をもつ）となるから、定常状態においては、

\[
\frac{
\epsilon-eV(T)-\mu(T)
}{
k_{\mathrm B}T
}
=D
\tag{2-5}
\]

（\(D\) は無次元定数）となっているはずである。式 (2-5) と式 (2-4) から、ゼーベック係数 \(dV/dT\) の表式を求めよ。

[4] 銅の場合について、\(T=300\,\mathrm K\) での \(dV/dT\) の値を計算せよ。ただし、\(A=1,D=0\) とせよ。なお一般に、金属のゼーベック係数は \(10^{-6}\,\mathrm{V/K}\) 程度（つまり、1 Kの温度差により作られる電位差は1マイクロV程度）と非常に小さい。
:::

## Q2-1 絶対零度の化学ポテンシャル

絶対零度では、

\[
f(\epsilon)=
\begin{cases}
1 & 0\le\epsilon\le\mu_0\\
0 & \epsilon>\mu_0
\end{cases}
\]

である。

したがって、

\[
N
=
\frac{
2^{7/2}\pi m_e^{3/2}V
}{
h^3
}
\int_0^{\mu_0}\sqrt\epsilon\,d\epsilon.
\]

\[
\int_0^{\mu_0}\sqrt\epsilon\,d\epsilon
=
\frac23\mu_0^{3/2}.
\]

よって、

\[
N
=
\frac{
2^{7/2}\pi m_e^{3/2}V
}{
h^3
}
\frac23\mu_0^{3/2}.
\]

\(\mu_0\) について解く。

\[
\mu_0^{3/2}
=
\frac{
3h^3
}{
2^{9/2}\pi m_e^{3/2}
}
\frac NV.
\]

両辺を \(2/3\) 乗すると、

### 最終解答

\[
\boxed{
\mu_0
=
\left(
\frac3\pi
\right)^{2/3}
\frac{h^2}{8m_e}
\left(
\frac NV
\right)^{2/3}
}
\]

である。

\(\hbar=h/(2\pi)\) を使う標準形では、

\[
\boxed{
\mu_0
=
\frac{\hbar^2}{2m_e}
\left(
3\pi^2\frac NV
\right)^{2/3}
}
\]

となる。

:::why title="絶対零度で μ0 がFermiエネルギーになるの？"
絶対零度では低いエネルギー状態から順に電子が詰まり、占有されている最も高いエネルギーが境界 \(\mu_0\) になる。このエネルギーをFermiエネルギーという。
:::

## Q2-2 銅のFermi温度

### 電子数密度

銅原子1個につき伝導電子1個なので、電子数密度は原子数密度と等しい。

\[
\frac NV
=
\frac{\rho}{M}N_{\mathrm A}.
\]

問題の値を使う：

\[
\rho=8.92\,\mathrm{g/cm^3},
\qquad
M=53.5\,\mathrm{g/mol}.
\]

SI単位へ変換して計算すると、

\[
\frac NV
=
\frac{
8.92\,\mathrm{g/cm^3}
}{
53.5\,\mathrm{g/mol}
}
\frac{6.02\times10^{23}}{10^{-6}\,\mathrm{m^3/cm^3}}.
\]

\[
\boxed{
\frac NV
\simeq
1.00\times10^{29}\,\mathrm{m^{-3}}
}
\]

である。

### Fermiエネルギー

前問の式へ代入すると、

\[
\boxed{
\mu_0
\simeq
1.26\times10^{-18}\,\mathrm J
}
\]

である。

電子ボルトでは、

\[
\mu_0
\simeq
\frac{1.26\times10^{-18}}
{1.60\times10^{-19}}
\simeq
7.9\,\mathrm{eV}.
\]

### Fermi温度

\[
\boxed{
T_F=\frac{\mu_0}{k_{\mathrm B}}
}
\]

なので、

\[
T_F
=

# 問3 Bayes推定と情報エントロピー

## 問題文（原文）

:::problem-original
ある製品を製造する2つのライン、「ラインA」と「ラインB」を持っている工場があるとする。どちらも単位時間あたりの製造スピードは同じである。どちらのラインも一定の確率でランダムに不良品が発生すると仮定する。

ラインAが不良品を出す確率は10%、ラインBが不良品を出す確率は5%であるとしよう。（実際にはこんなに不良品率が高いと商売にならないはずだが、計算を簡単にするためである。）

以下の問いに答えよ。

[1] 製品は、出荷前に徹底的に検査され、チェックレポートが作成される。そのレポートがもつ情報量を、シャノンのエントロピーの定義

\[
S=-\sum_ip_i\log_2p_i
\qquad[\mathrm{bit}]
\tag{4-1}
\]

を使って、ラインAから出た製品およびラインBから出た製品それぞれについて有効数字2桁程度まで求め、その大小について簡単に議論せよ。

[2] 出荷前のある製品を検査したところ、不良品であることがわかったとしよう。これが、ラインAで作られたものか、ラインBで作られたものかを、次の手順で推定せよ。

(2.1) まず、2つの確率事象 \(x,y\) について一般的な考察をおこなう。この2つは独立とは限らない。次の2種類の条件付き確率

- \(x\) の場合に \(y\) となる確率：\(p(y|x)\)
- \(y\) の場合に \(x\) である確率：\(p(x|y)\)

について、次式が一般的に成り立つことを証明せよ。

\[
p(x|y)
=
\frac{p(y|x)p(x)}
{\sum_xp(y|x)p(x)}.
\tag{4-2}
\]

ただし、右辺にあらわれる \(p(x)\) は、\(y\) にかかわらず \(x\) が起きる確率をあらわす。

(2.2) 事象 \(x\) は製品が作られたラインの区別 \(x=\{A,B\}\) とする。事象 \(y\) は製品の質 \(y=\{\text{良、不良}\}\) に対応させる。式 (4-2) を用いて、「出てきた不良品がラインAで作られたものである確率」を求めよ。

[3] 出荷される製品はもちろん検査によって「良品」と判断されたものだけである。出荷された製品に添付されているチェックレポートがもつ情報量（シャノンのエントロピー）を計算せよ。その値は「不良品」も含めた全製品のチェックレポートの情報量とは異なっているはずである。両者の違いについて簡単に考察せよ。
:::

:::

## Q3-2-1 Bayesの定理の証明

結合確率 \(p(x,y)\) を2通りに表す。

\[
p(x,y)=p(x|y)p(y)
\]

また、

\[
p(x,y)=p(y|x)p(x).
\]

したがって、

\[
p(x|y)p(y)
=
p(y|x)p(x).
\]

よって、

\[
p(x|y)
=
\frac{p(y|x)p(x)}{p(y)}.
\]

全確率の公式より、

\[
p(y)
=
\sum_xp(y|x)p(x).
\]

したがって、

### 最終解答

\[
\boxed{
p(x|y)
=
\frac{p(y|x)p(x)}
{\sum_xp(y|x)p(x)}
}
\]

である。

:::why title="分母の和は何を表すの？"
事象 \(y\) が起こる可能性を、すべての原因 \(x\) について足し合わせた全確率 \(p(y)\) である。
:::

## Q3-2-2 不良品がラインA製である確率

両ラインの製造速度が同じなので、事前確率は、

\[
p(A)=p(B)=\frac12.
\]

不良率は、

\[
p(\mathrm{不良}|A)=0.10
\]

\[
p(\mathrm{不良}|B)=0.05.
\]

Bayesの定理より、

\[
p(A|\mathrm{不良})
=
\frac{
p(\mathrm{不良}|A)p(A)
}{
p(\mathrm{不良}|A)p(A)
+
p(\mathrm{不良}|B)p(B)
}.
\]

\[
=
\frac{
0.10\times\frac12
}{
0.10\times\frac12
+
0.05\times\frac12
}.
\]

\[
=
\frac{0.10}{0.15}.
\]

### 最終解答

\[
\boxed{
p(A|\mathrm{不良})
=
\frac23
\simeq0.667
}
\]

である。

したがって、不良品を見つけた場合、それがラインAで作られた確率は約67%。

### 直感的な説明

両ラインから100個ずつ作ったと考える。

- Aの不良品：10個
- Bの不良品：5個

不良品は合計15個で、そのうち10個がA製。

\[
\frac{10}{15}=\frac23.
\]

## Q3-3 出荷製品レポートの情報量

出荷される製品は良品と確定している。

したがって、レポートに残る不確定性は、

\[
\boxed{\text{その良品がA製かB製か}}
\]

である。

### 良品であるときの製造ラインの確率

Bayesの定理より、

\[
p(A|\mathrm{良})
=
\frac{
0.90\times\frac12
}{
0.90\times\frac12
+
0.95\times\frac12
}.
\]

\[
=
\frac{0.90}{1.85}
\simeq0.48649.
\]

\[
\boxed{
p(A|\mathrm{良})
\simeq0.4865
}
\]

同様に、

\[
\boxed{
p(B|\mathrm{良})
=
\frac{0.95}{1.85}
\simeq0.5135
}
\]

である。

### 条件付きエントロピー

\[
S(X|\mathrm{良})
=
-\left[
p(A|\mathrm{良})
\log_2p(A|\mathrm{良})
+
p(B|\mathrm{良})
\log_2p(B|\mathrm{良})
\right].
\]

### 最終解答

\[
\boxed{
S(X|\mathrm{良})
\simeq0.99947\,\mathrm{bit}
}
\]

ほぼ、

\[
\boxed{1\,\mathrm{bit}}
\]

である。

### 全製品のレポートとの比較

製造ラインと品質を両方含めた4つの事象は、

\[
p(A,\mathrm{良})=0.5\times0.90=0.45
\]

\[
p(A,\mathrm{不良})=0.5\times0.10=0.05
\]

\[
p(B,\mathrm{良})=0.5\times0.95=0.475
\]

\[
p(B,\mathrm{不良})=0.5\times0.05=0.025.
\]

したがって、全製品のレポートの情報エントロピーは、

\[
S_{\mathrm{all}}
=
-\sum p\log_2p.
\]

\[
\boxed{
S_{\mathrm{all}}
\simeq1.3777\,\mathrm{bit}
}
\]

である。

よって、

\[
\boxed{
S(X|\mathrm{良})
<
S_{\mathrm{all}}
}
\]

となる。

### 違いの意味

出荷製品では「良品である」ことがすでに判明しているため、品質についての不確定性が消えている。

そのため、ラインと品質の両方が不明な全製品の場合より、レポートの情報量は小さい。

一方、AとBの良品率は90%と95%で大きくは違わないので、良品であることを知っても製造ラインはほぼ半々のままであり、ラインに関するエントロピーはほぼ1 bitになる。

> [!IMPORTANT]
> 有効数字2桁だけで計算すると \(0.99947\) bitも \(1.00\) bitに見え、差が分からなくなる。この小問では、ライン差によるわずかな情報量の低下を見るため、少し多めの桁で計算する。

## Q4-1 Gaussian鎖の張力

### 鎖の微視的状態数

各セグメントは1次元上で、

- \(+b\) 方向
- \(-b\) 方向

のどちらかを向くとする。

プラス方向のセグメント数を \(n_+\)、マイナス方向を \(n_-\) とする。

\[
N=n_++n_-
\]

末端間距離は、

\[
x=(n_+-n_-)b.
\]

この2式を解くと、

\[
n_+
=
\frac N2
\left(
1+\frac{x}{Nb}
\right)
\]

\[
n_-
=
\frac N2
\left(
1-\frac{x}{Nb}
\right).
\]

この配置を作る場合の数は、

\[
\boxed{
W(x)
=
\frac{N!}{n_+!n_-!}
}
\]

である。

### エントロピー

\[
S(x)=k_{\mathrm B}\ln W(x).
\]

セグメント間に相互作用がない理想鎖では、内部エネルギーを位置によらない定数とみなせる。

したがって、自由エネルギーの \(x\) 依存性は、

\[
F(x)=-TS(x)+\text{定数}
\]

である。

張力は、鎖を伸ばすために必要な力として、

\[
f=\frac{\partial F}{\partial x}
\]

と定義する。

Stirlingの公式、

\[
\ln n!\simeq n\ln n-n
\]

を使って微分すると、

\[
\boxed{
f(x)
=
\frac{k_{\mathrm B}T}{2b}
\ln
\left(
\frac{1+x/(Nb)}
{1-x/(Nb)}
\right)
}
\]

を得る。

### 小変形 \(x\ll Nb\) の近似

\[
u=\frac{x}{Nb}\ll1
\]

と置く。

\[
\ln\left(\frac{1+u}{1-u}\right)
=
2u+\frac{2u^3}{3}+\cdots.
\]

1次まで残すと、

\[
\ln\left(\frac{1+u}{1-u}\right)
\simeq2u.
\]

したがって、

\[
f
\simeq
\frac{k_{\mathrm B}T}{2b}
2\frac{x}{Nb}.
\]

### 最終解答

\[
\boxed{
f
\simeq
\frac{k_{\mathrm B}T}{Nb^2}x
}
\qquad(x\ll Nb)
\]

である。

これはHookeの法則と同じ形であり、等価ばね定数は、

\[
\boxed{
k_{\mathrm{eff}}
=
\frac{k_{\mathrm B}T}{Nb^2}
}
\]

である。

:::why title="なぜ相互作用のない鎖にも張力が生じるの？"
鎖を伸ばすと、取り得る折れ曲がり方の数が減ってエントロピーが低下する。系はエントロピーの大きい縮れた状態へ戻ろうとするため、エントロピー起源の復元力が生じる。
:::

### 温度依存性

\[
k_{\mathrm{eff}}\propto T.
\]

したがって、理想Gaussian鎖のエントロピー弾性では、温度が高いほど同じ伸びに対する張力が大きくなる。

金属ばねの弾性とは異なる特徴である。

### 試験答案

\[
x=(n_+-n_-)b,
\qquad
N=n_++n_-.
\]

したがって、

\[
n_\pm
=
\frac N2
\left(
1\pm\frac{x}{Nb}
\right).
\]

\[
W=\frac{N!}{n_+!n_-!},
\qquad
F=-k_{\mathrm B}T\ln W.
\]

Stirling近似を用いて、

\[
f=\frac{\partial F}{\partial x}
=
\frac{k_{\mathrm B}T}{2b}
\ln
\frac{1+x/(Nb)}{1-x/(Nb)}.
\]

\(x\ll Nb\) では、

\[
\boxed{
f\simeq\frac{k_{\mathrm B}T}{Nb^2}x
}.
\]

## Q4-2 DNAの広がりと核内収納

### Gaussian鎖としての広がり

セグメント数は、

\[
N=6.0\times10^9.
\]

セグメント長は、

\[
b=0.34\,\mathrm{nm}
=
0.34\times10^{-9}\,\mathrm m.
\]

Gaussian鎖の末端間二乗平均は、

\[
\langle x^2\rangle=Nb^2.
\]

したがって、

\[
R_{\mathrm{rms}}
=
\sqrt{\langle x^2\rangle}
=
\sqrt N\,b.
\]

\[
\sqrt{6.0\times10^9}
=
\sqrt{60}\times10^4
\simeq
7.75\times10^4.
\]

よって、

\[
R_{\mathrm{rms}}
\simeq
(7.75\times10^4)
(0.34\times10^{-9})\,\mathrm m.
\]

### 最終解答

\[
\boxed{
R_{\mathrm{rms}}
\simeq
2.6\times10^{-5}\,\mathrm m
=
26\,\mu\mathrm m
}
\]

である。

細胞核の半径は、

\[
3\,\mu\mathrm m
\]

なので、相互作用のないGaussian鎖としての自然な広がりは、核半径の約9倍である。

### 参考：DNAの輪郭長

全セグメントを直線状へ並べた長さは、

\[
L=Nb.
\]

\[
=
(6.0\times10^9)(0.34\,\mathrm{nm})
\simeq2.0\,\mathrm m.
\]

約2 mのDNAが、数 \(\mu\mathrm m\) の細胞核へ収まっていることになる。

---

## 自由エネルギーを使った収納の議論

自由エネルギーは、

\[
\boxed{
F=U-TS
}
\]

である。

### 1. エントロピー

DNAを自然な広がり \(26\,\mu\mathrm m\) より小さい半径 \(3\,\mu\mathrm m\) の領域へ閉じ込めると、取り得る立体配置の数が減る。

したがって、

\[
\boxed{\Delta S<0}
\]

である。

その結果、

\[
-T\Delta S>0
\]

となり、エントロピー項は自由エネルギーを上昇させる。

つまり、エントロピーだけを考えれば、DNAは核内へ強く圧縮されることを嫌う。

### 2. 内部エネルギー

DNAと核内タンパク質の間に引力的相互作用があると、巻き付いたり凝縮したりすることで内部エネルギーが下がる。

\[
\boxed{\Delta U<0}
\]

例えばDNAは、正電荷を持つヒストンタンパク質へ巻き付き、ヌクレオソームを形成する。

さらに高次のクロマチン構造へ折り畳まれる。

### 3. 自由エネルギー

収納が自発的に起こるには、

\[
\boxed{
\Delta F
=
\Delta U-T\Delta S<0
}
\]

でなければならない。

圧縮によるエントロピー低下は、

\[
-T\Delta S>0
\]

として不利に働く。

しかし、DNA-ヒストン間の引力などによる内部エネルギーの低下が十分大きければ、

\[
|\Delta U|>|T\Delta S|
\]

となり、全体として自由エネルギーが低下する。

### 最終結論

\[
\boxed{
\text{DNAは、エントロピー損失を上回る内部エネルギー低下によって、核内へ安定に収納される}

# 年度全体の重要ポイント

- 古典調和振動子の分配関数は \(Z=k_{\mathrm B}T/(\hbar\omega)\)。
- 二次形式の自由度1つにつき平均エネルギーは \(\frac12k_{\mathrm B}T\)。
- 絶対零度の自由電子気体では、化学ポテンシャルはFermiエネルギーに等しい。
- 銅のFermi温度は室温より非常に高く、室温でも強く縮退している。
- Seebeck係数は化学ポテンシャルの温度依存性から導ける。
- 情報エントロピーは、結果が予測しにくいほど大きい。
- Bayesの定理は、観測結果から原因の確率を更新する。
- Gaussian鎖の弾性は、内部エネルギーではなく主としてエントロピーに由来する。
- DNAの核内収納には、エントロピー損失を補う内部エネルギー低下が必要である。
