# この年度で必要な予備知識

## 1. 複素速度ポテンシャル

二次元の非圧縮・渦なし流れでは、複素座標

\[
z=x+iy
\]

を使い、複素速度ポテンシャルを

\[
f(z)=\phi(x,y)+i\psi(x,y)
\]

と表せる。

- \(\phi\)：速度ポテンシャル
- \(\psi\)：流れ関数

複素速度との関係は、

\[
\boxed{
\frac{df}{dz}=u-iv
}
\]

である。

流線は、

\[
\boxed{\psi(x,y)=C}
\]

で表される。

## 2. 直交座標と極座標の速度成分

\[
\boldsymbol v=(u,v)
\]

を極座標成分で表すと、

\[
\boxed{
v_r=u\cos\theta+v\sin\theta
}
\]

\[
\boxed{
v_\theta=-u\sin\theta+v\cos\theta
}
\]

である。

逆に、

\[
u=v_r\cos\theta-v_\theta\sin\theta
\]

\[
v=v_r\sin\theta+v_\theta\cos\theta
\]

である。

## 3. 循環

閉曲線 \(C\) に沿う循環は、

\[
\boxed{
\Gamma=\oint_C\boldsymbol v\cdot d\boldsymbol x
}
\]

で定義される。

半径 \(r\) の円周上では、

\[
d\boldsymbol x=r\,d\theta\,\boldsymbol e_\theta
\]

なので、

\[
\boxed{
\Gamma=\int_0^{2\pi}v_\theta r\,d\theta
}
\]

となる。

## 4. Navier-Stokes方程式

非圧縮性Newton流体について、

\[
\boxed{
\rho
\left[
\frac{\partial\boldsymbol v}{\partial t}
+
(\boldsymbol v\cdot\nabla)\boldsymbol v
\right]
=
-\nabla p+\mu\nabla^2\boldsymbol v
}
\]

である。

- \(\rho\)：密度
- \(p\)：圧力
- \(\mu\)：粘性係数
- \(\nu=\mu/\rho\)：動粘性係数

## 5. Stokes近似

慣性項が粘性項に比べて十分小さい場合、

\[
\rho
\left[
\frac{\partial\boldsymbol v}{\partial t}
+
(\boldsymbol v\cdot\nabla)\boldsymbol v
\right]
\]

を無視して、

\[
\boxed{
0=-\nabla p+\mu\nabla^2\boldsymbol v
}
\]

と近似する。

## 6. 軸対称流のStokes流れ関数

球座標 \((r,\theta)\) で軸対称流を考える。

Stokes流れ関数 \(\psi(r,\theta)\) を用いると、

\[
\boxed{
v_r=
\frac{1}{r^2\sin\theta}
\frac{\partial\psi}{\partial\theta}
}
\]

\[
\boxed{
v_\theta=
-\frac{1}{r\sin\theta}
\frac{\partial\psi}{\partial r}
}
\]

である。

この定義を使うと、非圧縮条件が自動的に満たされる。

## 7. レイノルズ数

代表速度を \(U\)、代表長さを \(L\) とすると、

\[
\boxed{
Re=\frac{\rho UL}{\mu}
=\frac{UL}{\nu}
}
\]

である。

レイノルズ数は、

\[
\boxed{
\frac{\text{慣性力の大きさ}}
{\text{粘性力の大きさ}}
}
\]

を表す無次元数である。

# 問題1 二重湧き出し

## 問題文（原文）

:::problem-original
二重湧き出しの複素速度ポテンシャル

\[
f(z)=-\frac{\mu}{z}
\]

について考える。

[1] \(\mu>0\) のとき、\(\psi=C\)（定数）の流線の式を求め、その図を描け。

[2] 速度ベクトル

\[
\boldsymbol v=(u,v)
\]

を求めよ。極座標

\[
z=re^{i\theta}
\]

を用いよ。

[3] 速度の \(\theta\) 成分 \(v_\theta\) を \(r\) と \(\theta\) で表せ。

v_\theta

なので、まず \(\boldsymbol v\) と接線単位ベクトル \(\boldsymbol l\) の \(x,y\) 成分を \(r,\theta\) で表すこと。

[4] 半径 \(r\) の円周 \(C\) に沿った循環

\[
\Gamma
=
\oint_C\boldsymbol v\cdot d\boldsymbol x
=
\int_0^{2\pi}v_\theta r\,d\theta
\]

を求めよ。

## Q1-1 流線の式

### 複素速度ポテンシャルを実部と虚部に分ける

極座標では、

\[
z=re^{i\theta}
\]

なので、

\[
\frac1z
=
\frac1r e^{-i\theta}
=
\frac1r(\cos\theta-i\sin\theta).
\]

したがって、

\[
f(z)
=
-\frac{\mu}{r}
(\cos\theta-i\sin\theta).
\]

\[
=
-\frac{\mu}{r}\cos\theta
+
i\frac{\mu}{r}\sin\theta.
\]

\[
f=\phi+i\psi
\]

と比較すると、

\[
\boxed{
\phi=-\frac{\mu}{r}\cos\theta
}
\]

\[
\boxed{
\psi=\frac{\mu}{r}\sin\theta
}
\]

である。

### 流線条件

流線は、

\[
\psi=C
\]

なので、

\[
\frac{\mu}{r}\sin\theta=C.
\]

\(C\ne0\) のとき、

\[
\boxed{
r=\frac{\mu}{C}\sin\theta
}
\]

である。

### 直交座標で表す

\[
\sin\theta=\frac{y}{r}
\]

なので、

\[
C=\frac{\mu y}{r^2}.
\]

\[
r^2=x^2+y^2
\]

より、

\[
C(x^2+y^2)=\mu y.
\]

\(C\ne0\) として整理する。

\[
x^2+y^2-\frac{\mu}{C}y=0.
\]

平方完成すると、

\[
\boxed{
x^2+
\left(
y-\frac{\mu}{2C}
\right)^2
=
\left(
\frac{\mu}{2C}
\right)^2
}
\]

である。

### 流線の形

これは、

- 中心：

\[
\left(
0,\frac{\mu}{2C}
\right)
\]

- 半径：

\[
\left|
\frac{\mu}{2C}
\right|
\]

の円である。

すべての円が原点を通る。

- \(C>0\)：上半平面側の円
- \(C<0\)：下半平面側の円
- \(C=0\)：\(x\) 軸上の流線

### 最終解答

\[
\boxed{
\psi=\frac{\mu}{r}\sin\theta
}
\]

\[
\boxed{
r=\frac{\mu}{C}\sin\theta
}
\]

または、

\[
\boxed{
x^2+
\left(
y-\frac{\mu}{2C}
\right)^2
=
\left(
\frac{\mu}{2C}
\right)^2
}
\]

である。

:::why title="なぜ流線が円になるの？"
極座標の式 \(r=a\sin\theta\) は、直交座標に直すと \(x^2+(y-a/2)^2=(a/2)^2\) となる。これは中心がy軸上にあり、原点を通る円の方程式である。
:::

### 試験答案

\[
f(z)
=
-\frac{\mu}{r}e^{-i\theta}
=
-\frac{\mu}{r}\cos\theta
+
i\frac{\mu}{r}\sin\theta.
\]

したがって、

\[
\psi=\frac{\mu}{r}\sin\theta.
\]

\[
\psi=C
\]

より、

\[
r=\frac{\mu}{C}\sin\theta.
\]

直交座標では、

\[
\boxed{
x^2+
\left(
y-\frac{\mu}{2C}
\right)^2
=
\left(
\frac{\mu}{2C}
\right)^2
}.
\]

## Q1-2 速度ベクトル

複素速度は、

\[
u-iv=\frac{df}{dz}.
\]

\[
f(z)=-\frac{\mu}{z}
\]

を微分すると、

\[
\frac{df}{dz}
=
\frac{\mu}{z^2}.
\]

極座標では、

\[
z^2=r^2e^{2i\theta}.
\]

したがって、

\[
\frac{\mu}{z^2}
=
\frac{\mu}{r^2}
e^{-2i\theta}.
\]

Eulerの公式を使うと、

\[
\frac{df}{dz}
=
\frac{\mu}{r^2}
(\cos2\theta-i\sin2\theta).
\]

\[
u-iv
=
\frac{\mu}{r^2}
(\cos2\theta-i\sin2\theta)
\]

と比較して、

\[
\boxed{
u=\frac{\mu}{r^2}\cos2\theta
}
\]

\[
\boxed{
v=\frac{\mu}{r^2}\sin2\theta
}
\]

である。

### 直交座標での表現

\[
\cos2\theta
=
\frac{x^2-y^2}{r^2}
\]

\[
\sin2\theta
=
\frac{2xy}{r^2}
\]

より、

\[
\boxed{
u=
\mu\frac{x^2-y^2}{(x^2+y^2)^2}
}
\]

\[
\boxed{
v=
\mu\frac{2xy}{(x^2+y^2)^2}
}
\]

とも書ける。

:::why title="複素速度が u+iv ではなく u-iv なの？"
複素速度ポテンシャルを \(f=\phi+i\psi\) と定義すると、Cauchy-Riemannの関係から \(df/dz=u-iv\) となる。講義で別の符号規約を使う場合は定義を確認する必要がある。
:::

### 最終解答

\[
\boxed{
\boldsymbol v
=
\left(
\frac{\mu}{r^2}\cos2\theta,\ 
\frac{\mu}{r^2}\sin2\theta
\right)
}
\]

である。

## Q1-3 周方向速度 \(v_\theta\)

### 接線単位ベクトル

極座標の単位ベクトルは、

\[
\boldsymbol e_r
=
(\cos\theta,\sin\theta)
\]

\[
\boldsymbol e_\theta
=
(-\sin\theta,\cos\theta).
\]

問題文の接線単位ベクトル \(\boldsymbol l\) は、

\[
\boxed{
\boldsymbol l=\boldsymbol e_\theta
=
(-\sin\theta,\cos\theta)
}
\]

である。

### 内積を計算する

\[
v_\theta
=
\boldsymbol v\cdot\boldsymbol e_\theta.
\]

\[
=
-u\sin\theta+v\cos\theta.
\]

前問の結果を代入する。

\[
v_\theta
=
-\frac{\mu}{r^2}
\cos2\theta\sin\theta
+
\frac{\mu}{r^2}
\sin2\theta\cos\theta.
\]

\[
=
\frac{\mu}{r^2}
\left[
\sin2\theta\cos\theta
-
\cos2\theta\sin\theta
\right].
\]

三角関数の公式、

\[
\sin A\cos B-\cos A\sin B
=
\sin(A-B)
\]

を使うと、

\[
v_\theta
=
\frac{\mu}{r^2}
\sin(2\theta-\theta).
\]

### 最終解答

\[
\boxed{
v_\theta
=
\frac{\mu}{r^2}\sin\theta
}
\]

である。

### 参考：動径方向速度

同様に、

\[
v_r
=
u\cos\theta+v\sin\theta
\]

なので、

\[
\boxed{
v_r
=
\frac{\mu}{r^2}\cos\theta
}
\]

である。

:::why title="流れ関数から直接求められないの？"
二次元極座標の流れ関数では \(v_r=(1/r)\partial\psi/\partial\theta\)、\(v_\theta=-\partial\psi/\partial r\) を使える。今回 \(\psi=\mu\sin\theta/r\) なので、同じ答えがすぐ得られる。
:::

## Q1-4 円周に沿う循環

前問より、

\[
v_\theta
=
\frac{\mu}{r^2}\sin\theta.
\]

したがって、

\[
\Gamma
=
\int_0^{2\pi}
v_\theta r\,d\theta.
\]

\[
=
\frac{\mu}{r}
\int_0^{2\pi}
\sin\theta\,d\theta.
\]

\[
\int_0^{2\pi}\sin\theta\,d\theta=0
\]

なので、

### 最終解答

\[
\boxed{\Gamma=0}
\]

である。

### 物理的な意味

二重湧き出しは渦流れではなく、原点の外側では渦なし流れである。

円周の上半分と下半分で周方向速度の寄与が打ち消し合うため、循環は0になる。

# 問題2 Stokes流と球の抵抗

## 問題文（原文）

:::problem-original
非圧縮性の粘性流体に対するNavier-Stokes方程式は、

\[
\rho
\left[
\frac{\partial\boldsymbol v}{\partial t}
+
(\boldsymbol v\cdot\nabla)\boldsymbol v
\right]
=
-\operatorname{grad}p
+
\mu\nabla^2\boldsymbol v
\tag{1}
\]

である。

特に、おそい流れでは、方程式 (1) は、

\[
\rho
\frac{\partial\boldsymbol v}{\partial t}
=
-\operatorname{grad}p
+
\mu\nabla^2\boldsymbol v
\tag{2}
\]

と近似できる。これをStokes近似という。

[1] Stokes近似が適用できる条件について議論せよ。

定常流では、方程式 (2) は、

\[
\mu\nabla^2\boldsymbol v
=
\operatorname{grad}p
\tag{3}
\]

となる。

\(x\) 方向の一様流中（遠方での速度 \(U\)）に、半径 \(a\) の球が原点に置かれているとき、方程式 (3) を満たす定常流を考える。流れは \(x\) 軸に関して軸対称とし、\(x\) 軸を極軸とする球座標 \((r,\theta)\) を用いる。

軸対称流ではStokes流れ関数 \(\psi(r,\theta)\) を定義できて、

\[
r\text{方向の速度}:
\qquad
v_r
=
\frac{1}{r^2\sin\theta}
\frac{\partial\psi}{\partial\theta}
\]

\[
\theta\text{方向の速度}:
\qquad
v_\theta
=
-\frac{1}{r\sin\theta}
\frac{\partial\psi}{\partial r}
\]

である。

遠方 \((r\to\infty)\) での境界条件は、

\[
v_r=\boxed{\text{ア}},
\qquad
v_\theta=\boxed{\text{イ}}
\]

である。

[2] ア、イに入る式を書け。また、\(\psi(r\to\infty)=\psi_\infty(r,\theta)\) を求めよ。

任意の点における \(\psi\) を、

\[
\psi
=
\psi_\infty(r,\theta)
+
A\frac{\sin^2\theta}{r}
-
Br\sin^2\theta
\tag{4}
\]

の形で求めることを考える。

式 (4) の右辺第1項は一様流、第2項は二重湧き出し、第3項はStokes源（粘性効果）を表す。

球面上 \((r=a)\) での境界条件を考えると、球面が流線となるので、

\[
\psi=0
\tag{5}
\]

にすると \(v_r=0\) も満たされる。

また、\(v_\theta=0\) より、

\[
\frac{\partial\psi}{\partial r}=0
\tag{6}
\]

である。

[3] 境界条件 (5)、(6) を満たすように、式 (4) 中の定数 \(A,B\) の値を定めよ。

[4] 上記の結果から、接線応力が、

\[
p_{r\theta}
=
-\frac{3\mu U}{2a}\sin\theta
\]

と求まる。

\(p_{r\theta}\) の \(x\) 方向成分を球面全体で積分して、球に働く摩擦抵抗 \(D_f\) を求めよ。
:::

## Q2-1 Stokes近似の適用条件

### 何を無視した近似か

Navier-Stokes方程式は、

\[
\rho
\frac{\partial\boldsymbol v}{\partial t}
+
\rho
(\boldsymbol v\cdot\nabla)\boldsymbol v
=
-\nabla p+\mu\nabla^2\boldsymbol v.
\]

式 (2) では、非線形な移流慣性項、

\[
\rho
(\boldsymbol v\cdot\nabla)\boldsymbol v
\]

を無視している。

したがって、Stokes近似を使える条件は、

\[
\boxed{
\left|
\rho
(\boldsymbol v\cdot\nabla)\boldsymbol v
\right|
\ll
\left|
\mu\nabla^2\boldsymbol v
\right|
}
\]

である。

### オーダー評価

代表速度を \(U\)、代表長さを \(L\) とする。

移流慣性項の大きさは、

\[
\rho
(\boldsymbol v\cdot\nabla)\boldsymbol v
\sim
\rho\frac{U^2}{L}.
\]

粘性項の大きさは、

\[
\mu\nabla^2\boldsymbol v
\sim
\mu\frac{U}{L^2}.
\]

比を取ると、

\[
\frac{
\rho U^2/L
}{
\mu U/L^2
}
=
\frac{\rho UL}{\mu}
=
Re.
\]

したがって、

\[
\boxed{Re\ll1}
\]

がStokes近似の基本的な適用条件である。

### 非定常項について

式 (2) では、

\[
\rho
\frac{\partial\boldsymbol v}{\partial t}
\]

は残している。

代表時間を \(T\) とすると、

\[
\frac{
\text{移流慣性項}
}{
\text{非定常慣性項}
}
\sim
\frac{U^2/L}{U/T}
=
\frac{UT}{L}.
\]

したがって、移流項を非定常項に対しても無視するには、

\[
\boxed{
\frac{UT}{L}\ll1
}
\]

であることが望ましい。

定常流では非定常項が0なので、単に、

\[
\boxed{Re\ll1}
\]

が重要になる。

### 最終解答

Stokes近似は、

- 代表速度が小さい
- 代表長さが小さい
- 粘性が大きい
- 密度が小さい

などにより、

\[
\boxed{
Re=\frac{\rho UL}{\mu}\ll1
}
\]

となり、移流慣性項が粘性項に比べて無視できるときに適用できる。

:::why title="『遅い流れ』だけでは条件として不十分なの？"
流速が小さくても、代表長さが非常に大きかったり粘性が非常に小さかったりするとReynolds数は大きくなる。Stokes近似の可否は速度だけでなく、\(\rho,U,L,\mu\) の組合せで決まる。
:::

## Q2-2 遠方流れと \(\psi_\infty\)

### 一様流を球座標成分へ分解する

遠方の流れは、

\[
\boldsymbol v_\infty
=
U\boldsymbol e_x.
\]

球座標の単位ベクトルとの関係は、

\[
\boldsymbol e_x
=
\cos\theta\,\boldsymbol e_r
-
\sin\theta\,\boldsymbol e_\theta.
\]

したがって、

\[
\boldsymbol v_\infty
=
U\cos\theta\,\boldsymbol e_r
-
U\sin\theta\,\boldsymbol e_\theta.
\]

よって、

\[
\boxed{
\text{ア}=U\cos\theta
}
\]

\[
\boxed{
\text{イ}=-U\sin\theta
}
\]

である。

:::why title="なぜ vθ に負号が付くの？"
\(\boldsymbol e_\theta\) は \(\theta\) が増える向きである。一様流 \(+x\) は、上半平面では \(\theta\) を小さくする向きに成分を持つため、\(v_\theta\) は負になる。
:::

### 遠方流れのStokes流れ関数

\[
v_r
=
\frac{1}{r^2\sin\theta}
\frac{\partial\psi_\infty}{\partial\theta}
=
U\cos\theta.
\]

したがって、

\[
\frac{\partial\psi_\infty}{\partial\theta}
=
Ur^2\sin\theta\cos\theta.
\]

\(\theta\) で積分する。

\[
\psi_\infty
=
\frac12Ur^2\sin^2\theta
+
F(r).
\]

流れ関数には任意の定数を加えても速度は変わらないので、

\[
F(r)=0
\]

と選べる。

### 最終解答

\[
\boxed{
\psi_\infty(r,\theta)
=
\frac12Ur^2\sin^2\theta
}
\]

である。

### \(v_\theta\) の確認

\[
v_\theta
=
-\frac{1}{r\sin\theta}
\frac{\partial}{\partial r}
\left(
\frac12Ur^2\sin^2\theta
\right).
\]

\[
=
-\frac{1}{r\sin\theta}
Ur\sin^2\theta.
\]

\[
=-U\sin\theta.
\]

遠方境界条件と一致する。

\psi

と書く。

\left(

B=Ua-\frac{A}{a^2}.

\frac12Ua^2+\frac{A}{a}
-

\left(

\right)
=0.

\frac12Ua^2+\frac{A}{a}

## Q2-4 球に働く摩擦抵抗

接線応力は、

\[
p_{r\theta}
=
-\frac{3\mu U}{2a}\sin\theta.
\]

### 接線方向の応力をx方向へ射影する

球座標単位ベクトルについて、

\[
\boldsymbol e_x
=
\cos\theta\,\boldsymbol e_r
-
\sin\theta\,\boldsymbol e_\theta.
\]

したがって、

\[
\boldsymbol e_\theta\cdot\boldsymbol e_x
=
-\sin\theta.
\]

接線応力のx方向成分は、

\[
p_{r\theta}
(\boldsymbol e_\theta\cdot\boldsymbol e_x).
\]

\[
=
\left(
-\frac{3\mu U}{2a}\sin\theta
\right)
(-\sin\theta).
\]

\[
=
\frac{3\mu U}{2a}
\sin^2\theta.
\]

### 球面要素

球面上の面積要素は、

\[
dS
=
a^2\sin\theta\,d\theta\,d\varphi.
\]

したがって、x方向成分の積分は、

\[
D_f
=
\int_0^{2\pi}
\int_0^\pi
\frac{3\mu U}{2a}
\sin^2\theta
a^2\sin\theta
\,d\theta\,d\varphi.
\]

\[
=
\frac{3\mu Ua}{2}
\left(
\int_0^{2\pi}d\varphi
\right)
\left(
\int_0^\pi\sin^3\theta\,d\theta
\right).
\]

\[
\int_0^{2\pi}d\varphi=2\pi
\]

\[
\int_0^\pi\sin^3\theta\,d\theta
=
\frac43.
\]

よって、

\[
D_f
=
\frac{3\mu Ua}{2}
(2\pi)
\left(
\frac43
\right).
\]

### 最終解答

摩擦抵抗の大きさは、

\[
\boxed{
D_f=4\pi\mu Ua
}
\]

である。

球に働く力の向きまで書けば、流れに逆向きなので、

\[
\boxed{
\boldsymbol D_f
=
-4\pi\mu Ua\,\boldsymbol e_x
}
\]

である。

:::why title="有名なStokes抵抗 6πμUa と違うの？"
\(6\pi\mu Ua\) は、接線応力による摩擦抵抗と圧力による形状抵抗を足した全抵抗である。この問題は \(p_{r\theta}\) だけを積分するため、摩擦抵抗 \(4\pi\mu Ua\) を求めている。残りの圧力抵抗は \(2\pi\mu Ua\) である。
:::

## Q3-1 レイノルズ数の定義と意味

### 定義

代表速度を \(U\)、代表長さを \(L\)、動粘性係数を

\[
\nu=\frac{\mu}{\rho}
\]

とする。

レイノルズ数は、

\[
\boxed{
Re
=
\frac{\rho UL}{\mu}
=
\frac{UL}{\nu}
}
\]

である。

### Navier-Stokes方程式の無次元化

非圧縮性流体の運動方程式を、

\[
\rho
\left[
\frac{\partial\boldsymbol v}{\partial t}
+
(\boldsymbol v\cdot\nabla)\boldsymbol v
\right]
=
-\nabla p+\mu\nabla^2\boldsymbol v
\]

とする。

無次元変数を、

\[
\boldsymbol x=L\boldsymbol x^*
\]

\[
\boldsymbol v=U\boldsymbol v^*
\]

\[
t=\frac{L}{U}t^*
\]

\[
p=\rho U^2p^*
\]

と定義する。

各項を代入し、\(\rho U^2/L\) で割ると、

\[
\boxed{
\frac{\partial\boldsymbol v^*}{\partial t^*}
+
(\boldsymbol v^*\cdot\nabla^*)\boldsymbol v^*
=
-\nabla^*p^*
+
\frac1{Re}
\nabla^{*2}\boldsymbol v^*
}
\]

となる。

この式では、慣性項の係数を1に取ると、粘性項の前に \(1/Re\) が付く。

したがって、レイノルズ数は、

\[
\boxed{
Re
\sim
\frac{\text{慣性項}}
{\text{粘性項}}
}
\]

を表す。

### オーダー評価でも確認する

慣性項は、

\[
\rho
(\boldsymbol v\cdot\nabla)\boldsymbol v
\sim
\rho\frac{U^2}{L}.
\]

粘性項は、

\[
\mu\nabla^2\boldsymbol v
\sim
\mu\frac{U}{L^2}.
\]

両者の比は、

\[
\frac{
\rho U^2/L
}{
\mu U/L^2
}
=
\frac{\rho UL}{\mu}
=
Re.
\]

### 物理的な意味

- \(Re\ll1\)
  - 粘性効果が支配的
  - 流れは滑らかで、乱れが減衰しやすい
  - Stokes流が成立しやすい

- \(Re\gg1\)
  - 慣性効果が支配的
  - はく離、渦放出、乱流などが起こりやすい
  - 粘性効果は物体表面近くの境界層に集中する

### 最終解答

\[
\boxed{
Re=\frac{\rho UL}{\mu}=\frac{UL}{\nu}
}
\]

であり、非圧縮性流体の運動方程式における、

\[
\boxed{
\text{慣性力と粘性力の比}
}
\]

を表す。

## Q3-2 円柱後流の変化

円柱直径を \(D\)、一様流速を \(U\) とすると、

\[
Re_D=\frac{\rho UD}{\mu}
\]

を用いる。

レイノルズ数が増えるにつれて、代表的に次のように変化する。

## (a) 極めて小さい \(Re_D\)

\[
Re_D\ll1
\]

- 粘性が支配的なほふく流れ
- 流線は円柱の前後でほぼ対称
- 明瞭なはく離や後流渦はない

模式：

```text
──────→  ○  ──────→
   流線は前後ほぼ対称
```

## (b) 小さいが有限の \(Re_D\)

レイノルズ数が数程度を超えると、

- 円柱後方ではく離が始まる
- 後方に左右対称な一対の定常渦が形成される
- レイノルズ数の増加とともに再循環領域が長くなる

模式：

```text
────→  ○  (↺  ↻)  ───→
          定常な双子渦
```

## (c) 中程度の \(Re_D\)

およそ、

\[
Re_D\simeq47
\]

を超えると、定常な対称後流が不安定になる。

- 上下の渦が交互に放出される
- von Kármán渦列が形成される
- 揚力が周期的に変動する

模式：

```text
────→  ○   ↺      ↻      ↺
            ↻      ↺      ↻
        交互に放出される渦列
```

## (d) さらに大きい \(Re_D\)

- 渦列が三次元化する
- 後流が不規則になる
- 乱流遷移が進む
- 広い乱流後流が形成される

模式：

```text
────→  ○  ≋≈≋≈≋≈≋≈
          不規則な乱流後流
```

## 高レイノルズ数での補足

さらにレイノルズ数が大きくなると、円柱表面の境界層が層流から乱流へ遷移する。

乱流境界層は運動量を多く持ち、はく離が後方へ移るため、圧力抵抗が急減する場合がある。これをdrag crisisという。

### 最終解答

レイノルズ数の増加に伴い、

\[
\boxed{
\text{前後対称なほふく流}
\to
\text{定常な双子渦}
\to
\text{周期的な渦放出}
\to
\text{三次元・乱流後流}
}
\]

へ変化する。

:::why title="なぜレイノルズ数が大きいと渦が発生しやすいの？"
粘性には速度差や乱れを拡散して滑らかにする働きがある。Reynolds数が大きくなると慣性効果が粘性効果より強くなり、はく離したせん断層の不安定性が成長して渦放出へ至る。
:::

## Q3-3 境界層方程式

平板に沿う方向を \(x\)、平板に垂直な方向を \(y\) とする。

速度成分を、

\[
u(x,y)
\]

\[
v(x,y)
\]

とする。

代表的な平板長さを \(L\)、外部流速を \(U\)、境界層厚さを \(\delta\) とする。

定常な二次元非圧縮流を考える。

### 基礎方程式

連続の式は、

\[
\boxed{
\frac{\partial u}{\partial x}
+
\frac{\partial v}{\partial y}
=0
}
\tag{C}
\]

である。

x方向のNavier-Stokes方程式は、

\[
\boxed{
u\frac{\partial u}{\partial x}
+
v\frac{\partial u}{\partial y}
=
-\frac1\rho
\frac{\partial p}{\partial x}
+
\nu
\left(
\frac{\partial^2u}{\partial x^2}
+
\frac{\partial^2u}{\partial y^2}
\right)
}
\tag{Nx}
\]

である。

y方向は、

\[
\boxed{
u\frac{\partial v}{\partial x}
+
v\frac{\partial v}{\partial y}
=
-\frac1\rho
\frac{\partial p}{\partial y}
+
\nu
\left(
\frac{\partial^2v}{\partial x^2}
+
\frac{\partial^2v}{\partial y^2}
\right)
}
\tag{Ny}
\]

である。

---

## Q3-3-(i) 境界層厚さの見積もり

### スケール

境界層内では、

\[
x\sim L
\]

\[
y\sim\delta
\]

\[
u\sim U.
\]

したがって、

\[
\frac{\partial}{\partial x}
\sim
\frac1L
\]

\[
\frac{\partial}{\partial y}
\sim
\frac1\delta.
\]

### 連続の式から \(v\) の大きさを求める

連続の式より、

\[
\frac{U}{L}
\sim
\frac{V}{\delta}.
\]

ここで、\(V\) は境界層内の垂直速度の代表値。

したがって、

\[
\boxed{
V
\sim
U\frac{\delta}{L}
}
\]

である。

境界層が薄い、

\[
\delta\ll L
\]

なら、

\[
V\ll U
\]

である。

:::why title="境界層内で v が u より小さいのはなぜ？"
流れは主に平板に沿って進み、薄い境界層の中で少しずつ外向きに押し出される。連続の式から、垂直速度は \(U\delta/L\) 程度に抑えられる。
:::

### x方向慣性項の大きさ

\[
u\frac{\partial u}{\partial x}
\sim
\frac{U^2}{L}.
\]

また、

\[
v\frac{\partial u}{\partial y}
\sim
\left(
U\frac{\delta}{L}
\right)
\frac{U}{\delta}
=
\frac{U^2}{L}.
\]

したがって、2つの慣性項は同じオーダーである。

### 粘性項の比較

x方向の粘性拡散は、

\[
\nu
\frac{\partial^2u}{\partial x^2}
\sim
\nu\frac{U}{L^2}.
\]

y方向の粘性拡散は、

\[
\nu
\frac{\partial^2u}{\partial y^2}
\sim
\nu\frac{U}{\delta^2}.
\]

\[
\delta\ll L
\]

なら、

\[
\frac{U}{\delta^2}
\gg
\frac{U}{L^2}.
\]

よって、境界層内ではy方向粘性拡散が支配的。

### 慣性項と粘性項の釣り合い

境界層の厚さは、

\[
\frac{U^2}{L}
\sim
\nu\frac{U}{\delta^2}
\]

という釣り合いで決まる。

両辺を \(U\) で割る。

\[
\frac{U}{L}
\sim
\frac{\nu}{\delta^2}.
\]

したがって、

\[
\delta^2
\sim
\frac{\nu L}{U}.
\]

\[
\boxed{
\delta
\sim
\sqrt{\frac{\nu L}{U}}
}
\]

である。

レイノルズ数、

\[
Re_L=\frac{UL}{\nu}
\]

を使うと、

\[
\frac{\delta}{L}
\sim
\sqrt{
\frac{\nu}{UL}
}.
\]

### 最終解答

\[
\boxed{
\frac{\delta}{L}
\sim
Re_L^{-1/2}
}
\]

または、

\[
\boxed{
\delta
\sim
\frac{L}{\sqrt{Re_L}}
}
\]

である。

### 局所的な厚さ

平板先端からの距離を \(x\) とすれば、

\[
Re_x=\frac{Ux}{\nu}
\]

に対して、

\[
\boxed{
\delta(x)
\sim
\sqrt{\frac{\nu x}{U}}
=
\frac{x}{\sqrt{Re_x}}
}
\]

である。

> [!IMPORTANT]
> この式はオーダーの見積もりである。Blasius解で境界層厚さを \(u=0.99U\) の位置と定義すると、係数を含めておよそ \(\delta\simeq5x/\sqrt{Re_x}\) となる。

---

## Q3-3-(ii) 境界層方程式の導出

### 小さなパラメータ

\[
\varepsilon=\frac{\delta}{L}\ll1
\]

と置く。

スケールは、

\[
u\sim U,
\qquad
v\sim\varepsilon U.
\]

### x方向運動方程式

元の式は、

\[
u u_x+v u_y
=
-\frac1\rho p_x
+
\nu(u_{xx}+u_{yy}).
\]

前節のオーダー評価より、

\[
u u_x
\sim
v u_y
\sim
\frac{U^2}{L}.
\]

\[
\nu u_{yy}
\sim
\frac{U^2}{L}.
\]

一方、

\[
\nu u_{xx}
\sim
\nu\frac{U}{L^2}.
\]

境界層関係、

\[
\frac{\nu}{UL}
\sim
\left(
\frac{\delta}{L}
\right)^2
=
\varepsilon^2
\]

を使うと、

\[
\nu u_{xx}
\]

は主要項より \(\varepsilon^2\) 小さい。

したがって、

\[
\nu u_{xx}
\]

を無視できる。

### y方向運動方程式

y方向慣性項は、

\[
u v_x
\sim
\frac{\varepsilon U^2}{L}.
\]

\[
v v_y
\sim
\frac{\varepsilon U^2}{L}.
\]

y方向粘性項も同じく主要なx方向項より小さい。

一方、圧力項は、

\[
\frac1\rho p_y
\]

である。

主要オーダーでは、

\[
\boxed{
\frac{\partial p}{\partial y}=0
}
\]

となる。

したがって、境界層内の圧力は厚さ方向にほぼ一定で、外部流の圧力、

\[
p(x,y)=p_e(x)
\]

と等しい。

:::why title="なぜ境界層内の圧力は外部流と同じなの？"
薄い境界層の厚さ方向には、圧力を大きく変化させるだけの主要な運動項がない。そのため圧力はy方向にほぼ一定となり、境界層の外側の圧力が内部へそのまま伝わる。
:::

### 境界層方程式

主要項だけを残すと、

\[
\boxed{
\frac{\partial u}{\partial x}
+
\frac{\partial v}{\partial y}
=0
}
\]

\[
\boxed{
u\frac{\partial u}{\partial x}
+
v\frac{\partial u}{\partial y}
=
-\frac1\rho
\frac{dp_e}{dx}
+
\nu
\frac{\partial^2u}{\partial y^2}
}
\]

\[
\boxed{
\frac{\partial p}{\partial y}=0
}
\]

を得る。

これが定常二次元非圧縮性流体の境界層方程式である。

### 平板上の一様流

平板外側の流れが一様で、

\[
U_e=\text{一定}
\]

なら、Euler方程式から、

\[
\frac{dp_e}{dx}=0.
\]

したがって、

\[
\boxed{
u\frac{\partial u}{\partial x}
+
v\frac{\partial u}{\partial y}
=
\nu
\frac{\partial^2u}{\partial y^2}
}
\]

となる。

### 境界条件

平板表面 \(y=0\) では、非すべり・非浸透条件より、

\[
\boxed{
u(x,0)=0,
\qquad
v(x,0)=0
}
\]

である。

境界層外縁では、

\[
\boxed{
u(x,y\to\infty)=U_e(x)
}
\]

である。

### 試験答案

\[
x\sim L,\quad y\sim\delta,\quad
u\sim U.
\]

連続の式から、

\[
v\sim U\frac{\delta}{L}.
\]

x方向慣性項とy方向粘性項を釣り合わせると、

\[
\frac{U^2}{L}
\sim
\nu\frac{U}{\delta^2}.
\]

したがって、

\[
\boxed{
\frac{\delta}{L}\sim Re_L^{-1/2}
}.
\]

また、

\[
\delta/L\ll1
\]

より、

\[
u_{xx}\ll u_{yy}
\]

であり、y方向運動方程式の主要項から、

\[
p_y=0.
\]

よって境界層方程式は、

\[
\boxed{
u_x+v_y=0
}
\]

\[
\boxed{
u u_x+v u_y
=
-\frac1\rho\frac{dp_e}{dx}
+\nu u_{yy}
}
\]

\[
\boxed{
p_y=0
}
\]

である。

# 年度全体の重要ポイント

- 二重湧き出しの流線は、原点を通る円群となる。
- 二重湧き出しの循環は0であり、渦流れではない。
- Stokes近似の条件は \(Re\ll1\)。
- 球まわりStokes流の摩擦抵抗は \(4\pi\mu Ua\)、全抵抗は \(6\pi\mu Ua\)。
- レイノルズ数は慣性力と粘性力の比を表す。
- 円柱後流は、定常双子渦から周期的渦放出、さらに乱流後流へ変化する。
- 層流境界層厚さは \(\delta/L\sim Re^{-1/2}\)。
- 境界層方程式では、流れ方向粘性拡散を無視し、圧力は境界層厚さ方向に一定とする。
