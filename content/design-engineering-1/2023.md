# この年度で必要な予備知識

## 1. 動力伝達要素

回転運動と動力を別の軸へ伝える代表的な方法として、次がある。

- 歯車伝動
- ベルト伝動
- チェーン伝動

伝達動力は、回転軸のトルクを \(T\)、角速度を \(\omega\) とすると、

\[
\boxed{P=T\omega}
\]

である。

回転数を \(n\,[\mathrm{min^{-1}}]\) とする場合は、

\[
\omega=\frac{2\pi n}{60}
\]

なので、

\[
\boxed{
P=\frac{2\pi nT}{60}
}
\]

となる。

## 2. ばね定数

力 \(F\) と変形量 \(\delta\) の関係が線形なら、

\[
\boxed{F=K\delta}
\]

である。

ボルト締結体の締付け線図では、

- ボルトのばね定数：\(K_b\)
- 被締結体の圧縮ばね定数：\(K_c\)

を使う。

## 3. 転がり疲れ

転がり軸受では、玉またはころと軌道面の接触部に繰返し接触応力が作用する。

長時間使用すると、軌道面の内部から疲労き裂が発生し、表面がうろこ状にはがれることがある。

これを、

\[
\boxed{\text{フレーキング}}
\]

という。

## 4. 設計における安全率

材料の降伏応力や破壊応力を、そのまま使用限界にはしない。

強度のばらつき、荷重の不確実性、摩耗、衝撃、加工誤差などを考慮して安全率 \(n\) を設ける。

\[
\boxed{
\text{許容応力}
=
\frac{\text{基準となる材料強度}}{\text{安全率}}
}
\]

である。

# Q1-(1) 歯車・平ベルト・チェーンの比較

## 歯車伝動

歯と歯を直接かみ合わせて動力を伝える。

### 長所

1. **滑りがない**

歯が直接かみ合うため、理論上の速度比を正確に保てる。

平歯車では、

\[
\boxed{
\frac{n_1}{n_2}
=
\frac{z_2}{z_1}
}
\]

- \(n_1,n_2\)：回転数
- \(z_1,z_2\)：歯数

である。

2. **大きな動力とトルクを伝えられる**

接触する歯面と歯元で力を受けるため、小型でも大きな力を伝達しやすい。

3. **伝達効率が高い**

適切な潤滑と歯面精度があれば、効率が高い。

4. **コンパクト**

同じ伝達能力なら、ベルト伝動より軸間距離を小さくできる。

5. **さまざまな軸配置に対応できる**

平行軸、交差軸、食違い軸に対応する歯車がある。

### 短所

1. **製作精度が必要で高価**

歯形、ピッチ、歯すじ、取付精度が性能へ大きく影響する。

2. **高速で騒音・振動が発生しやすい**

歯のかみ合い誤差や剛性変化によって、周期的な振動が生じる。

3. **潤滑が必要**

潤滑不足では歯面の摩耗、焼付き、ピッチングが起こる。

4. **大きな軸間距離には不向き**

軸間距離を大きくすると歯車自体も大型化する。

5. **衝撃を逃がしにくい**

過負荷時にも滑らないため、歯や軸へ大きな衝撃が作用する。

---

## 平ベルト伝動

プーリへベルトを巻き付け、ベルトとプーリの摩擦力によって動力を伝える。

### 長所

1. **大きな軸間距離へ対応できる**

離れた軸の間でも、軽いベルトで動力を伝えられる。

2. **構造が簡単で安価**

歯車に比べて製造と保守が容易。

3. **静かで滑らか**

柔軟なベルトが振動と衝撃を吸収する。

4. **過負荷保護作用がある**

過大な負荷ではベルトが滑り、機械の重大な破損を防ぐ場合がある。

5. **高速運転に向く**

ベルトが軽く、比較的高速で使用できる。

### 短所

1. **滑りとクリープがある**

回転速度比を完全には一定に保てない。

2. **張力が軸受荷重となる**

動力を伝えるために初張力が必要であり、軸と軸受へ常時ラジアル荷重が加わる。

3. **設置空間が大きい**

プーリ径と軸間距離が大きくなることがある。

4. **環境の影響を受ける**

油、水、温度、経年劣化で摩擦係数やベルト強度が変化する。

5. **大トルク伝達能力は歯車やチェーンより低い**

同じ寸法なら、確実なかみ合いを使う伝動より伝達能力が低い。

:::why title="平ベルトで速度比が完全に一定にならない理由"
プーリとの間に巨視的な滑りがなくても、ベルトは張力側で伸び、ゆるみ側で縮む。この弾性変形に伴う微小な相対運動をクリープという。
:::

---

## チェーン伝動

スプロケットの歯とチェーンをかみ合わせて動力を伝える。

### 長所

1. **滑りがない**

平均速度比を正確に保てる。

\[
\boxed{
\frac{n_1}{n_2}
=
\frac{z_2}{z_1}
}
\]

2. **ベルトより大きな荷重を伝えやすい**

引張強度の高い金属製チェーンを使用する。

3. **比較的大きな軸間距離に対応できる**

歯車ほど軸間距離に制約されず、ベルトより確実に動力を伝える。

4. **大きな初張力を必要としない**

摩擦力ではなく歯とのかみ合いで伝えるため、ベルトほど大きな初張力は不要。

5. **温度、油、水などへの耐性が比較的高い**

適切な材料と潤滑を選べば、厳しい環境でも使用できる。

### 短所

1. **多角形作用がある**

チェーンはスプロケットへ円ではなく多角形状に巻き付くため、瞬間速度が周期的に変動する。

2. **騒音・振動が発生しやすい**

ローラと歯の衝突、多角形作用、ピッチ誤差の影響を受ける。

3. **潤滑と保守が必要**

ピンとブシュの摩耗によりチェーンのピッチが伸びる。

4. **高速運転に限界がある**

衝撃と遠心力が増え、騒音、摩耗、疲労が問題になる。

5. **取付け精度が必要**

スプロケットの平行度や芯出しが悪いと、偏摩耗や脱落が起こる。

---

## 比較まとめ

| 項目 | 歯車 | 平ベルト | チェーン |
|---|---|---|---|
| 速度比 | 正確 | 滑り・クリープで変動 | 平均的に正確 |
| 軸間距離 | 短い | 長い | 中〜長 |
| 伝達能力 | 大 | 小〜中 | 中〜大 |
| 騒音 | 高速で増えやすい | 小さい | 比較的大きい |
| 衝撃吸収 | 小さい | 大きい | 小〜中 |
| 初張力 | 不要 | 必要 | 小さい |
| 潤滑 | 必要 | 通常不要 | 必要 |
| 過負荷時 | 部品破損のおそれ | 滑って保護可能 | 部品破損のおそれ |

### 試験答案

歯車は滑りがなく速度比が正確で、効率と伝達能力が高く、装置を小型化できる。一方、製作・取付精度と潤滑が必要で、高速では騒音や振動が発生しやすく、長い軸間距離には不向きである。

平ベルトは安価で静かであり、大きな軸間距離に対応し、衝撃吸収と過負荷時の滑りによる保護作用を持つ。一方、滑りとクリープで速度比が変化し、初張力による軸受荷重が大きく、伝達能力も比較的小さい。

チェーンは滑らず、ベルトより大きな荷重を比較的長い軸間距離で伝えられ、初張力も小さい。一方、多角形作用による速度変動・騒音・振動があり、潤滑と張り調整が必要で、高速運転には限界がある。

# Q1-(2) バックラッシと転位歯車

## バックラッシ

一対の歯車をかみ合わせたとき、ピッチ円上で測った、

\[
\boxed{
\text{一方の歯車の歯溝幅}
-
\text{相手歯車の歯厚}
}
\]

を円周方向バックラッシという。

歯面が一方側で接触しているとき、反対側の歯面間に存在するすきまである。

## バックラッシが必要な理由

完全にすきまを0にすると、次の要因により歯面が強く押し合い、回転不能や焼付きが起こる。

- 加工誤差
- 組立誤差
- 軸受の変位
- 歯車と箱の熱膨張
- 歯の弾性変形
- 潤滑油膜の確保

したがって、適切なバックラッシは、

\[
\boxed{
\text{円滑な回転と潤滑を確保するために必要}
}
\]

である。

## バックラッシが大きすぎる場合

- 正逆転時の位置決め誤差
- 歯面の衝突
- 騒音と振動
- 衝撃荷重
- 制御系の不感帯

が増える。

## バックラッシが小さすぎる場合

- 発熱
- 摩擦増加
- 焼付き
- 歯面の早期損傷
- 回転抵抗増加

が起こる。

:::why title="高精度機械ではバックラッシを完全に0にすればよいのでは？"
理想的に0へ近づけたいが、温度変化や加工誤差があるため、単純にすきまをなくすと歯面が拘束される。高精度用途では、ばねで2枚歯車を反対方向へ予圧するなど、機構的にバックラッシを除去する。
:::

---

## 転位歯車

標準ラック工具を歯車中心から半径方向へずらして歯切りした歯車を、

\[
\boxed{\text{転位歯車}}
\]

という。

工具の移動量を、

\[
xm
\]

と表す。

- \(x\)：転位係数
- \(m\)：モジュール

工具を歯車中心から外側へ移動する場合を正転位、

\[
x>0
\]

内側へ移動する場合を負転位、

\[
x<0
\]

という。

## 正転位の主な効果

- 小歯数歯車の切下げを防ぐ
- 歯元の歯厚を増やし、曲げ強度を高める
- 歯先円直径を大きくする
- 歯厚配分を変更できる

## 負転位の主な効果

- 歯元歯厚が減少する
- 歯先円直径が小さくなる
- 大歯車側へ負転位を与え、大小歯車の強度を均等化する場合がある

## 転位歯車を用いる目的

1. **切下げの防止**

標準歯車では歯数が少ないと、創成歯切り時に工具が歯元を削り過ぎる。

正転位により歯元を厚くできる。

2. **歯元強度の調整**

小歯車は大歯車よりかみ合い回数が多く、歯元が弱くなりやすい。

小歯車へ正転位、大歯車へ負転位を与えて強度を配分する。

3. **中心距離の調整**

一対の転位係数の和、

\[
x_1+x_2
\]

を選ぶことで、標準中心距離と異なる中心距離を設定できる。

4. **かみ合い性能の調整**

歯厚、すべり率、歯先干渉、かみ合い率などを調整できる。

### 変わらない基本量

同じ工具を使う限り、

- モジュール
- 基準圧力角
- 基礎ピッチ
- 歯数
- 基準円直径

は基本的に変わらない。

歯形上の使用部分と歯厚、歯先・歯底位置などが変化する。

### 試験答案

バックラッシは、歯車を片側歯面で接触させたときに反対側歯面間へ生じる円周方向のすきまである。加工誤差、熱膨張、潤滑油膜を許容するため必要だが、大きすぎると位置決め誤差、衝撃、騒音が増え、小さすぎると発熱や焼付きが起こる。

転位歯車は、ラック工具を基準位置から \(xm\) だけ半径方向へ移動して歯切りした歯車である。正転位は小歯数歯車の切下げ防止と歯元強化に用いられ、転位係数の組合せによって中心距離や強度配分、かみ合い状態を調整できる。

# Q2-(1) ボルトの締付け線図

## 解答図

![ボルト締結体の締付け線図](/exams/design-engineering-1/2023/bolt-tightening-diagram.svg)

## ボルトと被締結体をばねとして考える

ボルトを締め付けると、

- ボルトは引っ張られて伸びる
- 被締結体は押されて縮む

したがって、ボルトと被締結体を2本のばねとしてモデル化できる。

\[
F_b=K_b\delta_b
\]

\[
F_c=K_c\delta_c
\]

- \(F_b\)：ボルト軸力
- \(F_c\)：被締結体に作用する圧縮締付力
- \(K_b\)：ボルトのばね定数
- \(K_c\)：被締結体の圧縮ばね定数

締付け直後は力のつり合いにより、

\[
\boxed{
F_b=F_c=F_i
}
\]

となる。

\(F_i\) を初期締付力という。

## 締付け線図

横軸へ変形量、縦軸へ力を取る。

- ボルト線：伸びと軸力の関係、傾き \(K_b\)
- 被締結体線：圧縮変形と締付力の関係、傾き \(K_c\)

両直線の交点が初期締付状態を表す。

## 外力が作用する場合

締結体を引き離す方向へ外力 \(W\) が作用すると、外力のすべてがそのままボルトへ加わるわけではない。

- ボルト軸力は増える
- 被締結体の圧縮力は減る

外力係数を、

\[
\boxed{
\Phi
=
\frac{K_b}{K_b+K_c}
}
\]

とすると、単純なばねモデルでは、

\[
\boxed{
\Delta F_b
=
\Phi W
}
\]

\[
\boxed{
\Delta F_c
=
(1-\Phi)W
}
\]

である。

したがって、外力作用後のボルト軸力は、

\[
\boxed{
F_b=F_i+\Phi W
}
\]

残留締付力は、

\[
\boxed{
F_c=F_i-(1-\Phi)W
}
\]

となる。

:::why title="外力 W の全部がボルト軸力の増加にならない理由"
外力が作用すると、ボルトがさらに伸びると同時に、圧縮されていた被締結体が弾性回復する。外力はこの2つの変形へ分担されるため、ボルト軸力の増加は \(\Phi W\) だけになる。
:::

## 接合面の分離

残留締付力が0になると、接合面が分離する。

\[
F_i-(1-\Phi)W_{\mathrm{sep}}=0.
\]

したがって、

\[
\boxed{
W_{\mathrm{sep}}
=
\frac{F_i}{1-\Phi}
=
F_i\frac{K_b+K_c}{K_c}
}
\]

である。

接合面が分離した後は被締結体が圧縮力を負担できないため、それ以降の外力増加はほぼ全てボルト軸力へ加わる。

## 疲労設計上の意味

変動外力 \(W\) に対するボルト軸力の変動は、

\[
\Delta F_b=\Phi W.
\]

したがって、

\[
\boxed{
K_b\text{ を小さくし、}K_c\text{ を大きくすると }\Phi\text{ が小さくなる}
}
\]

ので、ボルトの疲労強度上有利。

具体的には、

- ボルトの有効長を長くする
- 細身ボルトを用いる
- 被締結体を厚く・高剛性にする
- 荷重作用位置を適切にする

などが有効。

## 締付力が小さすぎる場合

- 接合面が分離する
- すべりが起こる
- ボルトへ大きな変動荷重が加わる
- ゆるみや疲労破壊が起こる

## 締付力が大きすぎる場合

- 締付け時にボルトが降伏する
- ねじ山を損傷する
- 被締結体を座屈・陥没させる
- 締付工具や座面を損傷する

### 試験答案

ボルトを引張ばね \(K_b\)、被締結体を圧縮ばね \(K_c\) として表し、力と変形量の関係を同一図上に描く。両線の交点が初期締付力 \(F_i\) を表す。

締結体へ分離方向外力 \(W\) が作用すると、

\[
\Phi=\frac{K_b}{K_b+K_c}
\]

として、

\[
F_b=F_i+\Phi W
\]

\[
F_c=F_i-(1-\Phi)W
\]

となる。

\(F_c=0\) になると接合面が分離し、その後の外力増分はほぼ全てボルトが負担する。したがって、適切な初期締付力を与え、外力係数を小さくすることが疲労強度上重要である。

# Q2-(2) ボールねじの長所と短所

ボールねじは、ねじ軸とナットの間に多数の鋼球を入れ、鋼球を循環させながら転がり接触で運動を伝えるねじ機構である。

回転運動と直線運動を相互に変換する。

## 長所

### 1. 機械効率が高い

滑りねじではねじ面同士が滑るが、ボールねじでは鋼球が転がる。

摩擦係数が小さいため、一般に90%以上の高い効率を得られる。

### 2. 摩擦と発熱が小さい

摩擦損失が少なく、高速送りや連続運転に向く。

### 3. スティックスリップが起こりにくい

静止摩擦と動摩擦の差が小さいため、低速でも滑らかに動き、高精度な位置決めができる。

### 4. バックラッシを小さくできる

2つのナット、オーバーサイズボールなどで予圧を与えれば、軸方向すきまを除去できる。

### 5. 精度と再現性が高い

研削仕上げされたねじでは、工作機械、ロボット、半導体装置などの精密送りに使用できる。

### 6. 摩耗が小さく寿命を予測しやすい

転がり接触なので、適切な潤滑・防塵下では長寿命。

## 短所

### 1. 自己保持性が低い

効率が高いので、軸方向荷重によってねじ軸が逆回転するバックドライブが起こりやすい。

垂直軸では、モータ停止時の落下を防ぐブレーキが必要。

:::why title="高効率なのに自己保持できないのはなぜ？"
摩擦が小さいため、モータからナットを動かしやすいだけでなく、ナット側の荷重からねじ軸も回しやすい。摩擦による逆転防止効果が小さい。
:::

### 2. 高価

ねじ溝と鋼球に高い形状・表面精度が必要で、ボール循環機構も複雑。

### 3. 異物に弱い

切りくずや粉じんが入り込むと、ボールとねじ溝を損傷する。

ワイパ、シール、蛇腹カバーなどが必要。

### 4. 潤滑が必要

潤滑不足では摩耗、発熱、焼付き、寿命低下が起こる。

### 5. 衝撃荷重に弱い

局所的な接触応力が高く、強い衝撃でねじ溝や鋼球に圧痕が生じる。

### 6. 高速回転に限界がある

長いねじ軸では、

- 危険速度による振れ回り
- 遠心力
- ボール循環部の衝撃
- ねじ軸の座屈

を考慮する必要がある。

### 7. 騒音が発生する場合がある

鋼球が循環部へ出入りするときに音と振動が発生する。

### 試験答案

ボールねじは、ねじ軸とナットの間で鋼球を循環させ、転がり接触によって回転運動と直線運動を変換する機構である。

長所は、高効率、低摩擦、低発熱、スティックスリップが少ないこと、予圧によりバックラッシを除去できること、高精度・長寿命であること。

短所は、高価で構造が複雑、潤滑と防塵が必要、衝撃や異物に弱いこと、高速時に危険速度などの制約があること、自己保持性が低く垂直軸ではブレーキが必要なことである。

# Q3-(1) 玉軸受の断面図と部品名

## 解答図

![ラジアル玉軸受とスラスト玉軸受](/exams/design-engineering-1/2023/bearing-cross-sections.svg)

## ラジアル玉軸受

主に軸へ垂直な方向の荷重、

\[
\boxed{\text{ラジアル荷重}}
\]

を支持する。

代表的な部品：

- 内輪
- 外輪
- 玉、転動体
- 保持器、リテーナ
- 内輪軌道溝
- 外輪軌道溝
- 必要に応じてシールまたはシールド

深溝玉軸受では、ある程度のアキシアル荷重も支持できる。

## スラスト玉軸受

主に軸方向の荷重、

\[
\boxed{\text{スラスト荷重、アキシアル荷重}}
\]

を支持する。

代表的な部品：

- 軸軌道盤、軸座金
- ハウジング軌道盤、ハウジング座金
- 玉
- 保持器
- 軌道溝

一般的なスラスト玉軸受はラジアル荷重をほとんど支持できない。

### 試験答案

ラジアル玉軸受は、内輪、外輪、玉、保持器、内外輪の軌道溝からなり、主にラジアル荷重を支持する。

スラスト玉軸受は、軸軌道盤、ハウジング軌道盤、玉、保持器からなり、主に軸方向荷重を支持する。

図中へ各名称を記入する。

# Q3-(2) 転がり軸受の寿命

## 転がり軸受の寿命とは

転がり接触部には、繰返しHertz接触応力が作用する。

長時間の使用により、軌道面または転動体にフレーキングが発生するまでの総回転数を、疲れ寿命として扱う。

同じ仕様で同じ条件の軸受でも、材料中の介在物や微小欠陥により寿命はばらつく。

そこで統計的な寿命を用いる。

## 基本定格寿命 \(L_{10}\)

一群の同一軸受を同じ条件で運転したとき、

\[
\boxed{
90\%\text{ の軸受が転がり疲れによる損傷を起こさずに到達できる寿命}
}
\]

を基本定格寿命 \(L_{10}\) という。

言い換えると、10%が疲労損傷した時点の寿命。

通常、単位は、

\[
\boxed{10^6\text{ 回転}}
\]

で表す。

:::why title="平均寿命ではなく90%残存寿命を使う理由"
軸受寿命は大きくばらつき、平均だけでは多くの軸受がそれ以前に故障する可能性がある。設計では信頼度を明確にした寿命を用いる必要がある。
:::

## 基本動定格荷重 \(C\)

基本動定格荷重とは、

\[
\boxed{
\text{基本定格寿命が }10^6\text{ 回転となる一定荷重}
}
\]

である。

- ラジアル軸受：一定の純ラジアル荷重
- スラスト軸受：一定の純アキシアル荷重

同一形状の軸受群について、90%の信頼度を基準とする。

単位はN。

## 等価動荷重 \(P\)

実際にはラジアル荷重 \(F_r\) とアキシアル荷重 \(F_a\) が同時に作用する場合がある。

これを、軸受寿命へ同じ影響を与える一定荷重へ換算したものが等価動荷重 \(P\)。

一般に、

\[
\boxed{
P=XF_r+YF_a
}
\]

- \(X\)：ラジアル荷重係数
- \(Y\)：アキシアル荷重係数

係数は軸受形式、荷重比、接触角などから選ぶ。

## 寿命式

\[
\boxed{
L_{10}
=
\left(
\frac CP
\right)^p
}
\]

- \(L_{10}\)：基本定格寿命、単位は百万回転
- \(C\)：基本動定格荷重
- \(P\)：等価動荷重
- \(p\)：寿命指数

玉軸受では、

\[
\boxed{p=3}
\]

ころ軸受では、

\[
\boxed{p=\frac{10}{3}}
\]

である。

したがって、玉軸受では、

\[
\boxed{
L_{10}
=
\left(
\frac CP
\right)^3
\times10^6\text{ 回転}
}
\]

となる。

## 時間寿命

回転速度を、

\[
n\,[\mathrm{min^{-1}}]
\]

とすると、

\[
\boxed{
L_{10h}
=
\frac{10^6}{60n}
\left(
\frac CP
\right)^p
}
\]

である。

単位は時間。

## 荷重の影響

玉軸受では寿命が荷重の3乗に反比例する。

例えば荷重を2倍にすると、

\[
\frac{L_{\mathrm{new}}}{L_{\mathrm{old}}}
=
\left(
\frac12
\right)^3
=
\frac18.
\]

したがって、わずかな過負荷でも寿命が大幅に短くなる。

## 実用上の補正

実際の軸受選定では、必要に応じて、

- 信頼度
- 潤滑状態
- 潤滑剤の粘度
- 汚染
- 材料
- 温度
- 取付誤差
- 衝撃荷重

を考慮して修正定格寿命を求める。

### 試験答案

基本動定格荷重 \(C\) は、一群の同一軸受が90%の信頼度で基本定格寿命 \(10^6\) 回転に達する一定荷重である。

実荷重を等価動荷重 \(P\) へ換算し、

\[
\boxed{
L_{10}
=
\left(
\frac CP
\right)^p
}
\]

により、百万回転単位の基本定格寿命を求める。

\[
p=3
\]

は玉軸受、

\[
p=10/3
\]

はころ軸受。

時間寿命は、

\[
\boxed{
L_{10h}
=
\frac{10^6}{60n}
\left(
\frac CP
\right)^p
}
\]

である。

# Q3-(3) 動圧軸受と静圧軸受

いずれも、軸と軸受面の間に油や気体の流体膜を形成して荷重を支持する滑り軸受である。

固体同士を十分に分離できれば、摩擦と摩耗を小さくできる。

## 動圧軸受

軸の回転や相対運動によって、潤滑流体が狭くなるくさび状すきまへ引き込まれる。

粘性流体がくさび内で圧縮されることで圧力が発生し、荷重を支持する。

\[
\boxed{
\text{相対運動によって自ら流体圧力を作る軸受}
}
\]

である。

### 特徴

**長所**

- 外部の高圧ポンプが不要
- 構造が比較的簡単
- 高速時に大きな負荷能力を得られる
- 流体膜による振動減衰性が高い
- 適切な運転状態では摩耗が小さい

**短所**

- 停止中や低速では十分な圧力を作れない
- 始動・停止時に固体接触しやすい
- ある程度の回転速度が必要
- 軸心位置が荷重と速度で変化する
- 油膜不安定振動が生じる場合がある

代表例：ジャーナル軸受、ティルティングパッド軸受。

:::why title="動圧軸受にはくさび形状が必要な理由"
平行なすきまへ流体を運んでも、入口と出口で圧力差を作りにくい。流路が狭くなるくさび形状では、粘性によって運ばれた流体が押し込まれ、荷重を支える圧力分布が形成される。
:::

---

## 静圧軸受

外部ポンプから加圧した流体を軸受面のポケットへ供給し、その圧力で荷重を支持する。

\[
\boxed{
\text{外部圧力源によって流体膜を作る軸受}
}
\]

である。

### 特徴

**長所**

- 軸が停止していても荷重を支持できる
- 始動・停止時にも固体接触を避けやすい
- 摩擦が非常に小さい
- 高い回転精度と位置決め精度
- 低速でも使用できる
- 大きな剛性を得られる設計が可能

**短所**

- ポンプ、配管、絞り、フィルタが必要
- 装置が複雑で高価
- 常時ポンプ動力を消費する
- 漏れ対策が必要
- 異物による絞りやすきまの閉塞に弱い
- 圧力源が停止すると支持能力を失う

代表例：精密工作機械の主軸、超精密位置決め装置。

---

## 違いのまとめ

| 項目 | 動圧軸受 | 静圧軸受 |
|---|---|---|
| 圧力発生 | 軸の相対運動 | 外部ポンプ |
| 停止時の支持 | 原則困難 | 可能 |
| 始動時摩耗 | 起こり得る | 小さい |
| 外部動力 | 原則不要 | ポンプ動力が必要 |
| 構造 | 比較的簡単 | 複雑 |
| 低速性能 | 低い | 高い |
| 精度 | 高いが運転条件に依存 | 非常に高くできる |

### 試験答案

動圧軸受は、軸の回転によって潤滑流体をくさび状すきまへ引き込み、粘性によって発生する動圧で荷重を支持する。外部ポンプが不要で高速・高荷重に向くが、停止時と低速時には油膜を形成できず、始動・停止時に摩耗しやすい。

静圧軸受は、外部ポンプで加圧した流体を軸受すきまへ供給し、その静圧で荷重を支持する。停止時から低摩擦で高精度だが、ポンプと配管が必要で高価であり、圧力源停止時には支持能力を失う。

# Q4-(1) 許容応力と必要軸径

## 許容応力

材料が降伏、破壊、疲労破壊などを起こさず、安全に使用できると定めた応力の上限を許容応力という。

一般に、

\[
\boxed{
[\sigma]
=
\frac{\sigma_{\mathrm{basis}}}{n}
}
\]

\[
\boxed{
[\tau]
=
\frac{\tau_{\mathrm{basis}}}{n}
}
\]

とする。

- \(\sigma_{\mathrm{basis}},\tau_{\mathrm{basis}}\)：降伏強さ、引張強さ、疲労限度など
- \(n\)：安全率
- \([\sigma]\)：許容垂直応力
- \([\tau]\)：許容せん断応力

延性材料の静荷重では降伏応力、脆性材料では破壊応力、繰返し荷重では疲労強度を基準にする。

## 設計の基本手順

1. 荷重と支持条件を求める
2. 軸に作用する軸力、曲げモーメント、トルクを求める
3. 最も危険な断面の最大応力を計算する
4. 応力集中、キー溝、段差、疲労、衝撃を補正する
5. 計算応力が許容応力以下になるように軸径 \(d\) を決める
6. 標準径へ切り上げ、剛性と危険速度も確認する

---

## 軸力 \(F\) を受ける中実丸軸

断面積は、

\[
A=\frac{\pi d^2}{4}.
\]

垂直応力は、

\[
\sigma=\frac FA
=
\frac{4F}{\pi d^2}.
\]

安全条件は、

\[
\sigma\le[\sigma].
\]

したがって、

\[
\boxed{
d
\ge
\sqrt{
\frac{4F}{\pi[\sigma]}
}
}
\]

である。

---

## 曲げモーメント \(M\) を受ける中実丸軸

断面二次モーメントは、

\[
I=\frac{\pi d^4}{64}.
\]

外周までの距離は、

\[
c=\frac d2.
\]

最大曲げ応力は、

\[
\sigma_b
=
\frac{Mc}{I}
=
\frac{32M}{\pi d^3}.
\]

\[
\sigma_b\le[\sigma_b]
\]

より、

\[
\boxed{
d
\ge
\left(
\frac{32M}{\pi[\sigma_b]}
\right)^{1/3}
}
\]

である。

:::why title="曲げを受ける軸径が荷重の立方根になる理由"
丸軸の断面係数は \(Z=\pi d^3/32\) であり、曲げ応力は \(M/Z\) となる。断面の強さが \(d^3\) に比例するため、必要径は曲げモーメントの立方根に比例する。
:::

---

## トルク \(T\) を受ける中実丸軸

極断面二次モーメントは、

\[
J=\frac{\pi d^4}{32}.
\]

最大せん断応力は、

\[
\tau
=
\frac{T(d/2)}J
=
\frac{16T}{\pi d^3}.
\]

\[
\tau\le[\tau]
\]

より、

\[
\boxed{
d
\ge
\left(
\frac{16T}{\pi[\tau]}
\right)^{1/3}
}
\]

である。

---

## 曲げとねじりを同時に受ける場合

最大曲げ応力とねじりせん断応力は、

\[
\sigma_b=\frac{32M}{\pi d^3}
\]

\[
\tau=\frac{16T}{\pi d^3}.
\]

### von Misesの相当応力

延性材料に対して、

\[
\sigma_{\mathrm{eq}}
=
\sqrt{\sigma_b^2+3\tau^2}.
\]

代入すると、

\[
\sigma_{\mathrm{eq}}
=
\frac{16}{\pi d^3}
\sqrt{4M^2+3T^2}.
\]

\[
\sigma_{\mathrm{eq}}
\le[\sigma]
\]

より、

\[
\boxed{
d
\ge
\left[
\frac{
16\sqrt{4M^2+3T^2}
}{
\pi[\sigma]
}
\right]^{1/3}
}
\]

である。

同じ式を、

\[
\boxed{
d
\ge
\left[
\frac{
32\sqrt{M^2+\frac34T^2}
}{
\pi[\sigma]
}
\right]^{1/3}
}
\]

と書いてもよい。

### 最大せん断応力説

Tresca基準を使う場合、

\[
\tau_{\max}
=
\sqrt{
\left(
\frac{\sigma_b}{2}
\right)^2
+\tau^2
}.
\]

代入すると、

\[
\tau_{\max}
=
\frac{16}{\pi d^3}
\sqrt{M^2+T^2}.
\]

したがって、

\[
\boxed{
d
\ge
\left[
\frac{
16\sqrt{M^2+T^2}
}{
\pi[\tau]
}
\right]^{1/3}
}
\]

である。

## 実際の設計で必要な補正

実軸には、

- キー溝
- 段差
- ねじ
- 止め輪溝
- 圧入部
- 横穴

があり、応力集中が生じる。

繰返し曲げや変動トルクでは、

- 疲労限度
- 切欠係数
- 寸法効果
- 表面粗さ
- 平均応力
- 衝撃係数

を考慮する。

また、強度だけでなく、

- ねじれ角
- たわみ
- 軸受の傾き
- 危険速度
- 振動

も確認する。

### 試験答案

許容応力は、材料の降伏・破壊・疲労強度を安全率で割り、安全に使用できる応力上限として定めた値である。

中実丸軸に曲げモーメント \(M\) が作用すると、

\[
\sigma_b=\frac{32M}{\pi d^3}.
\]

したがって、

\[
\boxed{
d\ge
\left(
\frac{32M}{\pi[\sigma_b]}
\right)^{1/3}
}
\]

となる。

トルク \(T\) の場合は、

\[
\tau=\frac{16T}{\pi d^3}
\]

より、

\[
\boxed{
d\ge
\left(
\frac{16T}{\pi[\tau]}
\right)^{1/3}
}.
\]

曲げとねじりが同時に作用する場合は、von Mises応力や最大せん断応力を求め、それが許容応力以下になるように \(d\) を決める。

# Q4-(2) 軸継手と軸ずれ

## 解答図

![軸継手が許容する偏心、偏角、エンドプレイ](/exams/design-engineering-1/2023/shaft-coupling-misalignment.svg)

## 軸継手とは

2本の軸の端部を接続し、一方の軸から他方の軸へ、

- トルク
- 回転運動
- 動力

を伝える機械要素を軸継手、カップリングという。

\[
\boxed{
\text{原動軸と従動軸を接続して動力を伝達する部品}
}
\]

である。

## 軸継手の種類

### 固定軸継手

2軸をほぼ完全に一体化する。

- フランジ形軸継手
- 筒形軸継手

高いねじり剛性を持つが、芯ずれをほとんど許容しない。

### たわみ軸継手

弾性部品、歯、リンク、膜板などの変形によって軸ずれを許容する。

- ゴムカップリング
- 歯車形軸継手
- オルダム継手
- ディスクカップリング
- ベローズカップリング

芯ずれの吸収、振動・衝撃の緩和、組立誤差の許容に使う。

> [!IMPORTANT]
> 軸継手は軸ずれを許容できても、軸受や軸へ生じる追加荷重を完全になくすわけではない。設計値以内に芯出しする必要がある。

---

## 偏心

2本の軸の中心線が平行ではあるが、横方向へ距離 \(e\) だけずれている状態。

\[
\boxed{
\text{平行な2軸の中心線の位置ずれ}
}
\]

である。

英語ではparallel misalignment、offset misalignment。

偏心を許容する軸継手では、1回転ごとに部材が周期的に変形する。

偏心が大きすぎると、

- 軸受反力増加
- カップリングの疲労
- 振動
- 発熱
- シール損傷

が起こる。

---

## 偏角

2本の軸の中心線が同一直線上になく、互いに角度 \(\theta\) を持つ状態。

\[
\boxed{
\text{2軸の中心線の角度誤差}
}
\]

である。

英語ではangular misalignment。

ユニバーサルジョイント、歯車形、ディスク形などは一定の偏角を許容できる。

偏角が大きすぎると、周期的な曲げ、速度変動、振動が生じる。

---

## エンドプレイ

軸方向の相対変位または軸方向すきま。

\[
\boxed{
\text{2軸が互いに軸方向へ近づく、または離れる変位}
}
\]

である。

英語ではaxial displacement、end play。

発生原因：

- 温度変化による軸の熱膨張
- スラスト荷重
- 軸受内部すきま
- 組立誤差
- 機械フレームの変形

軸方向変位を許容しない継手で拘束すると、大きなスラスト力が軸受へ加わる。

:::why title="モータと被駆動機を正確に芯出ししても、エンドプレイ対策が必要な理由"
運転中は軸が発熱して伸び、機械フレームも変形する。停止時に正しく芯出しされていても、運転温度では軸方向位置が変化する可能性がある。
:::

## 3種類のまとめ

\[
\boxed{
\begin{aligned}
\text{偏心：}&
\text{軸線は平行だが横へずれる}\\
\text{偏角：}&
\text{軸線同士に角度がある}\\
\text{エンドプレイ：}&
\text{軸方向へ相対移動する}
\end{aligned}
}
\]

実際には、これらが同時に生じる複合ミスアライメントもある。

### 試験答案

軸継手は、2本の軸を接続してトルクと回転運動を伝える機械要素であり、固定形とたわみ形がある。

偏心は、2軸の中心線が平行なまま横方向へずれる誤差。

偏角は、2軸の中心線が角度を持つ誤差。

エンドプレイは、熱膨張やスラスト荷重などによって2軸が軸方向へ相対移動する誤差。

たわみ軸継手は、これらを弾性変形などで一定範囲まで許容し、軸・軸受へ加わる過大な力を抑える。

# 年度全体の重要ポイント

- 歯車は高精度・高負荷、ベルトは静粛・長軸間、チェーンは確実なかみ合いと中長軸間が特徴。
- バックラッシは円滑な運転に必要だが、大きすぎると位置決め誤差と衝撃を増やす。
- 転位歯車は工具位置をずらして歯切りし、切下げ、強度、中心距離を調整する。
- 締付け外力は、ボルト軸力の増加と被締結体締付力の減少へ分配される。
- ボールねじは高効率・高精度だが、自己保持性が低く、防塵と潤滑が必要。
- 基本動定格荷重 \(C\) は、寿命 \(10^6\) 回転に対応する基準荷重。
- 軸受寿命は \(L_{10}=(C/P)^p\) で評価する。
- 動圧軸受は相対運動、静圧軸受は外部ポンプで圧力を作る。
- 軸径は、計算応力が許容応力以下となるように決める。
- 軸継手は偏心、偏角、エンドプレイを一定範囲で許容する。
