# この年度で必要な予備知識

## 1. マクスウェル方程式

一様な導体中で、自由電荷密度を0とした場合、

\[
\nabla\cdot\boldsymbol E=0,
\qquad
\nabla\cdot\boldsymbol B=0
\]

\[
\nabla\times\boldsymbol E
=
-\frac{\partial\boldsymbol B}{\partial t}
\]

\[
\nabla\times\boldsymbol B
=
\mu\sigma\boldsymbol E
+
\mu\epsilon
\frac{\partial\boldsymbol E}{\partial t}
\]

を用いる。

ここで、

- \(\epsilon\)：導体の誘電率
- \(\mu\)：導体の透磁率
- \(\sigma\)：電気伝導率

である。

## 2. ベクトル恒等式

任意のベクトル場 \(\boldsymbol A\) に対して、

\[
\boxed{
\nabla\times(\nabla\times\boldsymbol A)
=
\nabla(\nabla\cdot\boldsymbol A)
-\nabla^2\boldsymbol A
}
\]

が成り立つ。

## 3. 複素表示

実際の電場を複素数で表し、最後に実部を取る。

\[
\boldsymbol E(\boldsymbol x,t)
=
\operatorname{Re}
\left[
\widetilde{\boldsymbol E}(\boldsymbol x,t)
\right].
\]

例えば、

\[
\widetilde{\boldsymbol E}
=
\boldsymbol E_0 e^{i(kz-\omega t)}
\]

と置けば、微分を掛け算へ変えられる。

\[
\frac{\partial}{\partial z}\longleftrightarrow ik,
\qquad
\frac{\partial}{\partial t}\longleftrightarrow-i\omega.
\]

## 4. 位相速度と群速度

位相速度は、

\[
\boxed{v_{\mathrm p}=\frac{\omega}{k}}
\]

群速度は、

\[
\boxed{v_{\mathrm g}=\frac{d\omega}{dk}}
\]

である。

- 位相速度：同じ位相の点が進む速度
- 群速度：波束やエネルギーが伝わる速度

## 5. 屈折率と誘電率

非磁性体で \(\mu_r\simeq1\) の場合、

\[
\boxed{n^2\simeq\epsilon_r}
\]

である。

吸収がある場合は \(\epsilon_r\) と \(n\) は複素数となるが、透明領域では実部を中心に考えられる。

# 問題1 導体中を進む減衰電磁波

## 問題文（原文）

:::problem-original
以下の式で表される、導体中の \(xyz\) 座標系のマクスウェル方程式を考える。

\[
\nabla\cdot\boldsymbol E(\boldsymbol x,t)=0
\tag{1}
\]

\[
\nabla\cdot\boldsymbol B(\boldsymbol x,t)=0
\tag{2}
\]

\[
\nabla\times\boldsymbol E(\boldsymbol x,t)
=
-\frac{\partial\boldsymbol B(\boldsymbol x,t)}{\partial t}
\tag{3}
\]

\[
\nabla\times\boldsymbol B(\boldsymbol x,t)
=
\mu\sigma\boldsymbol E(\boldsymbol x,t)
+
\mu\epsilon
\frac{\partial\boldsymbol E(\boldsymbol x,t)}{\partial t}
\tag{4}
\]

ここで、

\[
\boldsymbol x=(x,y,z)^T
\]

であり、\(\boldsymbol E,\boldsymbol B\) は、それぞれ、電場、磁場ベクトルを表す。また、\(\epsilon,\mu,\sigma\) は、それぞれ導体の誘電率、透磁率、電気伝導率で全て実数とする。

このマクスウェル方程式の解のうち、電場が次の式で表される電磁波を考える。

\[
\boldsymbol E(\boldsymbol x,t)
=
\boldsymbol E_0e^{-\beta z}
\cos(\alpha z-\omega t),
\qquad
\boldsymbol E_0
=
\frac1{\sqrt2}
\begin{pmatrix}
1\\
-1\\
0
\end{pmatrix}
\tag{5}
\]

ここで、\(\alpha,\beta\) は正の実数とする。以下の問いに答えよ。

1-1. 電場が式 (5) で表される電磁波がどのような偏光かを説明せよ。

1-2. \(k=\alpha+i\beta\) とするとき、式 (5) を \(k\) を用いた複素表示で表せ。

1-3. マクスウェル方程式を変形し、\(\boldsymbol E(\boldsymbol x,t)\) の波動方程式を導け。

1-4. 分散関係式（\(\alpha,\beta\) と \(\omega\) の関係式）は2つの式で表せる。これらの式を求めよ。

1-5. 磁場 \(\boldsymbol B(\boldsymbol x,t)\) の式を \(\alpha,\beta\) を用いて表せ。
:::

## Q1-1 電磁波の偏光

### 波の進行方向

位相が一定の点を考える。

\[
\alpha z-\omega t=\text{一定}.
\]

これを \(z\) について解くと、

\[
z=\frac{\omega}{\alpha}t+\text{一定}.
\]

時間が増えると \(z\) が増えるため、この波は

\[
\boxed{+z\text{ 方向へ進む}}
\]

ことが分かる。

### 電場の向き

電場は常に、

\[
\boldsymbol E_0
=
\frac1{\sqrt2}
\begin{pmatrix}
1\\
-1\\
0
\end{pmatrix}
\]

と同じ向きを向く。

したがって、

\[
E_y=-E_x,
\qquad
E_z=0.
\]

電場ベクトルの先端は、時間が経過しても回転せず、\(xy\) 平面内の1本の直線上を往復する。

### 最終解答

\[
\boxed{\text{直線偏光}}
\]

であり、偏光方向は

\[
\boxed{
\frac1{\sqrt2}
(\boldsymbol e_x-\boldsymbol e_y)
}
\]

である。

すなわち、\(xy\) 平面内で \(+x\) 軸から \(-y\) 方向へ45度傾いた方向の直線偏光である。

また、振幅は

\[
e^{-\beta z}
\]

に従って \(+z\) 方向へ進むにつれて減衰する。

:::why title="円偏光や楕円偏光ではないの？"
円偏光や楕円偏光では、x成分とy成分の間に位相差があり、電場ベクトルの向きが時間とともに回転する。この問題では両成分が同じ \(\cos(\alpha z-\omega t)\) を持つため、向きが固定されている。
:::

### 試験答案

\[
E_y=-E_x,\qquad E_z=0
\]

であり、x成分とy成分の位相差は0である。したがって、電場ベクトルは

\[
\frac1{\sqrt2}(1,-1,0)^T
\]

の方向を往復する。

よって、これは \(xy\) 平面内の直線偏光である。

## Q1-2 複素表示

### 解法方針

\[
k=\alpha+i\beta
\]

とすると、

\[
ikz=i\alpha z-\beta z.
\]

したがって、

\[
e^{i(kz-\omega t)}
=
e^{-\beta z}
e^{i(\alpha z-\omega t)}.
\]

この実部を取れば、

\[
e^{-\beta z}\cos(\alpha z-\omega t)
\]

が得られる。

### 最終解答

複素電場を、

\[
\boxed{
\widetilde{\boldsymbol E}(\boldsymbol x,t)
=
\boldsymbol E_0
e^{i(kz-\omega t)}
}
\]

と置けば、

\[
\boxed{
\boldsymbol E(\boldsymbol x,t)
=
\operatorname{Re}
\left[
\boldsymbol E_0
e^{i(kz-\omega t)}
\right]
}
\]

である。

:::why title="なぜ k=α+iβ で減衰を表せるの？"
指数の中で \(ikz=i\alpha z-\beta z\) となるため、実部 \(\alpha\) は位相の進行を、虚部 \(\beta\) は振幅の指数減衰を表す。
:::

### 重要ポイント

- この符号規約では \(e^{i(kz-\omega t)}\) を用いる。
- \(k=\alpha+i\beta\) なので、\(\beta>0\) なら \(+z\) 方向へ減衰する。

## Q1-3 導体中の電場の波動方程式

### Step 1：Faradayの法則の両辺に curl を取る

\[
\nabla\times\boldsymbol E
=
-\frac{\partial\boldsymbol B}{\partial t}.
\]

両辺に \(\nabla\times\) を作用させる。

\[
\nabla\times
(\nabla\times\boldsymbol E)
=
-\frac{\partial}{\partial t}
(\nabla\times\boldsymbol B).
\]

### Step 2：左辺をベクトル恒等式で変形する

\[
\nabla\times
(\nabla\times\boldsymbol E)
=
\nabla(\nabla\cdot\boldsymbol E)
-\nabla^2\boldsymbol E.
\]

\[
\nabla\cdot\boldsymbol E=0
\]

なので、

\[
\nabla\times
(\nabla\times\boldsymbol E)
=
-\nabla^2\boldsymbol E.
\]

### Step 3：右辺へAmpère-Maxwellの法則を代入する

\[
\nabla\times\boldsymbol B
=
\mu\sigma\boldsymbol E
+
\mu\epsilon
\frac{\partial\boldsymbol E}{\partial t}.
\]

したがって、

\[
-\frac{\partial}{\partial t}
(\nabla\times\boldsymbol B)
=
-\mu\sigma
\frac{\partial\boldsymbol E}{\partial t}
-\mu\epsilon
\frac{\partial^2\boldsymbol E}{\partial t^2}.
\]

### Step 4：両辺を整理する

\[
-\nabla^2\boldsymbol E
=
-\mu\sigma
\frac{\partial\boldsymbol E}{\partial t}
-\mu\epsilon
\frac{\partial^2\boldsymbol E}{\partial t^2}.
\]

両辺へ \(-1\) を掛けると、

### 最終解答

\[
\boxed{
\nabla^2\boldsymbol E
=
\mu\sigma
\frac{\partial\boldsymbol E}{\partial t}
+
\mu\epsilon
\frac{\partial^2\boldsymbol E}{\partial t^2}
}
\]

または、

\[
\boxed{
\nabla^2\boldsymbol E
-\mu\epsilon
\frac{\partial^2\boldsymbol E}{\partial t^2}
-\mu\sigma
\frac{\partial\boldsymbol E}{\partial t}
=0
}
\]

である。

:::why title="真空中の波動方程式と何が違うの？"
導体では電流密度 \(\boldsymbol J=\sigma\boldsymbol E\) が生じる。そのため、波動方程式に1階時間微分の項 \(\mu\sigma\partial_t\boldsymbol E\) が加わり、電磁波のエネルギーがジュール熱として失われて減衰する。
:::

### 試験答案

\[
\nabla\times(\nabla\times\boldsymbol E)
=
-\frac{\partial}{\partial t}
(\nabla\times\boldsymbol B).
\]

\[
\nabla\times(\nabla\times\boldsymbol E)
=
\nabla(\nabla\cdot\boldsymbol E)-\nabla^2\boldsymbol E
=
-\nabla^2\boldsymbol E.
\]

一方、

\[
-\partial_t(\nabla\times\boldsymbol B)
=
-\mu\sigma\partial_t\boldsymbol E
-\mu\epsilon\partial_t^2\boldsymbol E.
\]

よって、

\[
\boxed{
\nabla^2\boldsymbol E
=
\mu\sigma\partial_t\boldsymbol E
+\mu\epsilon\partial_t^2\boldsymbol E
}.
\]

## Q1-4 分散関係式

### 複素電場を波動方程式へ代入する

\[
\widetilde{\boldsymbol E}
=
\boldsymbol E_0
e^{i(kz-\omega t)}.
\]

z方向だけに依存するため、

\[
\nabla^2\widetilde{\boldsymbol E}
=
\frac{\partial^2\widetilde{\boldsymbol E}}{\partial z^2}
=
-k^2\widetilde{\boldsymbol E}.
\]

また、

\[
\frac{\partial\widetilde{\boldsymbol E}}{\partial t}
=
-i\omega\widetilde{\boldsymbol E}
\]

\[
\frac{\partial^2\widetilde{\boldsymbol E}}{\partial t^2}
=
-\omega^2\widetilde{\boldsymbol E}.
\]

波動方程式へ代入すると、

\[
-k^2\widetilde{\boldsymbol E}
=
-i\mu\sigma\omega\widetilde{\boldsymbol E}
-\mu\epsilon\omega^2\widetilde{\boldsymbol E}.
\]

したがって、

\[
\boxed{
k^2
=
\mu\epsilon\omega^2+i\mu\sigma\omega
}
\]

である。

### \(k=\alpha+i\beta\) を代入する

\[
k^2
=
(\alpha+i\beta)^2
=
\alpha^2-\beta^2+2i\alpha\beta.
\]

一方、

\[
\mu\epsilon\omega^2+i\mu\sigma\omega
\]

の実部と虚部を比較する。

### 最終解答

\[
\boxed{
\alpha^2-\beta^2
=
\mu\epsilon\omega^2
}
\]

\[
\boxed{
2\alpha\beta
=
\mu\sigma\omega
}
\]

である。

:::why title="なぜ実部と虚部を別々に比較できるの？"
複素数 \(a+ib=c+id\) が等しいためには、実部 \(a=c\) と虚部 \(b=d\) がそれぞれ等しくなければならないからである。
:::

### 参考：\(\alpha,\beta\) を解いた形

2式から、

\[
\alpha^2+\beta^2
=
\mu\omega
\sqrt{\epsilon^2\omega^2+\sigma^2}.
\]

したがって、

\[
\boxed{
\alpha
=
\omega
\sqrt{
\frac{\mu\epsilon}{2}
\left[
\sqrt{
1+
\left(
\frac{\sigma}{\epsilon\omega}
\right)^2
}
+1
\right]
}
}
\]

\[
\boxed{
\beta
=
\omega
\sqrt{
\frac{\mu\epsilon}{2}
\left[
\sqrt{
1+
\left(
\frac{\sigma}{\epsilon\omega}
\right)^2
}
-1
\right]
}
}
\]

である。

## Q1-5 磁場の式

### 複素表示で求める

Faradayの法則は、

\[
\nabla\times\widetilde{\boldsymbol E}
=
-\frac{\partial\widetilde{\boldsymbol B}}{\partial t}.
\]

平面波では、

\[
\nabla\times\widetilde{\boldsymbol E}
=
ik\boldsymbol e_z
\times
\widetilde{\boldsymbol E}.
\]

また、

\[
-\frac{\partial\widetilde{\boldsymbol B}}{\partial t}
=
i\omega\widetilde{\boldsymbol B}.
\]

したがって、

\[
ik\boldsymbol e_z
\times
\widetilde{\boldsymbol E}
=
i\omega\widetilde{\boldsymbol B}.
\]

よって、

\[
\boxed{
\widetilde{\boldsymbol B}
=
\frac{k}{\omega}
\boldsymbol e_z
\times
\widetilde{\boldsymbol E}
}
\]

である。

### 偏光方向を計算する

\[
\boldsymbol e_z
\times
\boldsymbol E_0
=
\frac1{\sqrt2}
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}.
\]

したがって、

\[
\widetilde{\boldsymbol B}
=
\frac{\alpha+i\beta}{\omega}
\frac1{\sqrt2}
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}
e^{-\beta z}
e^{i(\alpha z-\omega t)}.
\]

実部を取る。

\[
\operatorname{Re}
\left[
(\alpha+i\beta)
e^{i\phi}
\right]
=
\alpha\cos\phi-\beta\sin\phi
\]

ただし、

\[
\phi=\alpha z-\omega t.
\]

### 最終解答

\[
\boxed{
\boldsymbol B(\boldsymbol x,t)
=
\frac{e^{-\beta z}}{\sqrt2\,\omega}
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}
\left[
\alpha\cos(\alpha z-\omega t)
-
\beta\sin(\alpha z-\omega t)
\right]
}
\]

である。

別の表現として、

\[
\delta=\tan^{-1}\left(\frac{\beta}{\alpha}\right)
\]

と置けば、

\[
\boxed{
\boldsymbol B
=
\frac{\sqrt{\alpha^2+\beta^2}}{\sqrt2\,\omega}
e^{-\beta z}
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}
\cos(\alpha z-\omega t+\delta)
}
\]

とも書ける。

### 向きの確認

\[
\boldsymbol E_0
\cdot
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}
=0.
\]

したがって、電場と磁場は互いに直交する。

また、

\[
\boldsymbol E_0
\times
\left[
\frac1{\sqrt2}
\begin{pmatrix}
1\\
1\\
0
\end{pmatrix}
\right]
=
\boldsymbol e_z.
\]

よって、エネルギーの流れる向きは \(+z\) 方向である。

:::why title="導体中では E と B に位相差があるの？"
複素波数 \(k=\alpha+i\beta\) が磁場の係数に掛かるため、磁場には余弦項だけでなく正弦項も現れる。これは導体中で電場と磁場の位相が完全には一致しないことを表す。
:::

# 問題2 ベクトル解析・導波管・正常分散

## 問題文（原文）

:::problem-original
以下の問いに答えよ。

2-1. \(\boldsymbol x,\boldsymbol x'\;(\boldsymbol x\ne\boldsymbol x')\) を \(xyz\) 座標系の位置ベクトルとするとき、以下の式が成り立つことを示せ。ただし、\(\nabla'\) は \(\boldsymbol x'\) についての微分を表す。

\[
\frac{\boldsymbol x-\boldsymbol x'}
{|\boldsymbol x-\boldsymbol x'|^3}
=
\nabla'
\left(
\frac1{|\boldsymbol x-\boldsymbol x'|}
\right)
\tag{6}
\]

2-2. \(z\) 方向に無限に長く、内部が真空の矩形導波管について、TEモード電磁波の分散関係式は以下の式で表せる。

\[
k^2
=
\mu_0\epsilon_0
(\omega^2-\omega_c^2)
\tag{7}
\]

ここで、\(k\) は波数の \(z\) 成分、\(\omega_c\) は遮断角周波数である。この電磁波の位相速度と群速度の式を書き、光速 \(c\) との大小関係を説明せよ。

2-3. 以下のローレンツモデルの比誘電率の式を用いて、誘電体の屈折率が正常分散になる場合の、屈折率の (1) 電磁波の角周波数、(2) 誘電体の密度、に対する依存性を説明せよ。

\[
\frac{\epsilon(\omega)}{\epsilon_0}
=
1+
\frac{Ne^2}{m_e\epsilon_0}
\left(
\frac{f}
{\omega_0^2-\omega^2-i\gamma\omega}
\right)
\tag{8}
\]

ここで、\(N\) は電子の数密度、\(e\) は素電荷、\(m_e\) は電子の質量、\(\omega_0\) は電子の固有角周波数の1つ、\(f\) は \(\omega_0\) を持つ電子の割合、\(\gamma\) は摩擦係数で正の実数である。

## Q2-1 クーロン核の勾配

### 解法方針

差のベクトルを、

\[
\boldsymbol r
=
\boldsymbol x-\boldsymbol x'
\]

と置く。

\[
r=|\boldsymbol r|
=
|\boldsymbol x-\boldsymbol x'|.
\]

ただし、微分は \(\boldsymbol x'\) について行う。

### 成分表示

\[
r
=
\sqrt{
(x-x')^2+
(y-y')^2+
(z-z')^2
}.
\]

まずx'成分を計算する。

\[
\frac{\partial}{\partial x'}
\left(
\frac1r
\right)
=
-\frac1{r^2}
\frac{\partial r}{\partial x'}.
\]

\[
\frac{\partial r}{\partial x'}
=
\frac{x'-x}{r}.
\]

したがって、

\[
\frac{\partial}{\partial x'}
\left(
\frac1r
\right)
=
-\frac1{r^2}
\frac{x'-x}{r}
=
\frac{x-x'}{r^3}.
\]

同様に、

\[
\frac{\partial}{\partial y'}
\left(
\frac1r
\right)
=
\frac{y-y'}{r^3}
\]

\[
\frac{\partial}{\partial z'}
\left(
\frac1r
\right)
=
\frac{z-z'}{r^3}.
\]

よって、

### 最終解答

\[
\boxed{
\nabla'
\left(
\frac1{|\boldsymbol x-\boldsymbol x'|}
\right)
=
\frac{\boldsymbol x-\boldsymbol x'}
{|\boldsymbol x-\boldsymbol x'|^3}
}
\]

である。

:::why title="x で微分するときと符号が逆になるの？"
\(\boldsymbol x-\boldsymbol x'\) は、\(\boldsymbol x'\) で微分すると \(-1\) が出る。さらに \(1/r\) の微分でも \(-1\) が出るため、2つの負号が打ち消し合って正になる。
:::

### 比較

\(\boldsymbol x\) について微分する場合は、

\[
\nabla
\left(
\frac1{|\boldsymbol x-\boldsymbol x'|}
\right)
=
-\frac{\boldsymbol x-\boldsymbol x'}
{|\boldsymbol x-\boldsymbol x'|^3}.
\]

したがって、

\[
\nabla'=-\nabla
\]

の関係が見える。

## Q2-2 矩形導波管の位相速度と群速度

### 分散関係式を光速で書く

真空中では、

\[
\mu_0\epsilon_0=\frac1{c^2}.
\]

したがって、

\[
k^2
=
\frac1{c^2}
(\omega^2-\omega_c^2).
\]

伝搬するためには、

\[
\omega>\omega_c
\]

が必要である。

\[
\boxed{
k
=
\frac1c
\sqrt{\omega^2-\omega_c^2}
}
\]

である。

### 位相速度

\[
v_{\mathrm p}
=
\frac{\omega}{k}.
\]

したがって、

\[
v_{\mathrm p}
=
\frac{c\omega}
{\sqrt{\omega^2-\omega_c^2}}.
\]

\[
\boxed{
v_{\mathrm p}
=
\frac{c}
{\sqrt{
1-(\omega_c/\omega)^2
}}
}
\]

である。

\[
0<
1-\left(\frac{\omega_c}{\omega}\right)^2
<1
\]

なので、

\[
\boxed{v_{\mathrm p}>c}.
\]

:::why title="位相速度が光速を超えても相対論に反しないの？"
位相速度は波の山や谷の位置が進む速度であり、情報やエネルギーそのものの伝搬速度ではない。情報伝達に関係する群速度は光速未満になる。
:::

### 群速度

分散関係を、

\[
\omega^2=c^2k^2+\omega_c^2
\]

と書く。

両辺を \(k\) で微分する。

\[
2\omega\frac{d\omega}{dk}
=
2c^2k.
\]

したがって、

\[
v_{\mathrm g}
=
\frac{d\omega}{dk}
=
\frac{c^2k}{\omega}.
\]

\(k\) を代入すると、

\[
\boxed{
v_{\mathrm g}
=
c
\sqrt{
1-\left(
\frac{\omega_c}{\omega}
\right)^2
}
}
\]

である。

したがって、

\[
\boxed{v_{\mathrm g}<c}.
\]

また、

\[
\boxed{
v_{\mathrm p}v_{\mathrm g}=c^2
}
\]

である。

### 遮断周波数付近

\[
\omega\to\omega_c+0
\]

では、

\[
v_{\mathrm p}\to\infty
\]

\[
v_{\mathrm g}\to0.
\]

### 高周波極限

\[
\omega\gg\omega_c
\]

では、

\[
v_{\mathrm p}\to c,
\qquad
v_{\mathrm g}\to c.
\]

### 試験答案

\[
k
=
\frac1c\sqrt{\omega^2-\omega_c^2}.
\]

したがって、

\[
\boxed{
v_{\mathrm p}
=
\frac{\omega}{k}
=
\frac{c}
{\sqrt{1-(\omega_c/\omega)^2}}
>c
}
\]

である。

また、

\[
\omega^2=c^2k^2+\omega_c^2
\]

を微分して、

\[
\boxed{
v_{\mathrm g}
=
\frac{d\omega}{dk}
=
\frac{c^2k}{\omega}
=
c\sqrt{1-(\omega_c/\omega)^2}
<c
}.
\]

さらに、

\[
v_{\mathrm p}v_{\mathrm g}=c^2.
\]

## Q2-3 ローレンツモデルと正常分散

### 比誘電率

\[
\epsilon_r(\omega)
=
\frac{\epsilon(\omega)}{\epsilon_0}.
\]

定数を、

\[
A
=
\frac{Ne^2f}{m_e\epsilon_0}
\]

と置くと、

\[
\epsilon_r(\omega)
=
1+
\frac{A}
{\omega_0^2-\omega^2-i\gamma\omega}.
\]

### 実部と虚部

\[
\Delta=\omega_0^2-\omega^2
\]

と置く。

\[
\frac1{\Delta-i\gamma\omega}
=
\frac{\Delta+i\gamma\omega}
{\Delta^2+\gamma^2\omega^2}.
\]

したがって、

\[
\operatorname{Re}\epsilon_r
=
1+
A
\frac{\Delta}
{\Delta^2+\gamma^2\omega^2}
\]

\[
\operatorname{Im}\epsilon_r
=
A
\frac{\gamma\omega}
{\Delta^2+\gamma^2\omega^2}.
\]

虚部は吸収を表す。

正常分散を考える透明領域では、共鳴から十分離れており吸収が小さいので、

\[
\gamma\omega
\ll
|\omega_0^2-\omega^2|
\]

と考えられる。

このとき、

\[
\epsilon_r
\simeq
1+
\frac{A}{\omega_0^2-\omega^2}.
\]

非磁性体なら、

\[
n^2\simeq\epsilon_r.
\]

---

### (1) 角周波数への依存性

共鳴より低い透明領域

\[
\omega<\omega_0
\]

を考える。

\[
n^2
\simeq
1+
\frac{A}{\omega_0^2-\omega^2}.
\]

右辺を \(\omega\) で微分すると、

\[
\frac{d(n^2)}{d\omega}
=
\frac{2A\omega}
{(\omega_0^2-\omega^2)^2}
>0.
\]

\(n>0\) なので、

\[
2n\frac{dn}{d\omega}
=
\frac{d(n^2)}{d\omega}.
\]

したがって、

\[
\boxed{
\frac{dn}{d\omega}>0
}
\]

である。

つまり、正常分散では、

\[
\boxed{
\text{角周波数 }\omega\text{ が大きくなるほど、屈折率 }n\text{ は大きくなる}
}
\]

と説明できる。

波長 \(\lambda\) は周波数と逆の関係にあるので、

\[
\boxed{
\text{波長が長くなるほど、屈折率は小さくなる}
}
\]

とも言える。

:::why title="共鳴の近くを除く必要があるの？"
共鳴周波数付近では吸収が強くなり、屈折率の変化が正常分散とは逆になる異常分散領域が現れる場合がある。この問題では「正常分散になる場合」なので、吸収の小さい領域を考える。
:::

---

### (2) 誘電体の密度への依存性

電子の数密度 \(N\) は、誘電体の物質密度にほぼ比例する。

\[
A=
\frac{Ne^2f}{m_e\epsilon_0}
\]

なので、\(\epsilon_r-1\) は \(N\) に比例する。

共鳴より低い透明領域では、

\[
\omega_0^2-\omega^2>0.
\]

したがって、

\[
\frac{\partial(n^2)}{\partial N}
=
\frac{e^2f}
{m_e\epsilon_0(\omega_0^2-\omega^2)}
>0.
\]

よって、

\[
\boxed{
\text{誘電体の密度が大きいほど、屈折率は大きくなる}
}
\]

と説明できる。

低密度で

\[
|n-1|\ll1
\]

なら、

\[
n
=
\sqrt{1+CN}
\simeq
1+\frac{CN}{2}
\]

なので、

\[
\boxed{n-1\propto N}
\]

と近似できる。

### より厳密な注意

実部を厳密に使うと、

\[
\frac{\partial\operatorname{Re}\epsilon_r}{\partial N}
=
\frac{e^2f}{m_e\epsilon_0}
\frac{\omega_0^2-\omega^2}
{(\omega_0^2-\omega^2)^2+\gamma^2\omega^2}.
\]

したがって、密度増加による屈折率の増減の符号は共鳴のどちら側かにも依存する。

ただし、通常の透明な正常分散領域として共鳴より低い側を考えれば、密度が増えるほど屈折率は増える。

### 試験答案

共鳴から十分離れた透明領域では、

\[
n^2\simeq\epsilon_r
\simeq
1+
\frac{Ne^2f}
{m_e\epsilon_0(\omega_0^2-\omega^2)}.
\]

共鳴より低い正常分散領域 \(\omega<\omega_0\) では、

\[
\frac{d(n^2)}{d\omega}
=
\frac{2Ne^2f\omega}
{m_e\epsilon_0(\omega_0^2-\omega^2)^2}
>0.
\]

したがって、

\[
\boxed{\frac{dn}{d\omega}>0}
\]

であり、周波数が増えると屈折率も増える。

また、

\[
n^2-1\propto N
\]

なので、

\[
\boxed{
\text{電子数密度、したがって誘電体の密度が増えると屈折率は増える}
}
\]

と分かる。

# 年度全体の重要ポイント

- 複素波数の実部 \(\alpha\) は位相定数、虚部 \(\beta\) は減衰定数である。
- 導体中の波動方程式には、電気伝導による1階時間微分項が現れる。
- 分散関係では、複素数の実部と虚部を分けて比較する。
- 導体中では、電場と磁場に位相差が生じる。
- 導波管では位相速度が光速を超えるが、群速度は光速を超えない。
- ローレンツモデルでは、共鳴付近の吸収と分散を同時に説明できる。
