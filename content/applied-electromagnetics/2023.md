# この年度で必要な予備知識

## 1. 真空中のマクスウェル方程式

自由電荷と電流がない真空中では、

\[
\nabla\cdot\boldsymbol E=0,
\qquad
\nabla\cdot\boldsymbol B=0
\]

\[
\nabla\times\boldsymbol E
=
-\frac{\partial\boldsymbol B}{\partial t}
\]

\[
\nabla\times\boldsymbol B
=
\mu_0\epsilon_0
\frac{\partial\boldsymbol E}{\partial t}
\]

が成り立つ。

真空中の光速は、

\[
\boxed{
c=\frac1{\sqrt{\mu_0\epsilon_0}}
}
\]

である。

## 2. 平面電磁波

\[
\boldsymbol E
=
\boldsymbol E_0\cos(\boldsymbol k\cdot\boldsymbol x-\omega t)
\]

の形の波を考える。

真空中の平面波では、

\[
\boxed{
\boldsymbol k\perp\boldsymbol E,
\qquad
\boldsymbol k\perp\boldsymbol B,
\qquad
\boldsymbol E\perp\boldsymbol B
}
\]

であり、

\[
\boxed{
\boldsymbol B
=
\frac1\omega
\boldsymbol k\times\boldsymbol E
}
\]

が成り立つ。

## 3. ポインティングベクトル

真空中での電磁エネルギーの流れを表すベクトルは、

\[
\boxed{
\boldsymbol S
=
\frac1{\mu_0}
\boldsymbol E\times\boldsymbol B
}
\]

である。

正弦波について、

\[
\left\langle\cos^2(\cdots)\right\rangle_t
=
\frac12
\]

なので、時間平均を求めるときに用いる。

## 4. 動径の微分

\[
r=|\boldsymbol x|
=
\sqrt{x^2+y^2+z^2}
\]

とすると、

\[
\boxed{
\nabla r=\frac{\boldsymbol x}{r}
}
\]

である。

## 5. 位相速度と群速度

位相速度は、

\[
\boxed{
v_{\mathrm p}=\frac{\omega}{k}
}
\]

群速度は、

\[
\boxed{
v_{\mathrm g}=\frac{d\omega}{dk}
}
\]

である。

## 6. 電力透過率

媒質1から媒質2へ平面波が入射するとき、電場振幅透過係数を

\[
t_E=\frac{E_0''}{E_0}
\]

とすると、電力透過率は、

\[
\boxed{
T
=
\frac{n_2\cos\theta_2}
{n_1\cos\theta_1}
|t_E|^2
}
\]

で与えられる。

# 問題1 真空中の平面電磁波

## 問題文（原文）

:::problem-original
以下の式で表される、真空中の \(xyz\) 座標系のマクスウェル方程式を考える。

\[
\nabla\cdot\boldsymbol E(\boldsymbol x,t)=0
\tag{1}
\]

\[
\nabla\cdot\boldsymbol B(\boldsymbol x,t)=0
\tag{2}
\]

\[
\nabla\times\boldsymbol E(\boldsymbol x,t)
=
-\frac{\partial\boldsymbol B(\boldsymbol x,t)}{\partial t}
\tag{3}
\]

\[
\nabla\times\boldsymbol B(\boldsymbol x,t)
=
\mu_0\epsilon_0
\frac{\partial\boldsymbol E(\boldsymbol x,t)}{\partial t}
\tag{4}
\]

ここで、

\[
\boldsymbol x=(x,y,z)^T
\]

であり、\(\boldsymbol E,\boldsymbol B\) は、それぞれ、電場、磁場ベクトルを表す。

このマクスウェル方程式の解のうち、電場が次の式で表される平面電磁波について、以下の問いに答えよ。

\[
\boldsymbol E(\boldsymbol x,t)
=
E_0
\begin{pmatrix}
1/2\\
\sqrt3/2\\
0
\end{pmatrix}
\cos(kz-\omega t)
\tag{5}
\]

ここで、\(E_0\) は実定数であり、\(k,\omega\) は正の実数とする。解答は答えのみでなく計算過程も記すこと。

1-1. \(z=0\) の \(xy\) 平面上における電場の \(x,y\) 成分の大きさを、それぞれ \(E_x,E_y\) と表す。式 (5) の電場が \(E_xE_y\) 平面上でどのような軌跡を描くかを図示し、題意の平面電磁波がどのような偏光かを説明せよ。

1-2. \(k\) を \(\omega\) を用いた式で表せ。

1-3. 磁場の式を求めよ。ただし、磁場の定数成分はゼロであるとする。

1-4. 時間平均したポインティングベクトルの式を求めよ。ベクトルの大きさだけでなく向きも記すこと。
:::

## Q1-1 平面電磁波の偏光

### \(z=0\) における電場成分

\[
\cos(kz-\omega t)
\]

へ \(z=0\) を代入する。

\[
\cos(-\omega t)=\cos\omega t
\]

なので、

\[
E_x(t)
=
\frac{E_0}{2}\cos\omega t
\]

\[
E_y(t)
=
\frac{\sqrt3E_0}{2}\cos\omega t.
\]

したがって、

\[
\boxed{
E_y=\sqrt3E_x
}
\]

である。

### \(E_xE_y\) 平面上の軌跡

電場ベクトルの先端は、

\[
E_y=\sqrt3E_x
\]

という原点を通る直線上を往復する。

成分の最大値は、

\[
|E_x|\le\frac{|E_0|}{2}
\]

\[
|E_y|\le\frac{\sqrt3|E_0|}{2}
\]

である。

電場の向きは、

\[
\boldsymbol e_{\mathrm pol}
=
\begin{pmatrix}
1/2\\
\sqrt3/2\\
0
\end{pmatrix}.
\]

x軸との角度を \(\vartheta\) とすると、

\[
\tan\vartheta
=
\frac{\sqrt3/2}{1/2}
=
\sqrt3.
\]

よって、

\[
\vartheta=\frac{\pi}{3}=60^\circ.
\]

### 最終解答

\[
\boxed{\text{直線偏光}}
\]

であり、偏光方向は、

\[
\boxed{
\boldsymbol e_{\mathrm pol}
=
\frac12\boldsymbol e_x
+
\frac{\sqrt3}{2}\boldsymbol e_y
}
\]

すなわち、x軸から反時計回りに60度の方向である。

:::why title="なぜ円偏光や楕円偏光ではないの？"
x成分とy成分が同じ \(\cos\omega t\) を持ち、両者の位相差が0だからである。成分比が常に一定なので、電場の向きは回転しない。
:::

### 試験答案

\[
E_x=\frac{E_0}{2}\cos\omega t,
\qquad
E_y=\frac{\sqrt3E_0}{2}\cos\omega t.
\]

したがって、

\[
E_y=\sqrt3E_x.
\]

よって、電場の先端は原点を通りx軸となす角が60度の直線上を往復するため、直線偏光である。

## Q1-2 \(k\) と \(\omega\) の関係

### 電場の波動方程式

Faradayの法則の両辺に curl を取る。

\[
\nabla\times(\nabla\times\boldsymbol E)
=
-\frac{\partial}{\partial t}
(\nabla\times\boldsymbol B).
\]

ベクトル恒等式、

\[
\nabla\times(\nabla\times\boldsymbol E)
=
\nabla(\nabla\cdot\boldsymbol E)
-\nabla^2\boldsymbol E
\]

と、

\[
\nabla\cdot\boldsymbol E=0
\]

を使うと、

\[
-\nabla^2\boldsymbol E
=
-\mu_0\epsilon_0
\frac{\partial^2\boldsymbol E}{\partial t^2}.
\]

したがって、

\[
\boxed{
\nabla^2\boldsymbol E
=
\mu_0\epsilon_0
\frac{\partial^2\boldsymbol E}{\partial t^2}
}
\]

である。

### 平面波を代入する

\[
\frac{\partial^2\boldsymbol E}{\partial z^2}
=
-k^2\boldsymbol E
\]

\[
\frac{\partial^2\boldsymbol E}{\partial t^2}
=
-\omega^2\boldsymbol E.
\]

波動方程式へ代入すると、

\[
-k^2\boldsymbol E
=
-\mu_0\epsilon_0\omega^2\boldsymbol E.
\]

よって、

\[
k^2
=
\mu_0\epsilon_0\omega^2.
\]

\(k,\omega>0\) なので、

### 最終解答

\[
\boxed{
k
=
\sqrt{\mu_0\epsilon_0}\,\omega
=
\frac{\omega}{c}
}
\]

である。

:::why title="なぜ正の平方根だけを選ぶの？"
問題文で \(k,\omega\) が正の実数と指定されているからである。また、位相 \(kz-\omega t\) は \(+z\) 方向へ進む波を表す。
:::

## Q1-3 磁場の式

### Faradayの法則を使う

電場を、

\[
\boldsymbol E
=
E_0\boldsymbol e_{\mathrm pol}
\cos\phi
\]

と置く。

\[
\phi=kz-\omega t
\]

\[
\boldsymbol e_{\mathrm pol}
=
\begin{pmatrix}
1/2\\
\sqrt3/2\\
0
\end{pmatrix}.
\]

真空中の平面波では、

\[
\boldsymbol B
=
\frac1\omega
\boldsymbol k\times\boldsymbol E
\]

である。

\[
\boldsymbol k=k\boldsymbol e_z.
\]

したがって、

\[
\boldsymbol B
=
\frac{k}{\omega}
\boldsymbol e_z
\times
\boldsymbol E.
\]

### 外積を計算する

\[
\boldsymbol e_z
\times
\begin{pmatrix}
1/2\\
\sqrt3/2\\
0
\end{pmatrix}
=
\begin{pmatrix}
-\sqrt3/2\\
1/2\\
0
\end{pmatrix}.
\]

また、

\[
\frac{k}{\omega}
=
\frac1c.
\]

したがって、

### 最終解答

\[
\boxed{
\boldsymbol B(\boldsymbol x,t)
=
\frac{E_0}{c}
\begin{pmatrix}
-\sqrt3/2\\
1/2\\
0
\end{pmatrix}
\cos(kz-\omega t)
}
\]

である。

\[
c=\frac1{\sqrt{\mu_0\epsilon_0}}
\]

を用いれば、

\[
\boxed{
\boldsymbol B
=
E_0\sqrt{\mu_0\epsilon_0}
\begin{pmatrix}
-\sqrt3/2\\
1/2\\
0
\end{pmatrix}
\cos(kz-\omega t)
}
\]

とも書ける。

:::why title="磁場の向きはどうやって確認するの？"
波の進行方向は \(\boldsymbol E\times\boldsymbol B\) の向きである。上の磁場を使うと \(\boldsymbol E\times\boldsymbol B\) は \(+z\) 方向となり、位相から分かる進行方向と一致する。
:::

### 計算による確認

\[
\boldsymbol e_{\mathrm pol}
\cdot
\begin{pmatrix}
-\sqrt3/2\\
1/2\\
0
\end{pmatrix}
=
-\frac{\sqrt3}{4}
+
\frac{\sqrt3}{4}
=0.
\]

したがって、

\[
\boldsymbol E\perp\boldsymbol B.
\]

## Q1-4 時間平均ポインティングベクトル

### 瞬時のポインティングベクトル

\[
\boldsymbol S
=
\frac1{\mu_0}
\boldsymbol E\times\boldsymbol B.
\]

電場と磁場の振幅方向は互いに直交し、両方の単位ベクトルの大きさは1である。

また、

\[
\boldsymbol e_{\mathrm pol}
\times
\begin{pmatrix}
-\sqrt3/2\\
1/2\\
0
\end{pmatrix}
=
\boldsymbol e_z.
\]

したがって、

\[
\boldsymbol S(\boldsymbol x,t)
=
\frac{E_0^2}{\mu_0c}
\cos^2(kz-\omega t)
\boldsymbol e_z.
\]

### 時間平均

\[
\left\langle\cos^2(kz-\omega t)\right\rangle_t
=
\frac12.
\]

よって、

### 最終解答

\[
\boxed{
\left\langle\boldsymbol S\right\rangle
=
\frac{E_0^2}{2\mu_0c}
\boldsymbol e_z
}
\]

である。

\[
\frac1{\mu_0c}
=
\epsilon_0c
\]

なので、

\[
\boxed{
\left\langle\boldsymbol S\right\rangle
=
\frac12\epsilon_0cE_0^2
\boldsymbol e_z
}
\]

とも書ける。

### 大きさと向き

大きさは、

\[
\boxed{
\left|
\left\langle\boldsymbol S\right\rangle
\right|
=
\frac{E_0^2}{2\mu_0c}
=
\frac12\epsilon_0cE_0^2
}
\]

向きは、

\[
\boxed{+z\text{ 方向}}
\]

である。

:::why title="なぜ時間平均で 1/2 が現れるの？"
電磁エネルギー流は電場と磁場の積に比例するため、正弦波では \(\cos^2\) が現れる。1周期平均は \(1/2\) である。
:::

### 試験答案

\[
\boldsymbol S
=
\frac1{\mu_0}\boldsymbol E\times\boldsymbol B
=
\frac{E_0^2}{\mu_0c}
\cos^2(kz-\omega t)
\boldsymbol e_z.
\]

したがって、

\[
\boxed{
\langle\boldsymbol S\rangle
=
\frac{E_0^2}{2\mu_0c}
\boldsymbol e_z
=
\frac12\epsilon_0cE_0^2
\boldsymbol e_z
}.
\]

# 問題2 ベクトル解析・プラズマ分散・Fresnel透過率

## 問題文（原文）

:::problem-original
以下の問いに答えよ。

2-1. \(xyz\) 座標系で、

\[
\boldsymbol x=(x,y,z),
\qquad
r=|\boldsymbol x|
\]

\(n\) を整数としたとき、

\[
\nabla r,\qquad
\nabla\left(\frac1r\right),
\qquad
\nabla r^n
\]

を、\(\boldsymbol x,r,n\) のうち必要なものを用いて表せ。ただし、原点を除くものとする。

2-2. 角周波数 \(\omega\) の平面電磁波に対する導体の屈折率 \(n(\omega)\) は、\(\omega\) が大きいときに次の式で近似できる。

\[
n(\omega)
=
\frac{ck}{\omega}
=
\sqrt{
1-\frac{\omega_p^2}{\omega^2}
}
\tag{6}
\]

ここで、\(\omega_p\) はプラズマ角周波数を表す。この平面電磁波の位相速度および群速度の式を書き、それらと光速 \(c\) との大小関係を説明せよ。

2-3. 図1のように、空気中に置かれた平行平板のガラス窓に垂直入射する平面電磁波を考える。電磁波の \(\omega\) は単一の値を持ち、空気の屈折率を1、ガラスの屈折率を実数 \(n\) とするとき、窓の片面（図の左側の面）の透過率 \(T\) を \(n\) を用いて表せ。

なお、透過率 \(T\) は、電磁波が屈折率 \(n_1\) の誘電体から \(n_2\) の誘電体に入射し、入射角、透過角が、それぞれ、\(\theta_1,\theta_2\) であった場合に、

\[
T
=
\frac{n_2\cos\theta_2}
{n_1\cos\theta_1}
\left(
\frac{E_0''}{E_0}
\right)^2
\tag{7}
\]

で与えられる。ここで、\(E_0,E_0''\) は、それぞれ、入射波、透過波の電場振幅を表し、p、s偏光の \(E_0''/E_0\) は、次のFresnelの式で表せる。

\[
\left(
\frac{E_0''}{E_0}
\right)_p
=
\frac{
2\cos\theta_1\sin\theta_2
}{
\sin(\theta_1+\theta_2)
\cos(\theta_1-\theta_2)
}
\tag{8}
\]

\[
\left(
\frac{E_0''}{E_0}
\right)_s
=
\frac{
2\cos\theta_1\sin\theta_2
}{
\sin(\theta_1+\theta_2)
}
\tag{9}
\]
:::

## Q2-1 \(\nabla r\)、\(\nabla(1/r)\)、\(\nabla r^n\)

### \(\nabla r\)

\[
r=\sqrt{x^2+y^2+z^2}.
\]

x成分は、

\[
\frac{\partial r}{\partial x}
=
\frac{x}{r}.
\]

同様に、

\[
\frac{\partial r}{\partial y}
=
\frac{y}{r}
\]

\[
\frac{\partial r}{\partial z}
=
\frac{z}{r}.
\]

したがって、

\[
\nabla r
=
\begin{pmatrix}
x/r\\
y/r\\
z/r
\end{pmatrix}.
\]

### 最終解答1

\[
\boxed{
\nabla r
=
\frac{\boldsymbol x}{r}
}
\]

である。

:::why title="∇r は何を表すの？"
\(r\) が最も急激に増える方向は原点から外向きの動径方向である。その単位ベクトルが \(\boldsymbol x/r\) である。
:::

---

### \(\nabla(1/r)\)

連鎖律より、

\[
\nabla\left(\frac1r\right)
=
-\frac1{r^2}\nabla r.
\]

したがって、

### 最終解答2

\[
\boxed{
\nabla\left(\frac1r\right)
=
-\frac{\boldsymbol x}{r^3}
}
\]

である。

---

### \(\nabla r^n\)

連鎖律より、

\[
\nabla r^n
=
nr^{n-1}\nabla r.
\]

\[
\nabla r=\frac{\boldsymbol x}{r}
\]

を代入すると、

### 最終解答3

\[
\boxed{
\nabla r^n
=
nr^{n-2}\boldsymbol x
}
\]

である。

### 試験答案

\[
r=(x^2+y^2+z^2)^{1/2}
\]

より、

\[
\boxed{
\nabla r=\frac{\boldsymbol x}{r}
}
\]

\[
\boxed{
\nabla(1/r)=-\frac{\boldsymbol x}{r^3}
}
\]

\[
\boxed{
\nabla r^n=nr^{n-2}\boldsymbol x
}.
\]

### 注意

原点では \(r=0\) となり、これらの式は特異になるため、問題文どおり原点を除く。

## Q2-2 プラズマ中の位相速度と群速度

### 分散関係

\[
n(\omega)
=
\frac{ck}{\omega}
=
\sqrt{
1-\frac{\omega_p^2}{\omega^2}
}.
\]

両辺から、

\[
k
=
\frac{\omega}{c}
\sqrt{
1-\frac{\omega_p^2}{\omega^2}
}.
\]

整理すると、

\[
\boxed{
k
=
\frac1c
\sqrt{\omega^2-\omega_p^2}
}
\]

である。

実数の波数を持つ伝搬波となるには、

\[
\boxed{\omega>\omega_p}
\]

が必要である。

### 位相速度

\[
v_{\mathrm p}
=
\frac{\omega}{k}.
\]

したがって、

\[
\boxed{
v_{\mathrm p}
=
\frac{c}
{\sqrt{
1-\omega_p^2/\omega^2
}}
}
\]

である。

\[
0<
\sqrt{
1-\omega_p^2/\omega^2
}
<1
\]

なので、

\[
\boxed{v_{\mathrm p}>c}.
\]

:::why title="位相速度が光速より大きくてもよいの？"
位相速度は同じ位相の点が移動する速度であり、情報やエネルギーの伝搬速度ではない。相対論で光速を超えられないのは、情報を運ぶ速度である。
:::

### 群速度

分散関係を、

\[
\omega^2=c^2k^2+\omega_p^2
\]

と書く。

両辺を \(k\) で微分する。

\[
2\omega
\frac{d\omega}{dk}
=
2c^2k.
\]

したがって、

\[
v_{\mathrm g}
=
\frac{d\omega}{dk}
=
\frac{c^2k}{\omega}.
\]

\(k\) を代入すると、

\[
\boxed{
v_{\mathrm g}
=
c
\sqrt{
1-\frac{\omega_p^2}{\omega^2}
}
}
\]

である。

したがって、

\[
\boxed{v_{\mathrm g}<c}.
\]

また、

\[
\boxed{
v_{\mathrm p}v_{\mathrm g}=c^2
}
\]

である。

### 屈折率との関係

\[
n(\omega)
=
\sqrt{
1-\frac{\omega_p^2}{\omega^2}
}
\]

なので、

\[
\boxed{
v_{\mathrm p}=\frac{c}{n}
}
\]

\[
\boxed{
v_{\mathrm g}=cn
}
\]

である。

### 極限

\[
\omega\to\omega_p+0
\]

では、

\[
v_{\mathrm p}\to\infty,
\qquad
v_{\mathrm g}\to0.
\]

\[
\omega\gg\omega_p
\]

では、

\[
v_{\mathrm p}\to c,
\qquad
v_{\mathrm g}\to c.
\]

### 試験答案

\[
k=\frac1c\sqrt{\omega^2-\omega_p^2}.
\]

よって、

\[
\boxed{
v_{\mathrm p}
=
\frac{\omega}{k}
=
\frac{c}
{\sqrt{1-\omega_p^2/\omega^2}}
>c
}
\]

である。

また、

\[
\omega^2=c^2k^2+\omega_p^2
\]

を微分して、

\[
\boxed{
v_{\mathrm g}
=
\frac{d\omega}{dk}
=
\frac{c^2k}{\omega}
=
c\sqrt{1-\omega_p^2/\omega^2}
<c
}.
\]

## Q2-3 ガラス界面の透過率

### 条件

空気からガラスへ垂直入射する。

\[
n_1=1,
\qquad
n_2=n
\]

\[
\theta_1=0,
\qquad
\theta_2=0.
\]

垂直入射では、p偏光とs偏光の区別はなく、同じ透過率になる。

### 電場振幅透過係数

垂直入射におけるFresnelの電場振幅透過係数は、

\[
\boxed{
\frac{E_0''}{E_0}
=
\frac{2n_1}{n_1+n_2}
}
\]

である。

\(n_1=1,n_2=n\) を代入すると、

\[
\boxed{
\frac{E_0''}{E_0}
=
\frac{2}{1+n}
}
\]

となる。

:::why title="問題文の式へ θ=0 を直接代入すると 0/0 にならない？"
式 (8)、(9) は角度を使った形なので、垂直入射を直接代入すると見かけ上 \(0/0\) になる。Snellの法則を用いて \(\theta_1,\theta_2\to0\) の極限を取ると、標準的な振幅透過係数 \(2n_1/(n_1+n_2)\) が得られる。
:::

### 電力透過率

\[
T
=
\frac{n_2\cos\theta_2}
{n_1\cos\theta_1}
\left(
\frac{E_0''}{E_0}
\right)^2.
\]

垂直入射では、

\[
\cos\theta_1
=
\cos\theta_2
=
1.
\]

したがって、

\[
T
=
n
\left(
\frac{2}{1+n}
\right)^2.
\]

### 最終解答

\[
\boxed{
T
=
\frac{4n}{(1+n)^2}
}
\]

である。

### 反射率による確認

垂直入射の反射率は、

\[
R
=
\left(
\frac{n-1}{n+1}
\right)^2.
\]

吸収がないため、

\[
T=1-R.
\]

\[
1-
\left(
\frac{n-1}{n+1}
\right)^2
=
\frac{(n+1)^2-(n-1)^2}{(n+1)^2}
=
\frac{4n}{(n+1)^2}.
\]

先ほどの答えと一致する。

### 注意

この問題で求めるのは、

\[
\boxed{\text{ガラス窓の左側の1つの界面}}
\]

における透過率である。

ガラス板全体の透過率を求める場合は、右側界面での反射や、板内部での多重反射を別途考慮する必要がある。

### 試験答案

垂直入射なので、

\[
n_1=1,\quad n_2=n,\quad
\theta_1=\theta_2=0.
\]

振幅透過係数は、

\[
\frac{E_0''}{E_0}
=
\frac{2}{1+n}.
\]

よって、

\[
T
=
n
\left(
\frac{2}{1+n}
\right)^2
\]

であるから、

\[
\boxed{
T=\frac{4n}{(1+n)^2}
}.
\]

# 年度全体の重要ポイント

- 電場成分の位相が同じなら直線偏光になる。
