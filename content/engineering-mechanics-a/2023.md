# この年度で必要な予備知識

## 1. ラグランジュ方程式

一般化座標を \(q\)、運動エネルギーを \(T\)、ポテンシャルエネルギーを \(U\) とすると、

\[
L=T-U
\]

をラグランジアンという。

運動方程式は、

\[
\boxed{
\frac{d}{dt}
\left(
\frac{\partial L}{\partial \dot q}
\right)
-
\frac{\partial L}{\partial q}
=0
}
\]

で与えられる。

## 2. 重心座標と相対座標

質量が等しい2物体では、

\[
X=\frac{x_1+x_2}{2}
\]

を重心座標、

\[
\xi=x_2-x_1
\]

を相対座標として使うと、運動を分けて考えやすい。

## 3. 剛体リンクの運動エネルギー

質量 \(m\) の剛体について、重心速度を \(v_G\)、重心まわりの慣性モーメントを \(I_G\)、角速度を \(\omega\) とすると、

\[
\boxed{
T=\frac12mv_G^2+\frac12I_G\omega^2
}
\]

である。

一様な細い棒の重心まわりの慣性モーメントは、

\[
\boxed{
I_G=\frac1{12}m\ell^2
}
\]

である。

## 4. ハミルトニアン

共役運動量を

\[
p=\frac{\partial L}{\partial\dot q}
\]

と定義する。

ハミルトニアンは、

\[
\boxed{
H(q,p)=p\dot q-L
}
\]

である。

## 5. ポアソン括弧

\[
\boxed{
[A,B]
=
\frac{\partial A}{\partial q}
\frac{\partial B}{\partial p}
-
\frac{\partial A}{\partial p}
\frac{\partial B}{\partial q}
}
\]

と定義する。

時間に陽に依存しない物理量 \(A(q,p)\) について、

\[
\frac{dA}{dt}=[A,H]
\]

なので、

\[
[A,H]=0
\]

ならば \(A\) は保存量である。

# Q1 ばねで連結された2台の台車

## 問題文（原文）

:::problem-original
図1に示すように、二つの力学台車がばね定数 \(k\) のばねで連結されている。いずれも質量は \(m\) で水平面上にある。

最初は右側の台車に外力が作用して、左側の台車が剛体壁に押し付けられている状態にある。その後、右側の台車が静かに解放されて外力が作用しなくなると、右側の台車が右方向へ移動していき、続いて左側の台車が壁から離れる。

図のようにばねの自然長を基準として台車の重心位置を \(x_1,x_2\) とおく。すなわち、初期状態ではばねが圧縮されて

\[
x_1=0,\qquad x_2=-d
\]

となっている。

解答においてはラグランジュの運動方程式を用いること。台車の運動には摩擦などの抵抗力は作用しないとする。

(a) 右側の台車が解放されてから左の台車が壁から離れるまでに経過する時間を求めよ。

(b) 左側の台車が壁を離れた後は一定値となる力学変数を示せ。

(c) 左側の台車が壁を離れた瞬間を時刻 \(t=0\) とする。その後の台車の動きを調べて、座標 \(x_1,x_2\) を \(t\) の関数として与えよ。
:::

## Q1-(a) 左側台車が壁を離れるまでの時間

### まず状況を整理する

左側台車が壁に押し付けられている間は、

\[
x_1=0
\]

のままで動かない。

したがって、壁を離れるまでは右側台車だけを考えればよい。

ばねの伸びは、

\[
x_2-x_1=x_2
\]

である。

初期状態では、

\[
x_2(0)=-d,\qquad \dot x_2(0)=0.
\]

### ラグランジアン

右側台車だけの運動エネルギーは、

\[
T=\frac12m\dot x_2^2.
\]

ばねのポテンシャルエネルギーは、

\[
U=\frac12kx_2^2.
\]

したがって、

\[
L=\frac12m\dot x_2^2-\frac12kx_2^2.
\]

ラグランジュ方程式より、

\[
m\ddot x_2+kx_2=0.
\]

固有角振動数を、

\[
\omega_0=\sqrt{\frac{k}{m}}
\]

と置く。

初期条件を満たす解は、

\[
\boxed{
x_2(t)=-d\cos\omega_0t
}
\]

である。

### 壁を離れる条件

ばねの左側台車に作用する力は、

\[
F_1=k(x_2-x_1)=kx_2
\]

である。

\(x_2<0\) の間は、ばねが左側台車を左向きに押すため、台車は壁から離れない。

初めて

\[
x_2=0
\]

となる瞬間に、壁からの垂直抗力が0になる。

\[
-d\cos\omega_0t=0.
\]

最初の正の解は、

\[
\omega_0t=\frac{\pi}{2}.
\]

### 最終解答

\[
\boxed{
t_{\mathrm{detach}}
=
\frac{\pi}{2\omega_0}
=
\frac{\pi}{2}\sqrt{\frac{m}{k}}
}
\]

である。

:::why title="なぜばねが自然長になった瞬間に離れるの？"
ばねが圧縮されている間は、左側台車を壁へ押す向きに力が働く。自然長になるとその力が0になり、その直後は右向きの力へ変わるため、台車が壁から離れる。
:::

### 離れる瞬間の速度

後の小問で使うため、右側台車の速度も求める。

\[
\dot x_2(t)
=
d\omega_0\sin\omega_0t.
\]

壁を離れる瞬間では、

\[
\omega_0t=\frac{\pi}{2}
\]

なので、

\[
\boxed{
\dot x_2=d\omega_0
=d\sqrt{\frac{k}{m}}
}
\]

である。

左側台車はそれまで静止していたため、

\[
\dot x_1=0.
\]

## Q1-(b) 壁を離れた後の保存量

壁を離れた後は、2台の台車とばねだけからなる系を考える。

外部から水平方向の力は作用しない。

### 運動方程式

ばねのポテンシャルエネルギーは、

\[
U=\frac12k(x_2-x_1)^2.
\]

左側台車について、

\[
m\ddot x_1=k(x_2-x_1).
\]

右側台車について、

\[
m\ddot x_2=-k(x_2-x_1).
\]

2式を加えると、

\[
m\ddot x_1+m\ddot x_2=0.
\]

したがって、

\[
\frac{d}{dt}
\left(
m\dot x_1+m\dot x_2
\right)=0.
\]

### 最終解答

全運動量

\[
\boxed{
P=m\dot x_1+m\dot x_2
}
\]

が保存される。

壁を離れた瞬間の値を使えば、

\[
\boxed{
P=md\sqrt{\frac{k}{m}}
=d\sqrt{mk}
}
\]

である。

:::why title="なぜ壁を離れる前は運動量が保存されないの？"
壁に接触している間は、壁から左側台車へ外力が作用している。壁を離れた後は水平方向の外力がなくなり、ばねの力は系の内部力なので、全運動量が保存される。
:::

## Q1-(c) 壁を離れた後の \(x_1(t),x_2(t)\)

壁を離れた瞬間を新しく

\[
t=0
\]

とする。

その瞬間、

\[
x_1(0)=0,\qquad x_2(0)=0
\]

\[
\dot x_1(0)=0,\qquad
\dot x_2(0)=v_0
\]

であり、

\[
v_0=d\sqrt{\frac{k}{m}}
\]

である。

### 重心座標

\[
X=\frac{x_1+x_2}{2}
\]

と置く。

2台の全運動量が保存されるので、

\[
\ddot X=0.
\]

初期条件は、

\[
X(0)=0
\]

\[
\dot X(0)=\frac{v_0}{2}.
\]

したがって、

\[
\boxed{
X(t)=\frac{v_0}{2}t
}
\]

である。

### 相対座標

\[
\xi=x_2-x_1
\]

と置く。

2台の運動方程式の差を取ると、

\[
\ddot\xi+\frac{2k}{m}\xi=0.
\]

相対運動の角振動数を、

\[
\Omega=\sqrt{\frac{2k}{m}}
\]

と置く。

初期条件は、

\[
\xi(0)=0
\]

\[
\dot\xi(0)=v_0.
\]

したがって、

\[
\xi(t)=\frac{v_0}{\Omega}\sin\Omega t.
\]

\[
\frac{v_0}{\Omega}
=
\frac{d\sqrt{k/m}}{\sqrt{2k/m}}
=
\frac{d}{\sqrt2}
\]

なので、

\[
\boxed{
\xi(t)=\frac{d}{\sqrt2}
\sin\left(\sqrt{\frac{2k}{m}}t\right)
}
\]

である。

### 元の座標へ戻す

\[
x_1=X-\frac{\xi}{2}
\]

\[
x_2=X+\frac{\xi}{2}.
\]

よって、

\[
\boxed{
x_1(t)
=
\frac{d}{2}\sqrt{\frac{k}{m}}\,t
-
\frac{d}{2\sqrt2}
\sin\left(\sqrt{\frac{2k}{m}}t\right)
}
\]

\[
\boxed{
x_2(t)
=
\frac{d}{2}\sqrt{\frac{k}{m}}\,t
+
\frac{d}{2\sqrt2}
\sin\left(\sqrt{\frac{2k}{m}}t\right)
}
\]

である。

### 運動の意味

- 系全体の重心は一定速度で右へ移動する。
- 2台の間隔は角振動数

\[
\sqrt{\frac{2k}{m}}
\]

で振動する。
- したがって、全体として右へ進みながら2台が互いに近づいたり離れたりする。

### 試験答案

壁を離れる瞬間を \(t=0\) とすると、

\[
x_1=x_2=0,\qquad
\dot x_1=0,\qquad
\dot x_2=d\sqrt{\frac{k}{m}}.
\]

重心座標 \(X=(x_1+x_2)/2\) と相対座標 \(\xi=x_2-x_1\) を用いると、

\[
\ddot X=0,\qquad
\ddot\xi+\frac{2k}{m}\xi=0.
\]

したがって、

\[
X=\frac{d}{2}\sqrt{\frac{k}{m}}t
\]

\[
\xi=\frac{d}{\sqrt2}
\sin\left(\sqrt{\frac{2k}{m}}t\right).
\]

よって、

\[
\boxed{
x_{1,2}
=
\frac{d}{2}\sqrt{\frac{k}{m}}t
\mp
\frac{d}{2\sqrt2}
\sin\left(\sqrt{\frac{2k}{m}}t\right)
}
\]

となる。

# Q2 2本のリンク・台車・鉛直剛体・ばね

## 問題文（原文）

:::problem-original
図2に示すように二つのリンクと力学台車、剛体、ばねがジョイントで接続され、鉛直面内に配置された一自由度振動系を仮定する。

ばね定数 \(k\) のばね上端は天井に固定されている。長さが \(a\) のリンクは両端にジョイントがあり、長さが \(2a\) のリンクは両端と中央にジョイントがある。

リンクの質量はそれぞれ \(m,2m\) であり、質量が均等に分布する剛体と仮定する。

力学台車は質量が \(M\) であり、水平面上を走行する。長さが \(2a\) のリンクの左端は鉛直方向に運動し、ばねはこの鉛直軸上にあることに注意する。質量 \(M\) の剛体の重心もこの軸上にある。

重力加速度は \(g\) とする。ばねが自然長のとき、リンクの傾斜角は \(\pi/6\) とする。ジョイントの回転などでエネルギーは散逸しないと仮定する。

(a) 図のように長さが \(a\) のリンクの傾斜角を \(\pi/6+\theta\) とおくとき、系全体の運動エネルギー \(T\) を求めよ。

(b) 系全体のポテンシャルエネルギー \(U\) を求めよ。

(c) 角 \(\theta\) の絶対値が小さいとして系の固有角振動数を求めよ。
:::

## 幾何学的な関係

短いリンクの角度を、

\[
\alpha=\frac{\pi}{6}+\theta
\]

と置く。

左下の固定支点を原点とする。

短いリンクの先端、すなわち長いリンクの中央ジョイントの座標は、

\[
C=(a\cos\alpha,\ a\sin\alpha)
\]

である。

長いリンクは長さ \(2a\) で、中央ジョイントから左右端までの長さはそれぞれ \(a\)。

右端は水平面上にあるので、

\[
R=(2a\cos\alpha,\ 0)
\]

となる。

左端は鉛直軸上にあるので、

\[
L=(0,\ 2a\sin\alpha)
\]

となる。

したがって、

- 台車の水平変位は \(2a\cos\alpha\)
- 上側剛体の鉛直変位は \(2a\sin\alpha\)

で表される。

:::why title="なぜ左右端の座標がこの形になるの？"
中央ジョイントから右端へのベクトルは、長さが \(a\) で水平成分 \(a\cos\alpha\)、鉛直成分 \(-a\sin\alpha\)。中央座標へ足すと右端は \((2a\cos\alpha,0)\) になる。左端は反対向きなので \((0,2a\sin\alpha)\) になる。
:::

---

## Q2-(a) 運動エネルギー

### 短いリンク

長さ \(a\)、質量 \(m\) の一様な棒が、左端の固定支点まわりに回転する。

支点まわりの慣性モーメントは、

\[
I_O=\frac13ma^2.
\]

したがって、

\[
T_{\mathrm{short}}
=
\frac12I_O\dot\theta^2
=
\frac16ma^2\dot\theta^2.
\]

### 長いリンク

長さ \(2a\)、質量 \(2m\) であり、重心は中央ジョイント \(C\) にある。

重心速度の大きさは、

\[
v_C=a\dot\theta.
\]

並進運動エネルギーは、

\[
\frac12(2m)v_C^2
=
ma^2\dot\theta^2.
\]

重心まわりの慣性モーメントは、

\[
I_C
=
\frac1{12}(2m)(2a)^2
=
\frac23ma^2.
\]

回転運動エネルギーは、

\[
\frac12I_C\dot\theta^2
=
\frac13ma^2\dot\theta^2.
\]

よって、

\[
T_{\mathrm{long}}
=
\frac43ma^2\dot\theta^2.
\]

### 上側の剛体 \(M\)

鉛直座標は、

\[
y_L=2a\sin\alpha.
\]

したがって、

\[
\dot y_L
=
2a\cos\alpha\,\dot\theta.
\]

運動エネルギーは、

\[
T_{\mathrm{upper}}
=
\frac12M(2a\cos\alpha)^2\dot\theta^2
=
2Ma^2\cos^2\alpha\,\dot\theta^2.
\]

### 台車 \(M\)

水平座標は、

\[
x_R=2a\cos\alpha.
\]

したがって、

\[
\dot x_R
=
-2a\sin\alpha\,\dot\theta.
\]

運動エネルギーは、

\[
T_{\mathrm{cart}}
=
2Ma^2\sin^2\alpha\,\dot\theta^2.
\]

### 合計

\[
T_{\mathrm{upper}}+T_{\mathrm{cart}}
=
2Ma^2(\cos^2\alpha+\sin^2\alpha)\dot\theta^2
=
2Ma^2\dot\theta^2.
\]

リンクの項を含めると、

\[
T
=
\left(
\frac16m+\frac43m+2M
\right)
a^2\dot\theta^2.
\]

したがって、

\[
\boxed{
T
=
\frac12(3m+4M)a^2\dot\theta^2
}
\]

である。

### 重要ポイント

- 長いリンクは「重心の並進」と「重心まわりの回転」の両方を持つ。
- 上側剛体と台車の項を足すと、三角関数が消えて一定の等価慣性になる。

## Q2-(b) ポテンシャルエネルギー

## ばねのポテンシャルエネルギー

ばねが自然長のとき、

\[
\alpha_0=\frac{\pi}{6}
\]

であり、

\[
2a\sin\alpha_0=a.
\]

角度 \(\alpha\) のとき、上側剛体の鉛直位置の変化は、

\[
\Delta \ell
=
2a
\left(
\sin\alpha-\frac12
\right).
\]

符号はばねの伸び・縮みによって変わるが、エネルギーでは二乗するため、

\[
U_{\mathrm{spring}}
=
\frac12k(\Delta\ell)^2.
\]

したがって、

\[
\boxed{
U_{\mathrm{spring}}
=
2ka^2
\left(
\sin\alpha-\frac12
\right)^2
}
\]

である。

## 重力ポテンシャルエネルギー

高さの基準による定数項は省略する。

### 短いリンク

重心の高さは、

\[
\frac{a}{2}\sin\alpha.
\]

したがって、

\[
U_{\mathrm{short}}
=
\frac12mga\sin\alpha.
\]

### 長いリンク

質量は \(2m\)、重心の高さは \(a\sin\alpha\)。

\[
U_{\mathrm{long}}
=
2mga\sin\alpha.
\]

### 上側剛体

高さの変化は \(2a\sin\alpha\) なので、

\[
U_{\mathrm{upper}}
=
2Mga\sin\alpha.
\]

台車の高さは一定なので、重力ポテンシャルは定数であり省略できる。

### 合計

\[
U_g
=
\left(
\frac12m+2m+2M
\right)
ga\sin\alpha.
\]

\[
\boxed{
U_g
=
\frac{ga}{2}(5m+4M)\sin\alpha
}
\]

である。

したがって、系全体のポテンシャルエネルギーは、

\[
\boxed{
U
=
2ka^2
\left(
\sin\alpha-\frac12
\right)^2
+
\frac{ga}{2}(5m+4M)\sin\alpha
}
\]

である。

ここで、

\[
\alpha=\frac{\pi}{6}+\theta.
\]

## Q2-(c) 固有角振動数

\[
|\theta|\ll1
\]

として、\(\theta=0\) の近くで2次まで展開する。

### 三角関数の展開

\[
\sin\left(\frac{\pi}{6}+\theta\right)
\approx
\frac12+\frac{\sqrt3}{2}\theta-\frac14\theta^2.
\]

### ばねエネルギー

\[
\sin\alpha-\frac12
\approx
\frac{\sqrt3}{2}\theta.
\]

したがって、

\[
U_{\mathrm{spring}}
\approx
2ka^2
\left(
\frac{\sqrt3}{2}\theta
\right)^2
=
\frac32ka^2\theta^2.
\]

### 重力エネルギー

\[
U_g
\approx
\frac{ga}{2}(5m+4M)
\left(
\frac12
+
\frac{\sqrt3}{2}\theta
-
\frac14\theta^2
\right).
\]

定数項を無視すると、

\[
U
\approx
\frac{\sqrt3ga}{4}(5m+4M)\theta
+
\frac12K_{\mathrm{eff}}\theta^2
\]

と書ける。

ここで、

\[
\boxed{
K_{\mathrm{eff}}
=
3ka^2
-
\frac{ga}{4}(5m+4M)
}
\]

である。

運動エネルギーは、

\[
T
=
\frac12I_{\mathrm{eff}}\dot\theta^2
\]

\[
\boxed{
I_{\mathrm{eff}}
=
(3m+4M)a^2
}
\]

である。

### 線形化した運動方程式

\[
I_{\mathrm{eff}}\ddot\theta
+
K_{\mathrm{eff}}\theta
+
\frac{\sqrt3ga}{4}(5m+4M)
=0.
\]

定数項は平衡位置をずらすだけである。

線形近似での平衡角を \(\theta_{\mathrm e}\) とし、

\[
\eta=\theta-\theta_{\mathrm e}
\]

と置くと、

\[
I_{\mathrm{eff}}\ddot\eta
+
K_{\mathrm{eff}}\eta=0.
\]

したがって、固有角振動数は、

\[
\boxed{
\omega_n
=
\sqrt{
\frac{
3ka^2-\dfrac{ga}{4}(5m+4M)
}{
(3m+4M)a^2
}
}
}
\]

である。

分子・分母から \(a\) を整理すると、

\[
\boxed{
\omega_n
=
\sqrt{
\frac{
3ka-\dfrac{g}{4}(5m+4M)
}{
(3m+4M)a
}
}
}
\]

とも書ける。

### 安定条件

実数の固有角振動数を持つためには、

\[
K_{\mathrm{eff}}>0
\]

が必要である。

\[
\boxed{
3ka^2>
\frac{ga}{4}(5m+4M)
}
\]

すなわち、

\[
\boxed{
12ka>g(5m+4M)
}
\]

である。

:::why title="なぜ重力の定数項は振動数に影響しないの？"
定数項は平衡位置をどこへ移すかを決める。一方、平衡位置の近くでどれだけ強く元へ戻そうとするかは二次項の係数で決まり、これが固有角振動数を決める。
:::

### 試験答案

\[
T=\frac12(3m+4M)a^2\dot\theta^2.
\]

\[
U=
2ka^2
\left[
\sin\left(\frac{\pi}{6}+\theta\right)-\frac12
\right]^2
+
\frac{ga}{2}(5m+4M)
\sin\left(\frac{\pi}{6}+\theta\right).
\]

\(|\theta|\ll1\) として2次まで展開すると、

\[
U
=
\text{const.}
+
\frac{\sqrt3ga}{4}(5m+4M)\theta
+
\frac12
\left[
3ka^2-\frac{ga}{4}(5m+4M)
\right]\theta^2.
\]

平衡位置からの変位について、

\[
(3m+4M)a^2\ddot\eta
+
\left[
3ka^2-\frac{ga}{4}(5m+4M)
\right]\eta=0.
\]

よって、

\[
\boxed{
\omega_n
=
\sqrt{
\frac{
3ka^2-\dfrac{ga}{4}(5m+4M)
}{
(3m+4M)a^2
}
}
}.
\]

# Q3 ハミルトン力学と正準変換

## 問題文（原文）

:::problem-original
1次元運動のラグランジアンが

\[
L(q,\dot q)=\dot q^2-q^2
\]

で与えられるとき、以下の問いに答えよ。

(1) \(q\) と \(p\) の関数であるハミルトニアン

\[
H(q,p)=p\dot q-L
\]

を求めよ。ただし、

\[
p=\frac{\partial L}{\partial\dot q}
\]

とする。

(2) ポアソン括弧式

\[
[A,B]
=
\frac{\partial A}{\partial q}
\frac{\partial B}{\partial p}
-
\frac{\partial A}{\partial p}
\frac{\partial B}{\partial q}
\]

を考えることにより、この運動の保存量（\(H\) の定数倍を除く）を1つ求めよ。

\(q,p\) を正準変数とする正準方程式は、

\[
\frac{dq}{dt}=\frac{\partial H}{\partial p},
\qquad
\frac{dp}{dt}=-\frac{\partial H}{\partial q}
\tag{A}
\]

である。

(3) (1) の \(H=E\)（正の定数）のとき、位相空間（\(q-p\) 面）内での軌道を図示せよ。その際、軌道上の運動の向きも矢印で記入すること。

(4) 軌道上の2点 \(A\) と \(B\) が、初期 \(t=0\) にそれぞれ

\[
A(0,p_0),\qquad B(q_0,0)
\]

にあったとき、任意の時刻 \(t\) におけるそれぞれの座標 \((q,p)\) を求めよ。

(5) 位相空間の原点 \(O(0,0)\) は位相空間内での不動点である。(4) の結果を用いて、\(\triangle OAB\) の面積が運動によって不変であることを示せ（Liouvilleの定理の例）。

(1) のハミルトニアン \(H(q,p)\) において、

\[
Q=\log\left(\frac1q\sin p\right),
\qquad
P=q\cot p
\tag{B}
\]

で定義される \((q,p)\to(Q,P)\) の独立変数変換を行う。

(6)

\[
\frac{\partial H}{\partial p},
\qquad
\frac{\partial H}{\partial q}
\]

をそれぞれ、

\[
\frac{\partial H}{\partial P},
\qquad
\frac{\partial H}{\partial Q}
\]

を用いて表せ。

(7) 正準方程式 (A) と (6) の結果を用いて、

\[
\frac{dQ}{dt},
\qquad
\frac{dP}{dt}
\]

を計算し、変換 (B) が正準変換であることを示せ。
:::

## Q3-(1) ハミルトニアン

ラグランジアンは、

\[
L=\dot q^2-q^2.
\]

共役運動量は、

\[
p=\frac{\partial L}{\partial\dot q}
=2\dot q.
\]

したがって、

\[
\dot q=\frac p2.
\]

ハミルトニアンは、

\[
H=p\dot q-L.
\]

\[
=p\frac p2
-
\left[
\left(\frac p2\right)^2-q^2
\right].
\]

\[
=\frac{p^2}{2}-\frac{p^2}{4}+q^2.
\]

### 最終解答

\[
\boxed{
H(q,p)=\frac{p^2}{4}+q^2
}
\]

である。

:::why title="通常の調和振動子と係数が違うのはなぜ？"
問題のラグランジアンには通常の \(1/2\) が付いていない。そのため、共役運動量が \(p=2\dot q\) となり、ハミルトニアンの運動量項が \(p^2/4\) になる。
:::

---

## Q3-(2) ポアソン括弧から保存量を求める

任意の微分可能な関数 \(F(H)\) を考える。

\[
[F(H),H]
=
F'(H)[H,H].
\]

ポアソン括弧の反対称性より、

\[
[H,H]=0.
\]

したがって、

\[
[F(H),H]=0.
\]

よって、\(F(H)\) は保存量である。

\(H\) の単なる定数倍ではない例として、

\[
A=H^2
\]

を選ぶ。

\[
[A,H]=[H^2,H]
=2H[H,H]=0.
\]

### 最終解答

\[
\boxed{
A(q,p)=H^2
=
\left(
\frac{p^2}{4}+q^2
\right)^2
}
\]

は保存量である。

### 補足

1自由度の自律系では、時間に依存しない独立な保存量は通常エネルギー1つである。そのため、ここで得た \(H^2\) は \(H\) とは独立ではないが、「\(H\) の定数倍ではない保存量」という問題の条件を満たす。

---

## Q3-(3) 位相空間内の軌道

\[
H=E
\]

より、

\[
\frac{p^2}{4}+q^2=E.
\]

標準形にすると、

\[
\boxed{
\frac{q^2}{E}
+
\frac{p^2}{4E}
=1
}
\]

である。

したがって、\(q-p\) 平面上の軌道は、

- \(q\) 方向の半径：\(\sqrt E\)
- \(p\) 方向の半径：\(2\sqrt E\)

の楕円である。

### 運動の向き

正準方程式は、

\[
\dot q=\frac{\partial H}{\partial p}=\frac p2
\]

\[
\dot p=-\frac{\partial H}{\partial q}=-2q.
\]

例えば右端

\[
(q,p)=(\sqrt E,0)
\]

では、

\[
\dot q=0,\qquad
\dot p=-2\sqrt E<0.
\]

したがって、右端から下向きへ進む。

よって軌道上の運動方向は、

\[
\boxed{\text{時計回り}}
\]

である。

---

## Q3-(4) 2点 \(A,B\) の時刻 \(t\) における座標

正準方程式は、

\[
\dot q=\frac p2
\]

\[
\dot p=-2q.
\]

第1式を時間微分すると、

\[
\ddot q=\frac{\dot p}{2}=-q.
\]

したがって、

\[
\boxed{\ddot q+q=0}
\]

である。

角振動数は1。

---

### 点 \(A\)

初期条件は、

\[
q_A(0)=0,\qquad
p_A(0)=p_0.
\]

\[
\dot q_A(0)=\frac{p_0}{2}.
\]

したがって、

\[
\boxed{
q_A(t)=\frac{p_0}{2}\sin t
}
\]

である。

\[
p_A=2\dot q_A
\]

より、

\[
\boxed{
p_A(t)=p_0\cos t
}
\]

である。

---

### 点 \(B\)

初期条件は、

\[
q_B(0)=q_0,\qquad
p_B(0)=0.
\]

\[
\dot q_B(0)=0.
\]

したがって、

\[
\boxed{
q_B(t)=q_0\cos t
}
\]

である。

\[
p_B=2\dot q_B
\]

より、

\[
\boxed{
p_B(t)=-2q_0\sin t
}
\]

である。

### 最終解答

\[
\boxed{
A(t):
\left(
\frac{p_0}{2}\sin t,\ 
p_0\cos t
\right)
}
\]

\[
\boxed{
B(t):
\left(
q_0\cos t,\ 
-2q_0\sin t
\right)
}
\]

---

## Q3-(5) \(\triangle OAB\) の面積保存

原点を \(O=(0,0)\) とする。

2つのベクトル

\[
\overrightarrow{OA}
=
\begin{pmatrix}
q_A\\
p_A
\end{pmatrix}
\]

\[
\overrightarrow{OB}
=
\begin{pmatrix}
q_B\\
p_B
\end{pmatrix}
\]

が作る三角形の面積は、

\[
S
=
\frac12
\left|
q_Ap_B-p_Aq_B
\right|.
\]

前問の結果を代入する。

\[
q_Ap_B
=
\left(
\frac{p_0}{2}\sin t
\right)
\left(
-2q_0\sin t
\right)
=
-p_0q_0\sin^2t.
\]

\[
p_Aq_B
=
(p_0\cos t)(q_0\cos t)
=
p_0q_0\cos^2t.
\]

したがって、

\[
q_Ap_B-p_Aq_B
=
-p_0q_0
(\sin^2t+\cos^2t).
\]

\[
=-p_0q_0.
\]

よって、

\[
\boxed{
S=\frac12|p_0q_0|
}
\]

となり、時間 \(t\) に依存しない。

したがって、

\[
\boxed{
\triangle OAB\text{ の面積は運動によって不変}
}
\]

である。

:::why title="これがLiouvilleの定理の例になる理由"
ハミルトン力学の時間発展では、位相空間内の領域は引き伸ばされたり変形したりしても、その面積や体積を保つ。この問題では三角形の面積を直接計算することで確認している。
:::

---

## Q3-(6) 偏微分の変換

変数変換は、

\[
Q=\log\left(\frac{\sin p}{q}\right)
\]

\[
P=q\cot p
\]

である。

### \(Q,P\) の偏微分

\[
\frac{\partial Q}{\partial p}
=
\cot p
\]

\[
\frac{\partial Q}{\partial q}
=
-\frac1q
\]

\[
\frac{\partial P}{\partial p}
=
-q\csc^2p
\]

\[
\frac{\partial P}{\partial q}
=
\cot p.
\]

### 連鎖律

\[
\frac{\partial}{\partial p}
=
\frac{\partial Q}{\partial p}
\frac{\partial}{\partial Q}
+
\frac{\partial P}{\partial p}
\frac{\partial}{\partial P}.
\]

したがって、

\[
\frac{\partial}{\partial p}
=
\cot p\frac{\partial}{\partial Q}
-
q\csc^2p\frac{\partial}{\partial P}.
\]

よって、

\[
\boxed{
\frac{\partial H}{\partial p}
=
\cot p\frac{\partial H}{\partial Q}
-
q\csc^2p\frac{\partial H}{\partial P}
}
\]

である。

同様に、

\[
\frac{\partial}{\partial q}
=
-\frac1q\frac{\partial}{\partial Q}
+
\cot p\frac{\partial}{\partial P}.
\]

よって、

\[
\boxed{
\frac{\partial H}{\partial q}
=
-\frac1q\frac{\partial H}{\partial Q}
+
\cot p\frac{\partial H}{\partial P}
}
\]

である。

### 間違えやすい点

- \(P=q\cot p\) の \(p\) 微分では、

\[
\frac{d}{dp}\cot p=-\csc^2p
\]

を使う。
- \(\partial H/\partial p\) を計算するときは、\(Q\) と \(P\) の両方が \(p\) に依存する。

---

## Q3-(7) 正準変換であることの証明

元の正準方程式は、

\[
\dot q=H_p
\]

\[
\dot p=-H_q
\]

である。

ここで、

\[
H_p=\frac{\partial H}{\partial p},
\qquad
H_q=\frac{\partial H}{\partial q}.
\]

---

### \(dQ/dt\)

連鎖律より、

\[
\dot Q
=
Q_q\dot q+Q_p\dot p.
\]

\[
=
-\frac1qH_p
+
\cot p(-H_q).
\]

\[
=
-\frac1qH_p-\cot pH_q.
\]

前問の結果を代入する。

\[
-\frac1qH_p
=
-\frac{\cot p}{q}H_Q
+
\csc^2pH_P.
\]

\[
-\cot pH_q
=
\frac{\cot p}{q}H_Q
-
\cot^2pH_P.
\]

足すと \(H_Q\) の項は消える。

\[
\dot Q
=
(\csc^2p-\cot^2p)H_P.
\]

恒等式

\[
\csc^2p-\cot^2p=1
\]

より、

\[
\boxed{
\dot Q=\frac{\partial H}{\partial P}
}
\]

を得る。

---

### \(dP/dt\)

\[
\dot P
=
P_q\dot q+P_p\dot p.
\]

\[
=
\cot pH_p
+
(-q\csc^2p)(-H_q).
\]

\[
=
\cot pH_p
+
q\csc^2pH_q.
\]

前問の結果を代入する。

\[
\dot P
=
\cot p
\left(
\cot pH_Q-q\csc^2pH_P
\right)
\]

\[
+
q\csc^2p
\left(
-\frac1qH_Q+\cot pH_P
\right).
\]

\(H_P\) の項は打ち消し合う。

\[
\dot P
=
(\cot^2p-\csc^2p)H_Q.
\]

\[
\cot^2p-\csc^2p=-1
\]

より、

\[
\boxed{
\dot P=-\frac{\partial H}{\partial Q}
}
\]

を得る。

---

### 最終結論

新しい変数 \(Q,P\) についても、

\[
\boxed{
\dot Q=\frac{\partial H}{\partial P}
}
\]

\[
\boxed{
\dot P=-\frac{\partial H}{\partial Q}
}
\]

が成立する。

したがって、変換

\[
\boxed{
Q=\log\left(\frac{\sin p}{q}\right),
\qquad
P=q\cot p
}
\]

は正準変換である。
