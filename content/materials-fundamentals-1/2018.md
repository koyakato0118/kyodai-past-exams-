# この年度で必要な予備知識

## 1. 結晶面と結晶方向

結晶面は丸括弧、

\[
(hkl)
\]

結晶方向は角括弧、

\[
[uvw]
\]

で表す。

同じ対称性を持つ面の集まりは、

\[
\{hkl\}
\]

同じ対称性を持つ方向の集まりは、

\[
\langle uvw\rangle
\]

と表す。

## 2. 六方晶のミラー・ブラヴェ指数

六方晶では、底面内に120度ずつ離れた3本の軸 \(x,y,u\) と、底面に垂直な \(z\) 軸を用いる。

面は、

\[
(hkil)
\]

方向は、

\[
[uvtw]
\]

と4つの指数で表す。

面については、

\[
\boxed{h+k+i=0}
\]

方向については、

\[
\boxed{u+v+t=0}
\]

が成り立つ。

### 面の指数の求め方

面が各軸を切る位置を格子定数単位で、

\[
p,\ q,\ r,\ s
\]

としたとき、その逆数、

\[
\frac1p,\ \frac1q,\ \frac1r,\ \frac1s
\]

を最小整数比へ直したものが \((hkil)\) である。

軸と平行なら切片は無限大なので、対応する指数は0。

負の側で軸を切る場合は、指数へバーを付ける。

## 3. すべり系

塑性変形は、転位が特定の面と方向に沿って移動することで起こる。

- 転位が動く面：すべり面
- 転位が動く方向：すべり方向
- 両者の組：すべり系

一般に、原子が最も密に並ぶ面と方向ほど、転位が移動しやすい。

## 4. Schmidの法則

単軸引張応力を \(\sigma\) とする。

引張軸とすべり面法線のなす角を \(\phi\)、引張軸とすべり方向のなす角を \(\lambda\) とすると、分解せん断応力は、

\[
\boxed{
\tau_{\mathrm R}
=
\sigma\cos\phi\cos\lambda
}
\]

である。

Schmid因子は、

\[
\boxed{
m=\cos\phi\cos\lambda
}
\]

である。

## 5. てこの法則

二相領域で、全体組成を \(C_0\)、左側相の組成を \(C_\alpha\)、右側相の組成を \(C_\gamma\) とすると、

\[
\boxed{
W_\alpha
=
\frac{C_\gamma-C_0}
{C_\gamma-C_\alpha}
}
\]

\[
\boxed{
W_\gamma
=
\frac{C_0-C_\alpha}
{C_\gamma-C_\alpha}
}
\]

である。

「求めたい相と反対側の線分の長さ」を使うと覚える。

# 問2 fccの結晶構造とすべり系

## 問題文（原文）

:::problem-original
fcc の結晶構造を図示し、fcc のすべり面とすべり方向のミラー指数を書きなさい。また、そうなる理由を説明しなさい。
:::

## fccの結晶構造

fccはface-centered cubic、面心立方構造である。

立方体の、

- 8個の頂点
- 6個の面の中心

に原子が配置される。

### 単位胞中の原子数

頂点原子は8個の単位胞で共有されるので、

\[
8\times\frac18=1.
\]

面心原子は2個の単位胞で共有されるので、

\[
6\times\frac12=3.
\]

したがって、

\[
\boxed{
1+3=4\text{ 原子/単位胞}
}
\]

である。

配位数は、

\[
\boxed{12}
\]

である。

## すべり面

fccで最も原子密度が高い面は、

\[
\boxed{\{111\}}
\]

である。

したがって、すべり面は、

\[
\boxed{\{111\}}
\]

となる。

## すべり方向

\(\{111\}\) 面内で最も原子が密に並ぶ方向は、

\[
\boxed{\langle110\rangle}
\]

である。

したがって、すべり方向は、

\[
\boxed{\langle110\rangle}
\]

となる。

完全転位のBurgersベクトルは、

\[
\boxed{
\boldsymbol b
=
\frac a2\langle110\rangle
}
\]

であり、その大きさは、

\[
|\boldsymbol b|
=
\frac a{\sqrt2}.
\]

### 例

\((111)\) 面内には、

\[
[\bar110],\quad
[10\bar1],\quad
[01\bar1]
\]

などの \(\langle110\rangle\) 方向がある。

方向が面内にあるかは、面法線との内積が0かで確認できる。

例えば、

\[
[111]\cdot[\bar110]
=
-1+1+0
=0.
\]

したがって、\([\bar110]\) は \((111)\) 面内にある。

## すべり系の数

独立な \(\{111\}\) 面は4面。

各面に独立な \(\langle110\rangle\) 方向が3方向ある。

したがって、

\[
\boxed{
4\times3=12\text{ すべり系}
}
\]

である。

## なぜこの面と方向なのか

転位の移動には、原子結合を順番に組み替える必要がある。

- 原子が最も密な面では、面間の凹凸が小さい
- 原子が最も密な方向では、隣の安定位置までの距離が短い
- Burgersベクトルが短いほど、転位の弾性エネルギーが小さい

転位のエネルギーは概ね、

\[
E_{\mathrm dislocation}
\propto
Gb^2
\]

である。

したがって、最密面 \(\{111\}\) と最密方向 \(\langle110\rangle\) の組が最もすべりやすい。

### 最終解答

\[
\boxed{
\text{fccのすべり系}
=
\{111\}\langle110\rangle
}
\]

すべり系は12個である。

:::why title="すべり方向を [110] ではなく <110> と書くの？"
fccには結晶対称性によって等価な複数の110型方向がある。1本だけでなく等価な方向の集合を表すため、山括弧 \(\langle110\rangle\) を使う。
:::

### 試験答案

fccは立方体の頂点と各面中心に原子を持つ面心立方構造である。

最密面は、

\[
\{111\}
\]

最密方向は、

\[
\langle110\rangle
\]

なので、

\[
\boxed{
\{111\}\langle110\rangle
}
\]

がすべり系となる。

これは原子密度が高く、完全転位のBurgersベクトル

\[
\boldsymbol b=\frac a2\langle110\rangle
\]

が短いため、転位が最も移動しやすいからである。

# 問3 bccのSchmid因子

## 問題文（原文）

:::problem-original
bcc の

\[
[010]
\]

方向に100 MPaの引っ張り応力を加えた。

\[
(110)
\]

面上、

\[
[\bar111]
\]

方向のSchmid因子と分解せん断応力を答えなさい。
:::

## 解法方針

Schmid因子は、

\[
m=\cos\phi\cos\lambda
\]

である。

- \(\phi\)：引張軸とすべり面法線の角度
- \(\lambda\)：引張軸とすべり方向の角度

## 引張軸

\[
\boldsymbol t=[010].
\]

## すべり面法線

\((110)\) 面の法線方向は、

\[
\boldsymbol n=[110].
\]

したがって、

\[
\cos\phi
=
\frac{
[010]\cdot[110]
}{
|[010]||[110]|
}.
\]

内積は、

\[
0\cdot1+1\cdot1+0\cdot0=1.
\]

大きさは、

\[
|[010]|=1
\]

\[
|[110]|=\sqrt2.
\]

よって、

\[
\boxed{
\cos\phi=\frac1{\sqrt2}
}
\]

である。

## すべり方向

\[
\boldsymbol d=[\bar111].
\]

\[
\cos\lambda
=
\frac{
[010]\cdot[\bar111]
}{
|[010]||[\bar111]|
}.
\]

内積は、

\[
0(-1)+1(1)+0(1)=1.
\]

\[
|[\bar111]|=\sqrt3.
\]

よって、

\[
\boxed{
\cos\lambda=\frac1{\sqrt3}
}
\]

である。

### 面内方向であることの確認

\[
[110]\cdot[\bar111]
=
-1+1+0=0.
\]

したがって、指定された方向は確かに \((110)\) 面内にある。

## Schmid因子

\[
m
=
\frac1{\sqrt2}
\frac1{\sqrt3}.
\]

### 最終解答1

\[
\boxed{
m=\frac1{\sqrt6}\simeq0.408
}
\]

である。

## 分解せん断応力

\[
\tau_{\mathrm R}
=
\sigma m.
\]

\[
\sigma=100\,\mathrm{MPa}
\]

なので、

\[
\tau_{\mathrm R}
=
100
\times
\frac1{\sqrt6}.
\]

### 最終解答2

\[
\boxed{
\tau_{\mathrm R}
\simeq40.8\,\mathrm{MPa}
}
\]

である。

:::why title="引張応力なのに、すべり面にはせん断応力が生じるの？"
斜めの面から見ると、引張力には面を垂直に引く成分だけでなく、面に沿ってずらす成分もある。この面内の成分が転位を動かす分解せん断応力である。
:::

### 試験答案

\[
\cos\phi
=
\frac{[010]\cdot[110]}
{|[010]||[110]|}
=
\frac1{\sqrt2}
\]

\[
\cos\lambda
=
\frac{[010]\cdot[\bar111]}
{|[010]||[\bar111]|}
=
\frac1{\sqrt3}.
\]

したがって、

\[
\boxed{
m=\cos\phi\cos\lambda
=\frac1{\sqrt6}
=0.408
}
\]

\[
\boxed{
\tau_{\mathrm R}
=
100m
=40.8\,\mathrm{MPa}
}
\]

となる。

# 問4 転位の消滅と交差すべり

## 問題文（原文）

:::problem-original
同じすべり面上に符号の異なる、すなわちBurgersベクトルが逆方向の刃状転位が1組存在する場合、最終的にどうなるか答えよ。

また、交差すべりが刃状転位では生じず、らせん転位では生じる理由を説明しなさい。
:::

## 符号の異なる刃状転位

Burgersベクトルを、

\[
+\boldsymbol b
\]

と、

\[
-\boldsymbol b
\]

とする。

符号が異なる転位の応力場は、互いを引き寄せる向きに力を及ぼす。

同じすべり面上にあるため、両者はすべり運動によって近づくことができる。

転位が同じ位置へ到達すると、Burgersベクトルの和は、

\[
\boldsymbol b+(-\boldsymbol b)=0.
\]

したがって転位線が消える。

### 最終解答

\[
\boxed{
\text{互いに引き合って接近し、最終的に対消滅する}
}
\]

その結果、余分な半原子面どうしが打ち消され、局所的には完全な結晶へ戻る。

:::why title="同符号の転位ならどうなるの？"
同符号の刃状転位は同じ向きの応力場を持ち、同じすべり面上では互いに反発する。異符号の場合だけ、近づいて消滅できる。
:::

---

## 交差すべり

交差すべりとは、転位が元のすべり面から、別のすべり面へ移る現象である。

## 刃状転位の場合

刃状転位では、

\[
\boldsymbol b\perp\boldsymbol l
\]

である。

ここで、

- \(\boldsymbol b\)：Burgersベクトル
- \(\boldsymbol l\)：転位線方向

すべり面は、

\[
\boxed{
\boldsymbol b\text{ と }\boldsymbol l
\text{ の両方を含む面}
}
\]

でなければならない。

互いに平行でない2本の方向ベクトルは、1つの平面を一意に決める。

したがって、刃状転位のすべり面は幾何学的に1つに定まり、別の面へすべることはできない。

別の面へ移るには、原子空孔の拡散を伴う上昇運動、climbが必要になる。

## らせん転位の場合

らせん転位では、

\[
\boldsymbol b\parallel\boldsymbol l.
\]

すべり面は \(\boldsymbol b\) と転位線を含む必要があるが、両者が同一直線なので、その直線を含む面は複数存在する。

したがって、十分な分解せん断応力を持つ別のすべり面へ移ることができる。

### 最終解答

\[
\boxed{
\begin{aligned}
\text{刃状転位：}&
\boldsymbol b\perp\boldsymbol l
\text{ なので、すべり面が一意に定まる}\\
\text{らせん転位：}&
\boldsymbol b\parallel\boldsymbol l
\text{ なので、同じ転位線を含む複数の面を選べる}
\end{aligned}
}
\]

よって交差すべりは、基本的にらせん転位で生じる。

### 試験答案

異符号の刃状転位は互いに引き合い、同じすべり面上を移動して接近する。接触すると、

\[
\boldsymbol b+(-\boldsymbol b)=0
\]

となるため対消滅する。

刃状転位では \(\boldsymbol b\) と転位線が直交し、この2方向によってすべり面が一意に定まる。

らせん転位では \(\boldsymbol b\) と転位線が平行であり、その直線を含む複数のすべり面を取れるため、交差すべりが可能である。

# 問5 0.3%C炭素鋼の徐冷

## 問題文（原文）

:::problem-original
炭素含有量0.3%の炭素鋼を900℃から徐冷した。

(1) 相と組織変化について述べなさい。

(2) 領域Iに存在する相を答えなさい。また、727℃、A1点直上でのそれぞれの相の量比と炭素濃度を求めなさい。ただし、P点は0.02%、S点は0.77%とする。

Fe-C状態図では、点P、S、Gで囲まれる領域が領域Iである。
:::

:::problem-asset
src: /exams/materials-fundamentals-1/2018/page-2.webp
alt: 材料基礎学1 2018年度 問5のFe-C状態図
:::

## 鋼の分類

共析組成は、

\[
0.77\%\mathrm C
\]

である。

今回の鋼は、

\[
0.30\%\mathrm C<0.77\%\mathrm C
\]

なので、

\[
\boxed{\text{亜共析鋼}}
\]

である。

---

## Q5-(1) 相と組織の変化

## 900℃

0.3%C鋼では、900℃はA3変態点より高い。

したがって、

\[
\boxed{
\gamma\text{ 単相}
}
\]

である。

組織名は、

\[
\boxed{\text{オーステナイト}}
\]

である。

## A3線を下回った後

冷却してA3線を横切ると、オーステナイトからフェライトが析出し始める。

相は、

\[
\boxed{
\alpha+\gamma
}
\]

となる。

- \(\alpha\)：フェライト
- \(\gamma\)：オーステナイト

組織としては、

\[
\boxed{
\text{初析フェライト}
+
\text{オーステナイト}
}
\]

である。

フェライトは炭素をほとんど固溶できないため、析出したフェライトから炭素がオーステナイト側へ排出される。

したがって、残ったオーステナイトの炭素濃度は、冷却とともに0.77%へ近づく。

:::why title="なぜ残ったオーステナイトの炭素濃度が上がるの？"

# Q6-(1) 応力拡大係数と破壊じん性

## 応力拡大係数

き裂先端の近くでは、応力が、

\[
\sigma_{ij}
\simeq
\frac{K}{\sqrt{2\pi r}}
f_{ij}(\theta)
\]

の形で大きくなる。

- \(r\)：き裂先端からの距離
- \(\theta\)：き裂先端まわりの角度
- \(K\)：応力拡大係数

モードI、開口型き裂では、

\[
\boxed{
K_{\mathrm I}
=
Y\sigma\sqrt{\pi a}
}
\]

と表される。

- \(\sigma\)：公称応力
- \(a\)：き裂長さ
- \(Y\)：試験片形状やき裂形状による補正係数

単位は、

\[
\boxed{
\mathrm{MPa}\sqrt{\mathrm m}
}
\]

である。

応力拡大係数は、

\[
\boxed{
\text{荷重、き裂寸法、形状によって決まる、き裂先端応力場の強さ}
}
\]

を表す。

## 破壊じん性

応力拡大係数が材料固有の限界値へ達すると、き裂が不安定に進展する。

平面ひずみ条件での限界値を、

\[
\boxed{
K_{\mathrm{IC}}
}
\]

と表し、破壊じん性という。

\[
\boxed{
K_{\mathrm I}\ge K_{\mathrm{IC}}
}
\]

で不安定破壊が始まる。

破壊じん性が高い材料は、き裂が存在しても破壊しにくい。

### 違い

\[
\boxed{
\begin{aligned}
K_{\mathrm I}:&
\text{ 現在の荷重とき裂による状態量}\\
K_{\mathrm{IC}}:&
\text{ 材料のき裂進展抵抗を表す材料特性}
\end{aligned}
}
\]

:::why title="単純な最大応力だけではき裂破壊を評価できないの？"
き裂先端の応力は、き裂長さや形状によって非常に大きく集中する。公称応力が同じでも、き裂が長いほど破壊しやすいため、\(\sigma\sqrt a\) を含む応力拡大係数で評価する。
:::

### 試験答案

応力拡大係数 \(K\) は、き裂先端近傍の特異応力場の強さを表し、モードIでは、

\[
K_{\mathrm I}=Y\sigma\sqrt{\pi a}
\]

である。

破壊じん性 \(K_{\mathrm{IC}}\) は、平面ひずみ条件で不安定き裂進展が開始する限界応力拡大係数であり、材料のき裂に対する抵抗を表す。

# Q6-(2) マルテンサイトが強くて硬い理由

マルテンサイトは、オーステナイトを急冷したときに生じる、拡散を伴わない変態組織である。

主な強化要因は次のとおり。

## 1. 炭素の過飽和固溶

急冷では炭素が拡散してセメンタイトを形成する時間がない。

そのため、本来の平衡固溶限を超える炭素が鉄格子中へ閉じ込められる。

侵入型炭素原子が大きな弾性ひずみ場を作り、転位の移動を妨げる。

\[
\boxed{\text{侵入型固溶強化}}
\]

である。

## 2. 体心正方格子、BCTの格子ひずみ

炭素が特定の八面体位置へ入るため、bcc格子の一軸が伸び、body-centered tetragonal、BCT構造となる。

正方晶ひずみが大きいほど、転位の移動に必要な応力が増える。

## 3. 高い転位密度

マルテンサイト変態では大きなせん断変形と体積変化が生じる。

変態ひずみを緩和するため、非常に多くの転位が導入される。

転位同士が絡み合い、他の転位の移動を妨げる。

\[
\boxed{\text{転位強化}}
\]

である。

## 4. 微細なラス・プレート組織

マルテンサイトは、細かいラス状または板状に形成される。

ラス境界、ブロック境界、パケット境界が転位運動の障害となる。

Hall-Petch型の微細組織強化が働く。

## 5. 双晶・内部欠陥

高炭素マルテンサイトでは変態双晶が形成される場合がある。

双晶境界も転位の移動を妨げる。

## 6. 変態に伴う残留応力

急冷による温度差と変態膨張により、大きな内部応力が残る。

これは硬さへ寄与する一方で、割れやすさの原因にもなる。

### 最終解答

マルテンサイトが強く硬い主な理由は、

\[
\boxed{
\begin{aligned}
&\text{炭素の過飽和固溶}\\
&\text{BCT格子の大きなひずみ}\\
&\text{高い転位密度}\\
&\text{微細なラス・プレート組織}\\
&\text{双晶などの高密度な内部境界}
\end{aligned}
}
\]

である。

### 注意

マルテンサイトは非常に硬いが、そのままではじん性が低く、脆い。

実用鋼では焼戻しによって、硬さとじん性のバランスを調整することが多い。

# Q6-(3) 日本刀が切削武器として優れる理由

日本刀の性能は、材料特性と形状の両方によって生じる。

## 1. 刃先が硬い

刃先は焼入れによって主にマルテンサイト組織となる。

したがって、

- 高硬度
- 高強度
- 耐摩耗性
- 刃先が塑性変形しにくい

という特徴を持つ。

鋭い刃先を長く維持できる。

## 2. 棟・心部が比較的軟らかく、じん性が高い

粘土を用いた差し焼入れでは、刃先が速く冷え、棟側は比較的ゆっくり冷える。

棟・心部にはフェライト、パーライト、焼戻し組織など、刃先より軟らかくじん性のある組織が残る。

そのため、衝撃を受けても刀全体が一気に脆性破壊しにくい。

## 3. 硬さとじん性を空間的に分けている

\[
\boxed{
\text{硬い刃先}
+
\text{粘り強い内部・棟}
}
\]

という傾斜的な材料設計になっている。

単一の均質材料では両立しにくい、

- 切れ味
- 耐摩耗性
- 耐衝撃性
- 折れにくさ

を同時に得ている。

## 4. 鍛錬と接合構造

折り返し鍛錬によって、

- 組成を均一化する
- スラグや空隙を細かく分散・圧着する
- 組織を細かくする

効果が得られる。

また、硬い鋼と比較的軟らかい鋼を組み合わせる構造により、性能を分担できる。

## 5. 刃の形状と湾曲

鋭い刃角は小さな面積へ力を集中させる。

湾曲した片刃を引くように動かすと、刃が対象物へ接触する位置が連続的に移動し、押し切りだけでなく引き切りが働く。

これにより、局所的な応力を高めながら切断を進められる。

### 最終解答

日本刀が優れる主な理由は、

\[
\boxed{
\text{マルテンサイトによる硬い刃先と、じん性の高い棟・心部を組み合わせた複合的構造}
}
\]

にある。

さらに、鍛錬による欠陥低減と、鋭い刃角・湾曲形状による応力集中と引き切り効果が、切断性能を高める。

:::why title="刀全体を最も硬いマルテンサイトにしないの？"
硬さを上げるほど、一般にじん性が低下して欠けや折損が起こりやすくなる。刃先だけを硬くし、他の部分へ粘りを持たせることで、切れ味と折れにくさを両立する。
:::

# Q6-(4) オーステナイト系ステンレスの特徴

代表例は、

\[
\boxed{\mathrm{SUS304}}
\]

であり、18Cr-8Ni系ステンレスとも呼ばれる。

## 1. 室温でオーステナイト組織

Niなどのオーステナイト安定化元素を添加し、室温でもfcc構造のオーステナイト、\(\gamma\)相を安定化している。

## 2. 優れた耐食性

約18%のCrを含み、表面へ薄く緻密なCr酸化物の不動態皮膜を形成する。

この皮膜が破れても、酸素があれば再形成される。

そのため、大気、水、各種薬品に対して良好な耐食性を示す。

## 3. 高い延性・じん性・加工性

fcc構造はすべり系が多い。

したがって、

- 延性が高い
- 成形加工しやすい
- 低温でもじん性が高い
- 明瞭な延性脆性遷移を起こしにくい

という特徴を持つ。

## 4. 通常は非磁性

溶体化処理した安定なオーステナイト状態では、基本的に非磁性である。

ただし、強い冷間加工により加工誘起マルテンサイトが生じると、磁性を帯びる場合がある。

## 5. 熱処理だけでは硬化しにくい

通常の焼入れによるマルテンサイト硬化は利用できない。

強度を高めるには、

- 冷間加工
- 固溶強化
- 析出硬化型合金での時効処理

などを使う。

## 6. 溶接性が良い

一般に溶接・加工しやすく、建築、食品設備、化学装置、家庭用品など広く使われる。

## 7. 注意すべき弱点

### 塩化物環境での応力腐食割れ

引張応力と塩化物イオンが共存すると、応力腐食割れを生じる場合がある。

### 孔食・すきま腐食

塩化物によって不動態皮膜が局所的に破れることがある。

### 鋭敏化

およそ500〜800℃付近で保持されると、粒界へCr炭化物が析出する場合がある。

粒界周辺でCrが不足し、粒界腐食が起こりやすくなる。

低炭素材や安定化鋼、適切な溶体化処理で対策する。

### 最終解答

オーステナイト系ステンレスは、

\[
\boxed{
\text{CrとNiにより室温でfccオーステナイトを安定化した、耐食性・延性・じん性・加工性に優れる鋼}
}
\]

である。

通常は非磁性で、低温じん性と溶接性にも優れる。

一方、焼入れ硬化しにくく、塩化物応力腐食割れや鋭敏化へ注意が必要である。
